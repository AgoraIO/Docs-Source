The `LocalUser` class. Defines the behavior and state of the local user. Each RTC connection has its own user. You can get the local user through [`get_local_user`](./rtcconnection#getlocaluser). Each user has two roles: the host (can send and receive audio and video streams) and the audience (can only receive audio and video streams).

### `set_user_role`

Sets the user role. You can call this method before or after the user connects to the channel.

```python
def set_user_role(self, role)
```

<Admonition type="caution" title="Note">
If the `role` set in the `token` parameter of [`Connect`](./rtcconnection#connect) is different from the role set here, the connection will fail.
</Admonition>

#### Parameters

|Parameter|Description'|
|`role`|User role. See [`ClientRoleType`](./apidatatype#clientroletype).|

#### Returns

- `0`: Success.
- < `0`: Failure.
