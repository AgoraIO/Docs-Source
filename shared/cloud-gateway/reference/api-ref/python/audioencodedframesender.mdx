The `AudioEncodedFrameSender` Class.

### `send_encoded_audio_frame`

Sets the path and size of the SDK log file.

```python
def send_encoded_audio_frame(self, buffer_ptr:int, buffer_size:int, frame:EncodedAudioFrame)
```

#### Parameters

| Parameter       | Description                                                                 |
|------------------|----------------------------------------------------------------------|
| `buffer_ptr`   | The encoded audio data to send.                                               |
| `buffer_size` | The size of the data (bytes).   |
| `frame` | The audio frame information, see [`EncodedAudioFrame`](#encodedaudioframe).   |

#### Returns

- 0: Success.
- < 0: Failure.
