1.  **Add an encryption key and salt to your <Vpl k="CLIENT" />**

    1.  To obtain a **32-byte key**, run the following command in a terminal window:

        ```bash
        openssl rand -hex 32
        ```

    1.  Paste the key returned as the value for the `encryptionKey` parameter in the `config.json` file.

    1.  To obtain a `base64` encoded  **32-byte salt**, run the following command in a terminal window:

        ```bash
        openssl rand -base64 32
        ```

    1.  Paste the salt returned as the value for the `salt` parameter in the `config.json` file.

1. **Run the reference app**

    <RunApp/>
