* **Create a Unity project**:

    1. In Unity Hub, select **Projects**, then click **New Project**.

    1. In **All templates**, select **3D(URP)**. Set the **Project name** and **Location**, then click **Create Project**.

    1. In **Projects**, double-click the project you created. Your project opens in Unity.
