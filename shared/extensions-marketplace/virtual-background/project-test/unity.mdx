1. **Enable virtual background** 

    1. Click on **Enable Virtual Background**. The background of your local video on the test device changes to a solid blue color, and the remote video in the web demo updates. You see the button text change, confirming that color background is enabled.

    1. Press the button again. The background changes to a blur.

    1. Press the button once more. You see an image in the background.

    1. Press the button again. Virtual background is disabled, and the actual background is restored.
