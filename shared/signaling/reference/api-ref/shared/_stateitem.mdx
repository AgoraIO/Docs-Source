`StateItem` contains the following properties:

|  Property  |  Type  | Description                          |
| :------------------ | :--------------------: | :------------------------------------- |
| `key`         |         string        | Key of user status. If the provided key already exists, the new value overwrites the previous one. If the provided key does not exist, a new key-value pair is created.用户状态的键。    |
| `value` | string | Value of user status.  |
