1. **Implement the geofencing logic**

   To enable geofencing in your <Vpl k="CLIENT" />, set `areaCode` property of the `RTCEngineContext` object before passing it to `initialize` method. In `App.tsx`, add the following code after `this._rtcContext.appId = appId;`:

    ```js
    this._rtcContext.areaCode = AreaCode.AreaCodeNa;
    ```
   If your <Vpl k="CLIENT" /> fails to connect to the specified region of <Vg k="AGORA_BACKEND" />, instead of connection to another <Vg k="AGORA_BACKEND" /> region, <Vpd k="SDK" /> throws an error. If a firewall is deployed in your network environment, ensure that you:

    * Whitelist certain domains
    * Allow all IP addresses 
    * Open the firewall ports defined in <Link to="../../{{product.PATH}}/develop/cloud-proxy">Use Cloud Proxy</Link>.
