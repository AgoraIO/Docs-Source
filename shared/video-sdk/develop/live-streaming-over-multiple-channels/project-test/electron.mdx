### Test channel media Relay

1.  In **preload.js**, update `appID`, `channel` and `destChannelName`.

2. [Generate a temporary token](../reference/manage-agora-account#generate-a-temporary-token) in <Vpl k="CONSOLE" /> using `appID` and `channel`. Use it to update `token` in `preload.js`. Use the same values to generate another token and update `sourceChannelToken`.

3. Generate a third token in <Vpl k="CONSOLE" /> using `appID` and `destChannelName`. Use it to update `destChannelToken` in `preload.js`.

4. In your browser, navigate to the <Link target="_blank" to="{{Global.DEMO_BASIC_VIDEO_CALL_URL}}"><Vg k="COMPANY" /> web demo</Link> and update _App ID_, _Channel_, and _Token_ with the values for your temporary token, then click **Join**.

5. Repeat the previous step on a second device, but this time use `appId`, `destChannelName`, and `destChannelToken` to **Join** the channel.

6.  Run the app
    
    Execute the following command in the terminal:
    ```bash
    npm start
    ```
    You see your app opens a window named **Get started with <Vpd k="PRODUCT" />**.

8. Select **Host** and press **Join**. You see the video from the web browser demo app connected to `channel` in the top frame of your development device.

9. Press **Start Channel Media Relay**. You see the video from the web browser demo app connected to `channel` relayed to the web browser demo app connected to `destChannelName`.

10. Press **Stop Channel Media Relay**. The media relaying is stopped.

### Test joining multiple channels

1.  In `preload.js`, update `appID`, `channel` and `secondChannelName`.

2. [Generate a temporary token](../reference/manage-agora-account#generate-a-temporary-token) in <Vpl k="CONSOLE" /> using `appID` and `channel`. Use it to update `token` in `preload.js`.

3. Generate a second token in <Vpl k="CONSOLE" /> using `appID` and `secondChannelName`. Use it to update `secondChannelToken` in `preload.js`.

4. In your browser, navigate to the <Link target="_blank" to="{{Global.DEMO_BASIC_VIDEO_CALL_URL}}"><Vg k="COMPANY" /> web demo</Link> and update _App ID_, _Channel_, and _Token_ with the values for your temporary token, then click **Join**.

5. Repeat the previous step on a second device, but this time use `appID`, `secondChannelName`, and `secondChannelToken` to join the channel.

6.  Run the app
    
    Execute the following command in the terminal:
    ```bash
    npm start
    ```
    You see your app opens a window named **Get started with <Vpd k="PRODUCT" />**.

8. Select **Audience** 

    1. Press **Join**. You see the video from the web browser demo app connected to `channel` in the top frame of your development device.
    2. Press **Join Second Channel**. You see the video from the web browser demo app connected to `secondChannelName` in the bottom frame of your development device.
    3. Press **Leave Second Channel** and then **Leave** to exit both channels.

9. Select **Host** 

    1. Press **Join**. You see the local video in the top frame of your development device. The web browser demo app connected to `channel` shows the video from your development device.
    2. Press **Join Second Channel**. You see the video from the web browser demo app connected to `secondChannelName` in the bottom frame of your development device.
    3. Press **Leave Second Channel** and then **Leave** to exit both channels.
