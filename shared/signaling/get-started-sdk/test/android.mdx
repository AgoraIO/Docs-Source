<PlatformWrapper platform="android">

Save the project.
Click Run > Select Device > Select Multiple Devices in the menu bar and add two devices. For example, select a virtual machine and a real machine.
Run the project. After running successfully, you can see the App interface in both devices.
Use a device as the receiving end and perform the following operations:
Enter your username and click LOGIN .
Enter the channel name and click SUB .
Use another device as the sender and do the following:
Enter a different username and click LOGIN .
Enter the same channel name.
Enter the message and click PUBLISH MSG .
Swap the above sending end and device end and repeat steps 4 and 5.
If both devices can send and receive messages, you have successfully integrated and used the RTM service correctly.
