### Sample project

The complete sample code and project is provided on GitHub:

- <Link to="{{Global.AGORA_MARKETPLACE_BASE_URL}}/FaceUnity/ios/ExtensionExample/ViewController.m">FaceUnity/ios</Link>

#### Run the project

1. Clone the repository:
```shell
	git clone https://github.com/AgoraIO-Community/AgoraMarketPlace.git
```
1. On the <Link to="{{Global.AGORA_CONSOLE_URL}}/marketplace/list/all">Extensions Marketplace Downloads</Link> page, download the iOS package of FaceUnity AR Filter. Unzip the package, save all `.framework` files to `FaceUnity/ios/ExtensionExample`.
1. Contact Agora to get the certificate file and resource package.
1. Save the certificate file `authpack.h` to `FaceUnity/ios/ExtensionExample`.
1. Save the required model and prop files from the resource package to `FaceUnity/ios/Resource` under the project folder.
1. In Terminal, go to the `FaceUnity/ios directory`, and run the following command to install dependencies using CocoaPods:
    ```
  	pod install
    ```
1. Open the project `FaceUnity/ios/ExtensionExample.xcworkspace` in Xcode.
1. Open the `Config.h` file and replace `<YOUR_APP_ID>` with your App ID. To get an App ID, see [Getting Started with Agora](../../reference/manage-agora-account#get-the-app-id).

   ```objective-c
   NSString *const appID = @"<YOUR_APP_ID>";
   NSString *const token = nil;
   ```
1. Connect an iOS device (not an emulator), and run the project.

    If successful, the sample project is installed on your Android or iOS device.

1. When the app is launched, the FaceUnity extension is enabled by default. You can see the `disableExtension` , `enableAITracking` , `setComposer`, and `setSticker `buttons on the user interface.
1. Perform the following steps:
   - Click `enableAlTracking` to ` enable `face, gesture, and body recognition. You can see the recognition results in the upper right corner.
   - Click `setComposer` to set the beauty effect, and drag the **Color Level** or** Filter Level** to adjust the beautification intensity.
   - Click `setSticker` to load the props. The default is a cat ear effect.
