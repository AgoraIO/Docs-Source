The `VideoFrameObserver` class.

#### `OnFrame`

Triggered when a video frame is received.

```go
OnFrame func(channelId string, userId string, frame *VideoFrame) bool
```

#### Parameters

| Parameter | Description |
|---------------|----------------------------------------------------------------------|
| `channelId` | The channel name.             |
| `userId` | The user ID of the remote user.             |
| `frame`| The video data. See [`VideoFrame`](./apidatatype#videoframe) for details.         |
