To test authentication using a token server:

1. **Configure the project**

    1. Setup an [Authentication server](../get-started/authentication-workflow?platform#create-and-run-a-token-server)
    1. <OpenConfig/>
    1. In `config.json`, set:
        - `token` to an empty string.
        - `serverUrl` to the base URL for your token server. For example: `https://agora-token-service-production-yay.up.railway.app`.

1. **Run the reference app**

    <RunApp/>

1. **Test server authentication**:

    For each user you want to add to the conversation:

    1. Launch an instance of the <Vpl k="CLIENT"/>.
    1. Enter a numeric user ID, then press **Login**.
    1. Enter a channel name, then press **Subscribe** to join a channel.
    1. Send and receive messages between users.
