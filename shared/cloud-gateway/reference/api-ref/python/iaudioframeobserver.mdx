### `on_playback_audio_frame_before_mixing`

Gets the data before the playback audio is mixed.

```python
def on_playback_audio_frame_before_mixing(self, agora_local_user, channelId, uid, frame:AudioFrame)
```

#### Parameters

| Parameter | Description |
|---------------|----------------------------------------------------------------------|
| `agora_local_user`        | The `LocalUserObserver` class.    |
| `channelId` | The channel name.             |
| `uid` | The remote user ID.             |
| `frame`| The audio raw data. See [`AudioFrame`](#audioframe)         |

#### Returns

- `0`：The audio frame data is invalid, and the data is discarded.
- `1`：The audio frame data is valid, and the data is sent.
