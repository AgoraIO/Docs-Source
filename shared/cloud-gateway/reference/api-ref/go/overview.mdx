| Method | Description |
| --- | --- |
| [`NewRtcConnection`](./rtcconnection#newrtcconnection) | Creates an RTC connection object. |
| [`Connect`](./rtcconnection#connect) | Connects to the RTC channel. |
| [`Disconnect`](./rtcconnection#disconnect) | Disconnects from the RTC channel. |
| [`RenewToken`](./rtcconnection#renewtoken) | Renews the token. |
| [`RegisterObserver`](./rtcconnection#registerobserver) | Registers an RTC connection observer. |
| [`UnregisterObserver`](./rtcconnection#unregisterobserver) | Destroys the RTC connection observer. |
| [`CreateDataStream`](./rtcconnection#createdatastream) | Creates a data stream. |

| [`Release`](./rtcconnection#release) | Releases the RTC connection. |

## RtcConnectionObserver Class

| Method | Description |
| --- | --- |
| [`OnConnected`](./rtcconnectionobserver#onconnected) | Occurs when the SDK connection state changes to `3`. |
| [`OnDisconnected`](./rtcconnectionobserver#ondisconnected) | Occurs when the SDK connection state changes to `1`. |
| [`OnTokenPrivilegeWillExpire`](./rtcconnectionobserver#ontokenprivilegewillexpire) | Occurs when the token service will expire in 30 seconds. |
| [`OnTokenPrivilegeDidExpire`](./rtcconnectionobserver#ontokenprivilegedidexpire) | Occurs when the token has expired. |
| [`OnUserJoined`](./rtcconnectionobserver#onuserjoined) | Occurs when a remote user joins the channel. |
| [`OnUserLeft`](./rtcconnectionobserver#onuserleft) | Occurs when a remote user leaves the channel. |
| [`OnStreamMessageError`](./rtcconnectionobserver#onstreammessageerror) | Occurs when there is an error receiving a data stream message. |
| [`OnError`](./rtcconnectionobserver#onerror) | Occurs when an error occurs. |
