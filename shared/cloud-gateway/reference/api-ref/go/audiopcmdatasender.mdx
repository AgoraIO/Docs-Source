<h2 className="anchor index-group-item-audiopcmdatasender index-api-newaudiopcmdatasender" id="newaudiopcmdatasender">NewAudioPcmDataSender</h2>

```go
func (mediaNodeFactory *MediaNodeFactory) NewAudioPcmDataSender() *AudioPcmDataSender {
    sender := C.agora_media_node_factory_create_audio_pcm_data_sender(mediaNodeFactory.cFactory)
    if sender == nil {
        return nil
    }
    return &AudioPcmDataSender{
        cSender: sender,
    }
}
```

<span className="index-desc-newaudiopcmdatasender">Creates a PCM data sender module. This method creates an `AudioPcmDataSender` object, which can be used in the [`NewCustomAudioTrackPcm`](./localaudiotrack#newcustomaudiotrackpcm) method.</span>

#### Returns

- `AudioPcmDataSender` object, if the method call succeeds.
- `nil`, if the method call fails.

<h2 className="anchor index-group-item-audiopcmdatasender index-api-sendaudiopcmdata" id="sendaudiopcmdata">SendAudioPcmData</h2>

```go
func (sender *AudioPcmDataSender) SendAudioPcmData(frame *AudioFrame) int {
    cData := C.CBytes(frame.Buffer)
    defer C.free(cData)
    return int(C.agora_audio_pcm_data_sender_send(sender.cSender, cData,
        C.uint(frame.RenderTimeMs), C.uint(frame.SamplesPerChannel),
        C.uint(frame.BytesPerSample), C.uint(frame.Channels),
        C.uint(frame.SamplesPerSec)))
}
```

<span className="index-desc-sendaudiopcmdata">Sends PCM format audio data to the local audio track.</span>

#### Parameters

| Parameter | Description |
|-----------|-------------|
| `frame`   | [`AudioFrame`](./apidatatype#audioframe) object containing the PCM audio data to be sent. |

#### Returns

- 0: Success.
- < 0: Failure.

<h2 className="anchor index-group-item-audiopcmdatasender index-api-release" id="release">Release</h2>

```go
func (sender *AudioPcmDataSender) Release() {
    if sender.cSender == nil {
        return
    }
    C.agora_audio_pcm_data_sender_destroy(sender.cSender)
    sender.cSender = nil
}
```

<span className="index-desc-release">Releases the `AudioPcmDataSender` object.</span>
