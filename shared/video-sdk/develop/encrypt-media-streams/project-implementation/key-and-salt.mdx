1. Generate a key and salt

    In a real-world app, you generate the key and salt on your server. In this demo, you generate them using OpenSSL from the terminal.

    - Execute the following command to generate a random, 32-byte String key:

        ```shell
        # Generate a random 32-byte String key
        openssl rand -hex 32
        dba643c8ba6b6dc738df43d9fd624293b4b12d87a60f518253bd10ba98c48453
        ```

    - Execute the following command to generate a random, Base64-encoded, 32-byte salt:

        ```shell
        # Generate a random, Base64 encoded, 32-byte salt
        openssl rand -base64 32
        X5w9T+50kzxVOnkJKiY/lUk82/bES2kATOt3vBuGEDw=
        ```
