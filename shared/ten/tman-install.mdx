Follow these steps to install and run <Vpd k="MANAGER" />

1. Open the [TEN Framework releases page](https://github.com/TEN-framework/ten_framework/releases) on GitHub.

1. Find the TEN Manager package for your OS and architecture. The package uses the naming structure `tman-[OS]-release-[ARCH].zip`. 

1. Download the package. For example, you can download the Linux package for x64 using:

    ```bash
    wget https://github.com/TEN-framework/ten-framework/releases/download/0.10.10/tman-linux-release-x64.zip
    ```

1. Extract the downloaded package:

    ```bash
    unzip tman-linux-release-x64.zip
    ```

1. Install TEN Manager by copying the binary to your system PATH so that you can run `tman` commands from any directory (Optional):

    ```bash
    sudo cp ten_manager/bin/tman /usr/local/bin/
    ```

1. Verify the installation:

    ```bash
    tman --version
    ```
