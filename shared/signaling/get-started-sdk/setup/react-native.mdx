Follow these steps to set up your project and add the Signaling SDK.

### Create a Project

To create a new React Native application using the [React Native Community CLI](https://github.com/react-native-community/cli), see [Get Started Without a Framework](https://reactnative.dev/docs/getting-started-without-a-framework).

### Install the SDK

To download and add the <Vg k="COMPANY" /> <Vpd k="SDK"/> to your React Native project:  

1. In your project directory, run one of the following commands:

   **npm:**

   ```bash
   npm install --save agora-react-native-rtm
   ```

   **Yarn:**

   ```bash
   yarn add agora-react-native-rtm
   ```

2. Navigate to the `ios` directory and run:

   ```bash
   pod install
   ```
