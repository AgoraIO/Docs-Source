Call `setAINSMode` to enable the audio noise suppression feature and select a noise suppression mode:

```cpp
// Enable Audio AI Noise Suppression
bool enableAINS = true;

// Set the mode for Audio AI Noise Suppression
// Choose one of the available modes: AINS_MODE_BALANCED, AINS_MODE_AGGRESSIVE, AINS_MODE_ULTRALOWLATENCY
AUDIO_AINS_MODE mode = AINS_MODE_BALANCED;

int result = rtcEngine->setAINSMode(enableAINS, mode);
```
