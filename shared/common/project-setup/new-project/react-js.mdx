1. **Install NodeJS**:

    Ensure that [Node.js](https://nodejs.org/en) is installed on your development machine.

1. **Initialize a new project**
    
    Open a terminal and run the following command to create and initialize a new project using `Vite`:

    ```bash
    npx create-vite my-agora-project --template react-ts
    ```
    This command uses `Vite` to scaffold a new React project with `TypeScript`.

1. **Install the project dependencies**:
    
    Install Agora SDK and the extensions required for your project:

    ```bash
    cd my-agora-project
    npm install agora-rtc-sdk-ng agora-rtc-react agora-extension-ai-denoiser agora-extension-spatial-audio agora-extension-virtual-background --save
    ```
