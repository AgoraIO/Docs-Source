When configuring the transcoded output media stream using the `videoOption`,  <Vg k="COMPANY"/> recommends using the default values. However, if you need custom settings, refer to the following table for guidance.

<Admonition type="info" info="info">
If the bitrate you specify is outside the recommended range, the server automatically adjusts it to a reasonable value.
</Admonition>
