<PlatformWrapper platform="react-native">

1. Execute the following command in the project root directory to run the sample project.

    * Target platform Android

        ```bash
        yarn example android
        ```

    * Target platform iOS

        ```bash
        yarn example ios
        ```

1. After the operation is complete, the following application appears on your device: 

    ![](/images/video-sdk/run-compile-rn-step-2.png)

    Showing Android as an example

1. Select the demo you want to experience from the menu. Choose **JoinChannelVideo** under **Basic** as an example, and click **join channel**.

    ![](/images/video-sdk/run-compile-rn-step-3.png)

1. To test various audio and video interaction use-cases, connect to the [Agora web demo](https://webdemo.agora.io/basicVideoCall/index.html), or install and run the sample project on a second device. Make sure you use the same app ID on both devices. When you join the same channel from two devices, you can see and hear each other.
