## IRecorderVideoFrameObserver class

Receives video frame callbacks from the recorder.


| API | Description |
|---|---|
|[`onYuvFrameCaptured`](#onyuvframecaptured) | Callback triggered when a YUV video frame is captured. |
|[`onEncodedFrameReceived`](#onencodedframereceived) | Callback triggered when an encoded frame or JPG image buffer is received. |
|[`onJPGFileSaved`](#onjpgfilesaved) | Callback triggered when a JPG file is saved to disk. |

### onYuvFrameCaptured

```java
default void onYuvFrameCaptured(String channelId, String userId, VideoFrame frame)
```

Triggered when a YUV video frame is captured.  
This occurs after calling [`enableRecorderVideoFrameCapture`](#enablerecordervideoframecapture) with `VideoFrameCaptureType` set to `VIDEO_FORMAT_YUV_FRAME_TYPE`.

| Parameter | Data Type | Description |
|-----------|-----------|-------------|
| `channelId` | `String` | Channel ID. |
| `userId` | `String` | User ID. |
| `frame` | `VideoFrame` | The captured YUV video frame. See [`VideoFrame`](#videoframe). |

### onEncodedFrameReceived

```java
default void onEncodedFrameReceived(String channelId, String userId, byte[] imageBuffer, EncodedVideoFrameInfo info)
```

Triggered when an encoded video frame or JPG image buffer is received.  
This occurs after calling [`enableRecorderVideoFrameCapture`](#enablerecordervideoframecapture) with `VideoFrameCaptureType` set to `VIDEO_FORMAT_ENCODED_FRAME_TYPE` or `VIDEO_FORMAT_JGP_FORMAT_TYPE`.

- For encoded frame type: Triggered each time a video frame is captured.
- For JPG format type: Triggered at the interval set in `RecorderVideoFrameCaptureConfig.jpgCaptureInternalInSec`.

| Parameter | Data Type | Description |
|-----------|-----------|-------------|
| `channelId` | `String` | Channel ID. |
| `userId` | `String` | User ID. |
| `imageBuffer` | `byte[]` | Encoded image buffer. |
| `info` | `EncodedVideoFrameInfo` | Encoded frame information. See [`EncodedVideoFrameInfo`](#encodedvideoframeinfo). |


### onJPGFileSaved

```java
default void onJPGFileSaved(String channelId, String userId, String filename)
```

Triggered when a JPG file is saved to disk.  
This occurs after calling [`enableRecorderVideoFrameCapture`](#enablerecordervideoframecapture) with `VideoFrameCaptureType` set to `VIDEO_FORMAT_JPG_FILE_TYPE`.  
The callback is triggered at the interval set in `RecorderVideoFrameCaptureConfig.jpgCaptureIntervalInSec`.

| Parameter | Data Type | Description |
|-----------|-----------|-------------|
| `channelId` | `String` | Channel ID. |
| `userId` | `String` | User ID. |
| `filename` | `String` | Absolute path of the saved JPG file. |
