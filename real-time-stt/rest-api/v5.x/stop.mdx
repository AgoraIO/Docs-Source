Use this endpoint to leave the channel and stop <Vpd k="NAME" />. To restart transcription, [`acquire`](../v5.x/acquire) a new one `buildToken` and call [`start`](../v5.x/start) again.

## Request

### Path parameters

<PathParameter name="appId" type="string" required={true}>
The App ID of the project
</PathParameter>
<PathParameter name="taskId" type="string" required={true}>
The unique identifier of the <Vpd k="NAME" /> task you received in the response body of the [`start`](../v5.x/start) method.
</PathParameter>

### Query parameters

<PathParameter name="builderToken" type="string" required={true}>
The `tokenName` value you obtained in the response body of the [`acquire`](../v5.x/acquire) method.  To stop a task, use the same `builderToken` you used to start the task.
</PathParameter>

## Response

- If the returned status code is `200`, the request was successful. 
- If the returned status code is not `200`, the request failed.
