The `type` field enables you to identify the message format and parse the data in the message payload for different message types appropriately. 

<Vg k="SIG" /> also provides the `customType` field in the <code>{props.publish}</code> and <code>{props.publishTopicMessage}</code> methods for sending additional custom message type information to the receiving end, along with the message. Use the `customType` field to identify the message type and parse the data in the message event.
