1. **Clone the repository**:

    To clone the repository to your local machine, open Terminal and navigate to the directory where you want to clone the repository. Then, use the following command:

    <ProductWrapper product="signaling" >
    ```bash
    git clone https://github.com/AgoraIO/signaling-sdk-samples-android
    ```
    </ProductWrapper>

1. **Install the dependencies**:

    Launch Android Studio. From the **File** menu, select **Open...** and navigate to the `signaling-reference-app` folder. Start Gradle sync to automatically install the  project dependencies.

    By default <Vpd  k="SDK" /> is installed automatically. However, you can also [Install <Vpd  k="SDK" /> manually](../reference/downloads#through-the-agora-website).
