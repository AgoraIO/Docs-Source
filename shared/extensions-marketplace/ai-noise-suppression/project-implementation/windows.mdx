<PlatformWrapper platform="windows">

1. **Integrate <Vg k="RTEE_NS" /> into your <Vpl k="CLIENT" />**

    If you installed the <Link to = "https://download.agora.io/sdk/release/Agora_Native_SDK_for_Android_v4.0.1_FULL.zip">Video SDK </Link> when you setup your  <Vpl k="CLIENT" />,  <Vg k="RTEE_NS" /> is already included.


1. **Initialize the variable to set <Vg k="RTEE_NS" /> options**

    In `AgoraImplementationDlg.h`, add the following variable declaration to `CAgoraImplementationDlg` class:

    ``` cpp
    VideoDenoiserOptions denoiserOptions;
    ```

1. **Set the <Vg k="RTEE_NS" /> mode**

    Auto mode automatically enables or disables the video noise reduction feature according to the ambient light. To set the <Vg k="RTEE_NS" /> mode to `auto`, in `AgoraImplementationDlg.cpp`, locate `setupVideoSDKEngine` and add the following code after `agoraEngine->initialize(context);`:

    ```cpp
    denoiserOptions.mode = VideoDenoiserOptions::VIDEO_DENOISER_AUTO;
    ```

1. **Adjust the <Vg k="RTEE_NS" /> level**

    By default, <Vg k="RTEE_NS" /> is set to 'HighQuality' to balance the <Vg k="RTEE_NS" /> quality and performance.

    - To set the value to 'Fast' and prioritize <Vg k="RTEE_NS" /> performance over quality, in `AgoraImplementationDlg.cpp` , put this line after `denoiserOptions.mode = VideoDenoiserOptions::VIDEO_DENOISER_AUTO;`:

        ```cpp
        denoiserOptions.level = VideoDenoiserOptions::VIDEO_DENOISER_LEVEL_FAST;
        ```

    - To give more focus to <Vg k="RTEE_NS" /> quality rather than performance, change the level to 'Strength':

        ```cpp
        denoiserOptions.level = VideoDenoiserOptions::VIDEO_DENOISER_LEVEL_STRENGTH;
        ```

6. **Set <Vg k="RTEE_NS" /> on <Vg k="ENGINE" />**

    In `AgoraImplementationDlg.cpp`, add the following code to `setupVideoSDKEngine` before `agoraEngine->setClientRole(CLIENT_ROLE_TYPE::CLIENT_ROLE_BROADCASTER);`:

    ```cpp
	agoraEngine->setVideoDenoiserOptions(true, denoiserOptions);
    ```
