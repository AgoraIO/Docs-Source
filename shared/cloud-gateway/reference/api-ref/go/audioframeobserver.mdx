<h2 className="anchor index-group-item-iaudioframeobserver index-api-onplaybackaudioframebeforemixing" id="onplaybackaudioframebeforemixing">OnPlaybackAudioFrameBeforeMixing</h2>

```go
OnPlaybackAudioFrameBeforeMixing  func(localUser *LocalUser, channelId string, uid string, frame *AudioFrame) bool
```

<span className="index-desc-onplaybackaudioframebeforemixing">Obtains the audio data before mixing for playback.</span>

#### Parameters

| Parameter        | Description                                                                 |
|------------------|-----------------------------------------------------------------------------|
| `localUser`      | [`LocalUser`](./localuser) class.                                           |
| `channelId`      | Channel name.                                                               |
| `uid`            | User ID of the remote user.                                                 |
| `frame`          | Raw audio data. See [`AudioFrame`](./apidatatype#audioframe) for details.   |

#### Returns

- `true`: The audio frame data is valid and will be sent.
- `false`: The audio frame data is invalid and will be discarded.
