### Get the sample project

1. Run the following command to clone the repository locally:

    ```bash
    git clone git@github.com:AgoraIO-Extensions/Agora-Unreal-RTC-SDK.git
    ```

1. The C++ API usage examples are located in `/Agora-Unreal-SDK-CPP-Example`. These include:

    | Description | Path	 |
    |:-------|:------------|
    | Basic functions |`/Agora-Unreal-SDK-CPP-Example/Content/API-Example/Basic`	|
    | Advanced functions |`/Agora-Unreal-SDK-CPP-Example/Content/API-Example/Advance` |

### Configure the sample project

Refer to the following steps to integrate the SDK:

1. Go to [SDKs](/sdks) to download the latest version of the <Vg k="VSDK" /> and unzip it.
1. Under your project root folder, create a folder named **Plugins**.
1. Copy **AgoraPlugin** from the SDK folder to **Plugins**.
