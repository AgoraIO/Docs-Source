### Beauty JSON Field Description

The JSON example and specific field description for achieving beauty effects are as follows:

```json
{
    "enable_mu": true,
    "browStyle": 1,
    "browColor": 2,
    "browStrength": 0.5,
    "lashStyle": 1,
    "lashColor": 2,
    "lashStrength": 0.5,
    "shadowStyle": 2,
    "shadowStrength": 0.5,
    "pupilStyle": 2,
    "pupilStrength": 0.5,
    "blushStyle": 1,
    "blushColor": 1,
    "blushStrength": 0.5,
    "lipStyle": 2,
    "lipColor": 2,
    "lipStrength": 0.5
}
```

- `enable_mu`: Boolean. Makeup switch. Settings:
  - `true`: Turn on the beauty effect.
  - `false`: Turn off the beauty effect.
- `browStyle`: Integer. Eyebrow type, optional values are:
  - `0`: closure
  - `1`: Type 1
  - `2`: Type 2
- `browColor`: Integer. Eyebrow color, optional values are:
  - `0`: none
  - `1`: black
  - `2`: brown
- `browStrength`: Float. Eyebrow strength, value range is [0.0, 1.0], where:
  - `0.0`: No strength
  - `1.0`: Maximum intensity
- `lashStyle`: Integer. Eyelash type, optional values are:
  - `0`: closure
  - `1`: Type 1
  - `2`: Type 2
- `lashColor`: Integer. Eyelash color, optional values are:
  - `0`: none
  - `1`: black
  - `2`: brown
- `lashStrength`: Float. Eyelash strength, value range is [0.0, 1.0].
- `shadowStyle`: Integer. Eyeshadow type, optional values are:
  - `0`: closure
  - `1`: Type 1
  - `2`: Type 2
- `shadowStrength`: Float. Eye shadow intensity, value range is [0.0, 1.0].
- `pupilStyle`: Integer. Pupil type, optional values are:
  - `0`: closure
  - `1`: Type 1
  - `2`: Type 2
- `pupilStrength`: Float. Pupil strength, value range is [0.0, 1.0].
- `blushStyle`: Integer. Blush type, optional values are:
  - `0`: closure
  - `1`: Type 1
  - `2`: Type 2
- `blushColor`: Integer. Blush color, optional values are:
  - `0`: none
  - `1`: Color No. 1
  - `2`: Color No. 2
  - `3`: Color No. 3
  - `4`: Color No. 4
  - `5`: Color No. 5
- `blushStrength`: Float. Blush intensity, value range is [0.0, 1.0].
- `lipStyle`: Integer. Lip gloss type, optional values are:
  - `0`: closure
  - `1`: Type 1
  - `2`: Type 2
- `lipColor`: Integer. Lip color, optional values are:
  - `0`: none
  - `1`: Color No. 1
  - `2`: Color No. 2
  - `3`: Color No. 3
  - `4`: Color No. 4
  - `5`: Color No. 5
- `lipStrength`: Float. Lip gloss intensity, value range is [0.0, 1.0].
