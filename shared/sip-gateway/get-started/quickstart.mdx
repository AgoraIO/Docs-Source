<Vpd k="NAME" /> allows developers to trigger inbound and outbound PSTN and SIP calls, which then connect into an Agora channel. This enables end users to participate with their phone for the audio leg of a conference call.

Contact support@agora.io to provision your app ID with this service. You will be provided with an authorization header to include in the API requests below.

## Inbound PSTN

The end user dials a phone number displayed to them and enters the PIN when prompted. With the correct PIN, they are connected into the Agora channel.

Direct Inward Dialing (DID) providers, such as Twilio, allow you to buy a phone number and have the calls forwarded to a SIP address. Agora will provide you with the SIP address to forward your calls to while provisioning on this service. You will also be able to customize the voice prompts played to your end users.

**Prototype**

- Method: `POST`
- Endpoint: `https://sipcm.agora.io/v1/api/pstn`

#### Request parameters

**Request body**

| Parameter | Data type        | Required/Optional      | Description                        |
|-----------|------------------|------------------------|------------------------------------|
| `appid`   | String           | Required               | The Agora project app ID.           |
| `token`   | String           | Optional               | The generated Agora RTC access token. |
| `uid`     | String/integer   | Optional               | The user UID.                       |
| `channel` | String           | Required               | The Agora channel name.             |
| `region`  | String           | Required               | The user's region where they will likely be located and calling from. Possible values include: <ul><li>`AREA_CODE_NA`: North America,</li><li>`AREA_CODE_EU`: Europe,</li><li>`AREA_CODE_AS`: Asia, excluding Mainland China,</li><li>`AREA_CODE_JP`: Japan,</li><li>`AREA_CODE_IN`: India,</li><li>`AREA_CODE_CN`: Mainland China.</li></ul> |

#### Request example

```json
{
  "action":"inbound",
  "appid":"fs9f52d9dcc1f406b93d97ff1f43c554f",
  "token":"006d2776fef40dc864dc7a438e3b871IACGnkorQsr0iCpBmzNwKEdzKuAv1b1zRcMy0cOradw6mHN/il8AAAAAIgCQqbAFeaVzZgQAAQDw0oBmAgDw0oBmAwDw0oBmBADw0oBm",
  "uid":"123",
  "channel":"pstn",
  "region":"AREA_CODE_NA",
}
```

#### Response parameters

| Parameter | Description                                                 |
|-----------|-------------------------------------------------------------|
| `did`     | The phone number to dial.                                   |
| `display` | The phone number to dial in a more friendly display format. |
| `pin`     | The pin to enter when prompted.                             |

#### Response example

**Request successful**

- Status code: 200 OK
- Content type: `application/json`

```json
{
    "did": "17377583411",
    "display": "+1 (737) 758 3411",
    "pin": "780592",
}
```

**Error code responses**

| Error code | Description                     |
|------------|---------------------------------|
| `401`      | Unauthorized                    |
| `500`      | Missing parameters              |
| `503`      | No resource currently available |

## Outbound PSTN

The end user receives a phone call that connects them directly to the channel when they answer.

**Prototype**

- Method: `POST`
- Endpoint: `https://sipcm.agora.io/v1/api/pstn`

#### Request parameters

**Request body**

| Parameter | Data type        | Required/Optional      | Description                        |
|-----------|------------------|------------------------|------------------------------------|
| `appid`   | String           | Required               | The Agora project app ID.           |
| `token`   | String           | Optional               | The generated Agora RTC access token. |
| `uid`     | String/integer   | Optional               | The user UID.                       |
| `channel` | String           | Required               | The Agora channel name.             |
| `prompt`  | String           | Required               | Play the callee a voice prompt and wait for them to press a digit. If set to `lazy`, then any DTMF may be pressed.|
| `to`      | String           | Required               | The end user's phone number to dial. You can optionally add a `#` followed by numbers which will be played as DTMF once the call connects. |
| `from`    | String           | Required               | The calling number displayed on the end user's phone while calling. |
| `sip`     | String           | Required               | The termination SIP URI. Leave blank if being routed by this service. |
| `timeout` | String           | Optional               | The max duration for an outbound call in seconds. Defaults to 3600 seconds (1 hour). |
| `region`  | String           | Required               | The user's region where they will likely be located and calling from. Possible values include: <ul><li>`AREA_CODE_NA`: North America,</li><li>`AREA_CODE_EU`: Europe,</li><li>`AREA_CODE_AS`: Asia, excluding Mainland China,</li><li>`AREA_CODE_JP`: Japan,</li><li>`AREA_CODE_IN`: India,</li><li>`AREA_CODE_CN`: Mainland China.</li></ul> |

#### Request example

```json
{
  "action":"outbound",
  "appid":"fs9f52d9dcc1f406b93d97ff1f43c554f",
  "token":"006d2776fef40dc864dc7a438e3b871IACGnkorQsr0iCpBmzNwKEdzKuAv1b1zRcMy0cOradw6mHN/il8AAAAAIgCQqbAFeaVzZgQAAQDw0oBmAgDw0oBmAwDw0oBmBADw0oBm",
  "uid":"123",
  "channel":"pstn",
  "region":"AREA_CODE_NA",
  "prompt":"true",
  "to":"+447712886400#333",
  "from":"+1800222333",
  "timeout":"3600",
  "sip":"acme.pstn.ashburn.twilio.com"
}
```

#### Response parameters

| Parameter | Required/Optional | Description                                                 |
|-----------|-------------------|-----------------------------------------|
| `success` | Required          | If the value is `true`, then the call has connected and the `callid` parameter will be included in the response. If `false`, then the call has failed and the `reason` parameter will be included in the response.|
| `callid`  | Optional          | This value can be used to end the call. |
| `reason`  | Optional          | The call failed because the user was busy or the destination was invalid.|

#### Response example

**Request successful**

- Status code: 200 OK
- Content type: `application/json`

```json
{
    "success": true,
    "callid": "88877-55Asdd7-55Asdd",
    "reason": "Busy",

}
```

**Error code responses**

| Error code | Description                     |
|------------|---------------------------------|
| `401`      | Unauthorized                    |
| `500`      | Missing parameters              |
| `503`      | No resource currently available |

## Inbound SIP

An inbound SIP address is requested. When the SIP address is dialled, the call will be routed to the requested user/channel session.

Using this API, you can bridge an outbound call from your provider with an inbound SIP address into Agora.

**Prototype**

- Method: `POST`
- Endpoint: `https://sipcm.agora.io/v1/api/pstn`

#### Request parameters

**Request body**

| Parameter | Data type        | Required/Optional      | Description                        |
|-----------|------------------|------------------------|------------------------------------|
| `appid`   | String           | Required               | The Agora project app ID.           |
| `token`   | String           | Optional               | The generated Agora RTC access token. |
| `uid`     | String/integer   | Optional               | The user UID.                       |
| `channel` | String           | Required               | The Agora channel name.             |
| `region`  | String           | Required               | The user's region where they will likely be located and calling from. Possible values include: <ul><li>`AREA_CODE_NA`: North America,</li><li>`AREA_CODE_EU`: Europe,</li><li>`AREA_CODE_AS`: Asia, excluding Mainland China,</li><li>`AREA_CODE_JP`: Japan,</li><li>`AREA_CODE_IN`: India,</li><li>`AREA_CODE_CN`: Mainland China.</li></ul> |

#### Request example

```json
{
  "action":"inboundsip",
  "appid":"fs9f52d9dcc1f406b93d97ff1f43c554f",
  "token":"006d2776fef40dc864dc7a438e3b871IACGnkorQsr0iCpBmzNwKEdzKuAv1b1zRcMy0cOradw6mHN/il8AAAAAIgCQqbAFeaVzZgQAAQDw0oBmAgDw0oBmAwDw0oBmBADw0oBm",
  "uid":"123",
  "channel":"test",
  "region":"AREA_CODE_NA",
}
```

#### Response parameters

| Parameter | Required/Optional | Description                                                 |
|-----------|-------------------|-----------------------------------------|
| `sip`     | Required          | The SIP address to dial to join the call.|

#### Response example

**Request successful**

- Status code: 200 OK
- Content type: `application/json`

```json
{
    "sip": "sip:lookup_783410988998@2.4.6.7:5080",
}
```

**Error code responses**

| Error code | Description                     |
|------------|---------------------------------|
| `401`      | Unauthorized                    |
| `500`      | Missing parameters              |
| `503`      | No resource currently available |

## Static PIN

If provisioned, the service can call out to an external REST endpoint providing the number dialed and PIN entered. The REST endpoint can choose to accept the PIN and return the details needed to join the user to the channel or error status 404, if PIN is not valid.

This API allows you to give your users a pin that will not expire.

**Prototype**

- Method: `POST`
- Endpoint: `https://example.customer.com/api/pinlookup`

#### Request parameters

**Request body**

| Parameter  | Description                        |
|------------|------------------------------------|
| `did`      | The phone number dialed.           |
| `pin`      | The pin entered.                   |

#### Request example

```json
{
  "did":"17177440111",
  "pin":"334455",
}
```

#### Response parameters

| Parameter | Data type        | Required/Optional      | Description                           |
|-----------|------------------|------------------------|---------------------------------------|
| `token`   | String           | Optional               | The generated Agora RTC access token. |
| `uid`     | String/integer   | Optional               | The user UID.                         |
| `channel` | String           | Required               | The Agora channel name.               |
| `appid`   | String           | Required               | The Agora project app ID.             |

#### Response example

**Request successful**

- Status code: 200 OK
- Content type: `application/json`

```json
{
  "token":"006d2776fef40dc864dc7a438e3b871IACGnkorQsr0iCpBmzNwKEdzKuAv1b1zRcMy0cOradw6mHN/il8AAAAAIgCQqbAFeaVzZgQAAQDw0oBmAgDw0oBmAwDw0oBmBADw0oBm",
  "uid":"123",
  "channel":"test",
  "appid":"fs9f52d9dcc1f406b93d97ff1f43c554f",
}
```

**Error code responses**

| Error code | Description                     |
|------------|---------------------------------|
| `404`      | Not found                       |

## End call

Use the call ID returned by the outbound call API to terminate an outbound call.

**Prototype**

- Method: `POST`
- Endpoint: `https://sipcm.agora.io/v1/api/pstn`

#### Request parameters

**Request body**

| Parameter  | Data type | Description                        |
|------------|-----------|------------------------------------|
| `appid`    | String    | The phone number dialed.           |
| `callid`   | String    | The pin entered.                   |

#### Request example

```json
{
  "action":"endcall",
  "appid":"fs9f52d9dcc1f406b93d97ff1f43c554f",
  "callid":"f577605c-eb3a-4efe-af1b-ee66d5297569",
}
```

#### Response example

**Request successful**

- Status code: 200 OK
- Content type: `application/json`

```json
{
  "success":true
}
```

**Error code responses**

| Error code | Description                     |
|------------|---------------------------------|
| `404`      | Not found                       |
