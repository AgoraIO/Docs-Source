You call the `enableContentInspect` method to enable the video screenshot upload function, and set the `inspectType` in `InspectConfiguration` to `supervise`. After the video screenshot upload function is successfully enabled, your <Vpl k="CLIENT"/> takes screenshots according to the frequency specified by the `interval` and uploads the screenshots to your third-party storage. The sample code is as follows:

```js
// Create an InspectConfiguration object and set the relevant parameters
const inspectConfig = {
  // Set the video screenshot interval to 2000 milliseconds (2 seconds)
  interval: 2000,
    // Set the function type to "supervise"
  inspectType: ['supervise'],
  // Set the address prefix of third-party cloud storage
  ossFilePrefix: 'https://your-cloud-storage.com/',
  // Add any additional information
  extraInfo: 'This is some additional information for content inspection.'
};

 
// Call the enableContentInspect method and pass in the inspectConfig object
client.enableContentInspect(inspectConfig)
  .then(() => {
    console.log('Content inspection enabled successfully.');
  })
  .catch((error) => {
    console.error('Failed to enable content inspection:', error);
  });
```
