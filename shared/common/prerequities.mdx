<PlatformWrapper platform="android">
- Android Studio 4.1 or higher.
- Android SDK API Level 24 or higher.
- A mobile device that runs Android 4.1 or higher.
</PlatformWrapper>
<PlatformWrapper platform="ios">
- Xcode 12.0 or higher.
- Two devices running iOS 9.0 or higher.
</PlatformWrapper>
<PlatformWrapper platform="macos">
- Xcode (the latest version recommended).
- An Apple device with macOS 10.0 or higher.
- CocoaPods 1.10.1 or higher.
</PlatformWrapper>
<PlatformWrapper platform="linux">
- Ubuntu 14.04 LTS (64 bit) or later, CentOS 7.0 (64 bit) or later.
- CMake 2.8 or later.
- glibc 2.14 or later (glibc 2.25 or later if you use Agora RTM SDK 1.4.2).
- C++ 11 or later.
</PlatformWrapper>
<PlatformWrapper platform="unity">
- Unity 2017 or later

    Unity 2020.3 or later has compatibility issues on Mac devices. For macOS app development, Agora recommends not using Unity 2020.3 or later.
- Android, iOS, Windows, or MacOS devices or emulators.
</PlatformWrapper>
<PlatformWrapper platform="windows">
- Windows 7 or higher.
- Visual Studio 2017 or higher.
- C++ 11 or higher.
</PlatformWrapper>
<PlatformWrapper platform="web">
- A [supported browser](../reference/supported-platforms#browsers).
- Physical media input devices, such as a camera and a microphone.
- A JavaScript package manager such as [npm](https://www.npmjs.com/package/npm).
</PlatformWrapper>
- A computer that can access the Internet.
  Ensure that no firewall is blocking your network communication, otherwise, the project may fail.
- An <Vg k="COMPANY"/> [account](../reference/manage-agora-account#create-an-agora-account) and [project](../reference/manage-agora-account#create-an-agora-project).
