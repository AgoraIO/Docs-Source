```python
def send_encoded_audio_frame(self, buffer_ptr:int, buffer_size:int, frame:EncodedAudioFrame)
```

<span className="index-desc-sendencodedaudioframe">Sends an encoded audio frame to the local audio track.</span>

#### Parameters

| Parameter       | Description                                                                 |
|------------------|----------------------------------------------------------------------|
| `buffer_ptr`   | The encoded audio data to send.                                               |
| `buffer_size` | The size of the data (bytes).   |
| `frame` | The audio frame information, see [`EncodedAudioFrame`](./apidatatype#encodedaudioframe).   |

#### Returns

- 0: Success.
- < 0: Failure.
