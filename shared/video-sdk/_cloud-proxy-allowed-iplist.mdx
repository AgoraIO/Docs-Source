
Agora deploys dedicated servers to run the cloud proxy service, of which Agora owns the IP addresses and fully controls the traffic on these addresses. To use Agora Cloud Proxy Force UDP or Force TCP modes, your end users must first configure their firewalls to trust the IP address and port ranges.

This page provides the Allowed IP List for the RTC Native SDK and the RTC Web SDK. Refer to the list according to the SDK version and the cloud proxy mode you use.

## Allowed IP List for RTC Native SDK v3.7.0 and later

<div class="alert info">When adding the IP addresses, note the following: <ul><li>If you use cloud proxy with <Link to ="../develop/geofencing">network geofencing</Link >, you can add only the IP addesses dedicated to your specified region.</li><li>If you do not enable network geofencing, you need to add all IP addresses to ensure connectivity.</li></ul></div>
