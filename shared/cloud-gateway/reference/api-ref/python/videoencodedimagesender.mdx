The `VideoEncodedImageSender` class.

### `send_encoded_video_image`

Sends an encoded video frame to the video track.

```python
def send_encoded_video_image(self, buffer_ptr:int, buffer_size:int, frame_info:EncodedVideoFrameInfo)
```

#### Parameters

| Parameter         | Description                                                                                       |
|--------------|--------------------------------------------------------------------------------------------|
| `buffer_ptr` | The video buffer.                                                                            |
| `buffer_size`| The size of the video data (bytes).                                                                      |
| `frame_info`      | The encoded video frame information: see [`EncodedVideoFrameInfo`](#encodedvideoframeinfo).                   |

#### Returns

- 0: Success.
- < 0: Failure.
