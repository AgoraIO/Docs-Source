<PlatformWrapper platform="flutter">

1. Connect the target device to your computer.

1. Open terminal and navigate to the `/examples` directory.

1. Execute the following command to launch the sample project on available devices:

    ```bash
    flutter run
    ```

1. After compilation is complete, you see the following application on your device:

    ![](https://agora-doc.s3.us-east-1.amazonaws.com/images/video-sdk/run-compile-flutter-step-4.png)


1. Select a demo you want to experience from the menu. Choose **JoinChannelVideo** under **Basic** as an example. Enter the channel name you specified when generating the temporary token in **Channel ID**, and click **Join channel**.

1. To test various audio and video interaction use-cases, connect to the [Agora web demo](https://webdemo.agora.io/basicVideoCall/index.html), or install and run the sample project on a second device. Make sure you use the same app ID on both devices. When you join the same channel from two devices, you can see and hear each other.

    ![](https://agora-doc.s3.us-east-1.amazonaws.com/images/video-sdk/run-compile-flutter-step-6.png)
