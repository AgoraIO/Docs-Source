        ```javascript
        agoraEngine.adjustPlaybackSignalVolume(parseInt(evt.target.value));
        agoraEngine.adjustUserPlaybackSignalVolume(remoteUid,parseInt(evt.target.value));
        agoraEngine.adjustAudioMixingVolume(parseInt(evt.target.value));
        agoraEngine.adjustAudioMixingPlayoutVolume(parseInt(evt.target.value));
        agoraEngine.adjustAudioMixingPublishVolume(parseInt(evt.target.value));
        ```

    d. To test other mute methods, locate the `document.getElementById("mute").onclick` event listener and replace the `muteRemoteAudioStream` call with one of the following:

        ```javascript
        agoraEngine.muteAllRemoteAudioStreams(isMuted);
        agoraEngine.muteLocalAudioStream(isMuted);
        ```
        Run the app again and tap **Mute** to test the effect of these mute methods.
