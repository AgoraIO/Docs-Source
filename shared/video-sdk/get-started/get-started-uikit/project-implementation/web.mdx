When the user taps a button, the <Vpl k="CLIENT" /> joins or leaves a channel. When another user joins the same channel, their video and audio is rendered in the <Vpl k="CLIENT" />. This simple workflow enables you to concentrate on implementing <Vg k="COMPANY" /> features and not UX bells and whistles.

To integrate real-time video with a ready-made user interface into your <Vpl k="CLIENT" /> using <Vg k="UIK" />:

<CH.Scrollycoding>

### Import the AgoraUIKit component from the `agora-react-uikit` package

Open the file `<app-name>/src/App.ts` and replace the contents with the provided code sample:

```ts app.ts focus=1:2
import React, { useState } from "react";
import AgoraUIKit from "agora-react-uikit";

const App = () => {
  const [videoCall, setVideoCall] = useState(true);

  return videoCall ? (
    <div style={{ display: "flex", width: "100vw", height: "100vh" }}>
      {/* Display Videocall here */}
    </div>
  ) : (
    <h3 onClick={() => setVideoCall(true)}>Join</h3>
  );
};

export default App;
```

---

### Create the App component and declare the variables

To connect to and disconnect from a channel, the `App` component uses the `videoCall` state variable. To join a call your CLIENT sets `videoCall` to `true`, passes the Agora `appID`, `channel` and `token` to a high-level `AgoraUIKit` component that handles the connection to the channel. To set the variables for this logic, add the provided code sample to `<app-name>/src/App.ts` after the `import` statements:

```ts app.ts focus=6:10
import React, { useState } from "react";
import AgoraUIKit from "agora-react-uikit";

const App = () => {
  const [videoCall, setVideoCall] = useState(true);

  return videoCall ? (
    <div style={{ display: "flex", width: "100vw", height: "100vh" }}>
      <AgoraUIKit
        rtcProps={{
          appId: "<Agora App ID>",
          channel: "<Channel Name>",
          token: "<Your Channel Token>",
        }}
      />
    </div>
  ) : (
    <h3 onClick={() => setVideoCall(true)}>Join</h3>
  );
};

export default App;
```

---

### Setup callbacks for <Vg k="UIK" /> events

The `callbacks` object contains <Vg k="ENGINE" /> events as keys and callback functions as their values. You use the `EndCall` event to update the state and unmount the `AgoraUIKit` component.

To do this add the provided code sample to `<app-name>/src/App.ts` after the `rtcProps``:

```ts app.ts focus=15:15
import React, { useState } from "react";
import AgoraUIKit from "agora-react-uikit";

const App = () => {
  const [videoCall, setVideoCall] = useState(true);

  return videoCall ? (
    <div style={{ display: "flex", width: "100vw", height: "100vh" }}>
      <AgoraUIKit
        rtcProps={{
          appId: "<Agora App ID>",
          channel: "<Channel Name>",
          token: "<Your Channel Token>",
        }}
        callbacks={{ EndCall: () => setVideoCall(false) }}
      />
    </div>
  ) : (
    <h3 onClick={() => setVideoCall(true)}>Join</h3>
  );
};

export default App;
```

---

### Display the user interface

That's the code for the UIKit, if `videoCall` is `true` you'll render the `AgoraUIKit` component. If it is `false` we show a _Join_ button.

```ts app.ts focus=1:26
import React, { useState } from "react";
import AgoraUIKit from "agora-react-uikit";

const App = () => {
  const [videoCall, setVideoCall] = useState(true);

  const callbacks = {
    EndCall: () => setVideoCall(false),
  };

  const rtcProps = {
    appId: "<Agora App ID>",
    channel: "<Channel Name>",
    token: "<Your Channel Token>",
  };

  return videoCall ? (
    <div style={{ display: "flex", width: "100vw", height: "100vh" }}>
      <AgoraUIKit rtcProps={rtcProps} callbacks={callbacks} />
    </div>
  ) : (
    <h3 onClick={() => setVideoCall(true)}>Join</h3>
  );
};

export default App;
```
