## Integrate the SDK

In the terminal, go to your project's root directory and run the following command to integrate the SDK:

```bash
pip install agora_python_server_sdk
```

## Get the sample project

You can [download the sample project](https://github.com/AgoraIO-Extensions/Agora-Python-Server-SDK/tree/release/2.1.0/agora_rtc/examples) to try it out.
