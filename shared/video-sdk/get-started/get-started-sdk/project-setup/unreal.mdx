1. **Create a new project**:

    1. In Unreal Editor, select the **Games** new project category.

    1. From **Unreal Project Browser**, select a **Blank template** and choose the following settings from **Project Defaults**.

        1. Select **C++**.

        2. From the **Target Platform** dropdown, select **Desktop**.

        3. Make sure the **Starter Content** check box is unchecked.

        4. In the **Project Name** field, input `AgoraImplementation` and click **Next**.

            You see your project opens in the Unreal Editor and the source code of your project opens in the Visual Studio.

1. **Integrate <Vpd k="SDK" />**:

    1. Go to [SDKs](/sdks), download the latest version of the Agora <Vpd k="SDK" />, and unzip the downloaded SDK to your project folder.

    2. In Solution Explorer, right-click your project, then click **Properties**. The **agoraBlueprint Property Pages** window opens. Go to the **VC++ Directories** menu and add the following string in the **External Include Directories** field, then click **OK**:

        ```
        $(SolutionDir)Plugins\AgoraPlugin\Source\AgoraPlugin\Public;$(SolutionDir)Plugins\AgoraPlugin\Source\AgoraPlugin\Private\Include;
        ```
