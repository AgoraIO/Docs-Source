The Hongtu Portrait 3D plug-in is an encapsulation of the Hongtu HTAvatar core API. Through the setExtensionProperty or setExtensionPropertyWithVendor method provided by Shengwang SDK v4.x and passing in the specified and parameters, you can quickly integrate Hongtu's portrait 3D special effects capabilities.

This article introduces how to integrate and use Hongtu Portrait 3D special effects series (hereinafter referred to as "Hongtu Portrait 3D plug-in") in your project, including Android and iOS platforms.

## Understand the tech

Take `setExtensionProperty` for example , the `key` parameter corresponds exactly to the API name of Hongtu, and the `value` parameters wrap some or all parameters of the API in JSON format. Therefore, `setExtensionProperty` as long as the specified `key` and `value` are passed in when calling, the corresponding Hongtu API can be called to realize the related functions of portrait 3D special effects. `setExtensionPropertyWithVendor` the same way.
