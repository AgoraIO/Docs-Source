In chat apps, a conversation is composed of all the messages in a peer-to-peer chat, chat group, or chatroom. The <Vg k="CHAT_SDK"/> supports managing messages by conversations, including retrieving and managing unread messages, deleting the historical messages on the local device, and searching historical messages.

This page introduces how to use the <Vg k="CHAT_SDK"/> to implement these functionalities.
