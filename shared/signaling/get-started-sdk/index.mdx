Use <Vpd k="SDK" /> to add low-latency, high-concurrency signaling and
synchronization capabilities for real-time systems to your <Vpl k="CLIENT" />. <Vg k="SIG" /> also helps you enhance the user experience in <Vg k="VIDEO" />, <Vg k="AUDIO" />,  <Vg k="ILS" />, and <Vg k="BS" /> applications.

This page shows you how to use the <Vpd k="SDK" /> to rapidly build a simple application that sends and receives messages. It shows you how to integrate the <Vpd k="SDK" /> in your project and implement pub/sub messaging through [Message channels]((../core-functionality/message-channel)). To get started with stream channels, follow this guide to create a basic <Vg k="SIG" /> app and then refer to the [Stream channels](../core-functionality/stream-channel) guide.

## Understand the tech

To use <Vg k="SIG" /> features in your <Vpl k="CLIENT" />, you initialize a <Vg k="SIG" /> client instance and add event listeners. To connect to <Vg k="SIG" />, you login using an authentication token. To send a message to a message channel, you publish the message. <Vg k="SIG" /> creates a channel when a user subscribes to it. To receive messages other users publish to a channel, your <Vpl k="CLIENT" /> listens for events.
