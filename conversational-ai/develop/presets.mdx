Presets are predefined configurations managed by Agora that let you use supported ASR, LLM, and TTS providers without supplying your own API keys. You can use a preset for any or all of the three provider categories.

## Prerequisites

- An Agora account with [Conversational AI Engine enabled](/conversational-ai/reference/enable-conversational-ai)
- A valid Agora app ID and app certificate
- For Curl Only: Customer ID and customer secret to use [Basic HTTP authentication](/conversational-ai/rest-api/restful-authentication#implement-basic-http-authentication)
  or an RTC token to use [token authentication](/conversational-ai/rest-api/restful-authentication#implement-token-authentication).

## Understand the tech

When a preset is specified for a provider category, the SDK or API uses Agora-managed configuration for that category. This means that no endpoint URL, API key, or model name is required. You can still use the `asr`, `llm`, and `tts` fields to supply additional settings such as a system prompt or a specific voice ID.

## Start an agent with presets

The following example starts an agent using Deepgram for ASR, OpenAI for LLM,
and MiniMax for TTS, with no BYOK credentials required for any provider.

<Tabs groupId="language">
<TabItem value="python" label="Python SDK">

```python
from agora_agent import Agent, Agora, Area, DeepgramSTT, OpenAI, MiniMaxTTS

client = Agora(
    area=Area.US,
    app_id='your-app-id',
    app_certificate='your-app-certificate',
)

# Omit API keys to use Agora-managed models for ASR, LLM, and TTS.
agent = (
    Agent(name='my-preset-agent')
    .with_stt(DeepgramSTT(model='nova-3', language='en-US'))
    .with_llm(OpenAI(
        model='gpt-4o-mini',
        system_messages=[{'role': 'system', 'content': 'You are a helpful assistant.'}],
        greeting_message='Hello! How can I help you today?',
        failure_message='Sorry, I could not process your request.',
        max_history=32,
    ))
    .with_tts(MiniMaxTTS(
        model='speech_2_6_turbo',
        voice_id='English_captivating_female1',
    ))
)

session = agent.create_session(
    client,
    channel='my-channel',
    agent_uid='1001',
    remote_uids=['1002'],
    idle_timeout=120,
)

agent_id = session.start()
print(f'Agent started: {agent_id}')
```

<TabItem value="go" label="Go SDK">

```go
package main

import (
    "context"
    "fmt"
    "log"

    "github.com/AgoraIO/agora-agents-go/v2/agentkit"
    "github.com/AgoraIO/agora-agents-go/v2/agentkit/vendors"
    "github.com/AgoraIO/agora-agents-go/v2/option"
)

func main() {
    ctx := context.Background()
    idleTimeout := 120

    client := agentkit.NewAgoraClient(agentkit.AgoraClientOptions{
        Area:           option.AreaUS,
        AppID:          "your-app-id",
        AppCertificate: "your-app-certificate",
    })

    // Omit API keys to use Agora-managed models for ASR, LLM, and TTS.
    agent := agentkit.NewAgent(
        agentkit.WithName("my-preset-agent"),
    ).WithStt(
        vendors.NewDeepgramSTT(vendors.DeepgramSTTOptions{
            Model:    "nova-3",
            Language: "en-US",
        }),
    ).WithLlm(
        vendors.NewOpenAI(vendors.OpenAIOptions{
            Model: "gpt-4o-mini",
            SystemMessages: []map[string]interface{}{
                {"role": "system", "content": "You are a helpful assistant."},
            },
            GreetingMessage: "Hello! How can I help you today?",
            FailureMessage:  "Sorry, I could not process your request.",
            MaxHistory:      intPtr(32),
        }),
    ).WithTts(
        vendors.NewMiniMaxTTS(vendors.MiniMaxTTSOptions{
            Model:   "speech_2_6_turbo",
            VoiceID: "English_captivating_female1",
        }),
    )

    session := agent.CreateSession(client, agentkit.CreateSessionOptions{
        Channel:     "my-channel",
        AgentUID:    "1001",
        RemoteUIDs:  []string{"1002"},
        IdleTimeout: &idleTimeout,
    })

    agentID, err := session.Start(ctx)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println("Agent started:", agentID)
}
```

<TabItem value="curl" label="Curl">

Pass the preset identifiers as a comma-separated string in the [`preset`](/conversational-ai/rest-api/agent/join#preset) field. You can specify one preset per category, and categories without a preset require your own provider credentials.

```bash
curl --request POST \
  --url https://api.agora.io/api/conversational-ai-agent/v2/projects/<your_app_id>/join \
  --header 'Authorization: agora token=<your_agora_token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "my-preset-agent",
    "preset": "deepgram_nova_3,openai_gpt_5_mini,minimax_speech_2_6_turbo",
    "properties": {
      "channel": "my-channel",
      "token": "<your_rtc_token>",
      "agent_rtc_uid": "1001",
      "remote_rtc_uids": ["1002"],
      "idle_timeout": 120,
      "llm": {
        "system_messages": [
          {
            "role": "system",
            "content": "You are a helpful assistant."
          }
        ],
        "greeting_message": "Hello! How can I help you today?",
        "failure_message": "Sorry, I could not process your request.",
        "max_history": 32
      },
      "tts": {
        "params": {
          "voice_setting": {
            "voice_id": "English_captivating_female1"
          }
        }
      },
      "asr": {
        "params": {
          "language": "en",
          "keyterm": "agora"
        }
      }
    }
  }'
```

In this example:

- The `preset` field configures Deepgram Nova 3 for ASR, GPT-5 Mini for LLM,
  and MiniMax Speech 2.6 Turbo for TTS.
- The `llm` field sets the system prompt, greeting, and history length without
  specifying an endpoint or API key, because those are provided by the preset.
- The `tts` field overrides the default voice selection for MiniMax.
- The `asr` field passes a keyword hint to Deepgram.

## Reference

- For a complete list of available presets, see [`preset`](/conversational-ai/rest-api/agent/join#preset).

    You can specify one preset per category. If you specify a preset for a
category, any API key or endpoint URL fields for that category are ignored.
- For the full list of request parameters, see [Start a conversational AI agent](../rest-api/agent/join).

- For provider-specific configuration options available within [`asr`](../models/asr/overview), [`llm`](../models/llm/overview), and [`tts`](../models/tts/overview) fields when using presets, see the relevant provider page.
