This class is used to create media components.

## `createMediaRtcRecorder`

```cpp
virtual agora_refptr<IAgoraMediaRtcRecorder> createMediaRtcRecorder() = 0;
````

Creates an [`IAgoraMediaRtcRecorder`](agora-media-rtc-recorder) instance for media recording.

#### Return value

* If the method call succeeds, returns a valid [`IAgoraMediaRtcRecorder`](agora-media-rtc-recorder) object.
* If the method call fails, returns a null pointer.

## `createAgoraMediaComponentFactory`

```cpp
AGORA_API agora::rtc::IMediaComponentFactory* AGORA_CALL createAgoraMediaComponentFactory();
```

Creates and returns a pointer to an `IMediaComponentFactory` object.

#### Return value

* If the method call succeeds, returns a pointer to an `IMediaComponentFactory` object.
* If the method call fails, returns a null pointer.


## `release`

Releases native resources associated with the factory.

```java
public int release()
```

#### Returns

* `0`: The method call was successful.
* `< 0`: The method call failed.
