### End call

Use the **callid** returned by the outbound call API to terminate an outbound call.

- URL: https://sipcm.agora.io/v1/api/pstn
- Method: POST

#### Request body parameters as JSON

```json
{
  "action":"endcall",
  "appid":"fs9f52d9dcc1f406b93d97ff1f43c554f",
  "callid":"f577605c-eb3a-4efe-af1b-ee66d5297569",
}
```

- **appid**(string): <Vg k="COMPANY" /> project app ID
- **callid**(string): Call ID of the ongoing call

#### Success response example

- **Status Code**: 200 OK
- **Content Type**: application/json
- **Body**:

   ```json
   {  
     "success":true
   }
   ```
  
#### Error code responses

- **404**: Not found

This API allows you to stop an outbound call.

### Cancel call

Pass the details returned by an inbound PSTN or SIP API call request to cancel it. If the call is in progress it will be stopped.

- URL: https://sipcm.agora.io/v1/api/pstn
- Method: POST

#### Request body parameters as JSON

```json
{
  "action":"cancelcall",
  "appid":"fs9f52d9dcc1f406b93d97ff1f43c554f",
  "sip":"sip:pstn_783410988998@2.4.6.7:5080",
  "did":"17377583411",
  "pin":"445566",
}
```

- **appid**(string): <Vg k="COMPANY" /> project app ID
- **sip**(string) [optional]: SIP address returned by inbound SIP API
- **did**(string) [optional]: did returned by the inbound API
- **pin**(string) [optional]: pin returned by the inbound API

#### Success response example

- **Status Code**: 200 OK
- **Content Type**: application/json
- **Body**:
   ```json
   {
     "success": true
   }
   ```

#### Error code responses

- **404**: Not found

This API allows you to cancel a previous call setup request.
