1. **Enable the plugin**

    When initializing `AgoraRtcEngineKit`, call `enableExtensionWithVendor` to enable the extension.

    ```c
    [self.agoraKit enableExtensionWithVendor:@"Texeljoy"
                                extension:@"HTAvatar"
                                    enabled:self.enable];
    ```

2. **Initialize the plugin**

    To initialize the plug-in, call `setExtensionPropertyWithVendor` and pass in the corresponding `key` and `value` parameters. 

    ```c
    - (void)initExtension{
        
        // resource copy
        [self.agoraKit setExtensionPropertyWithVendor:@"Texeljoy" 
                                            extension:@"HTAvatar" 
                                                key:@"htARCopyResource" 
                                                value:[self toJson:@{
            @"path":[[NSBundle mainBundle] pathForResource:@"HTEffect" ofType:@"bundle"]
        }]];

        // Offline authentication method
        [self.agoraKit setExtensionPropertyWithVendor:@"Texeljoy"
                                            extension:@"HTAvatar"
                                                key:@"htARInitOffline"
                                                value:[self toJson:@{
                                                    @"license":HTSDKKey
                                                }]];
        
        // (optional) set the image format
        [self.agoraKit setExtensionPropertyWithVendor:@"Texeljoy"
                                            extension:@"HTAvatar"
                                                key:@"htARImageFormat"
                                                value:[self toJson:@{
                                                    @"imageFormat":@5
                                                }]];
    }
    ```

3. **Set up a 3D effect**

    To implement a 3D effect, call `setExtensionPropertyWithVendor` and pass in the corresponding `key` and `value`. The specific sequence is as follows:

    1. Pass in the key to `htARRenderEnable` to enable AR special effects.
    1. Pass in a key to `htARSetModel` to set the AR model.

    You can combine multiple calls according to the needs of your scene. For the complete `key` and `value` descriptions, refer to the [Interface description](#interface-description).
