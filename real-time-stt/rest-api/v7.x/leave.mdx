Use this method to leave the channel and stop <Vpd k="NAME" />.

## Request

### Path parameters

<PathParameter name="appId" type="string" required={true}>
The App ID of the project
</PathParameter>
<PathParameter name="agentId" type="string" required={true}>
The unique identifier of the agent you received in the response body of the [`join`](../v7.x/join) method.
</PathParameter>

## Response

- If the returned status code is `200`, the request was successful. 
- If the returned status code is not `200`, the request failed.
