<TabItem value="new" label="Create a new project" default>

To initialize a new Vite project, take the following steps:

1. Open a terminal and run the following command:

    ```
    npm create vite@latest agora_web_quickstart -- --template vanilla
    ```

    This creates a new folder named `agora_web_quickstart` and initialize a Vite project inside it using the `vanilla` JavaScript template.

1. Navigate to the newly created folder. Download and set up dependencies for your project:

    ```
    cd agora_web_quickstart
    npm install
    ```

 1. To add `AgoraRTCClient` code that implements specific application logic, create a file named `agoraLogic.js` in your project's `src` folder.

 1. To create a user interface, add an `index.html` file to your project. The UI consists of buttons to join and leave a channel. Refer to [Create a user interface](#create-a-user-interface) to get a bare bones html layout.

 1. Include the `agoraLogic.js` file in your html file.

    <CodeBlock language="html">
    {`<script type="module" src="/src/agoraLogic.js"></script>`}
    </CodeBlock>

</TabItem>
