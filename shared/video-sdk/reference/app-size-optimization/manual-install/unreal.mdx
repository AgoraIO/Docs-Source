To manually install <Vpd k="SDK" />:

1. Create a folder called `Plugins` in the root directory of your project folder.

1. Unzip the latest version of the [<Vpd k="SDK" />](/sdks) to `<Project Folder>/Plugins`.

1. In Solution Explorer, right-click your project, then click **Properties**. The **`<Project Name>` Property Pages** window opens. Go to the **VC++ Directories** menu and add the following string in the **External Include Directories** field, then click **OK**:

    ```
    $(SolutionDir)Plugins\AgoraPlugin\Source\AgoraPlugin\Public;
    ```
