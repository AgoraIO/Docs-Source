1.  Generate a temporary token in Agora Console.

2. In your browser, navigate to the <Link target="_blank" to="{{Global.DEMO_BASIC_VIDEO_CALL_URL}}"><Vg k="COMPANY" /> web demo</Link> and update _App ID_, _Channel_, and _Token_ with the values for your temporary token, then click **Join**.

3.  In the `ViewController`, update `appID`, `channelName`, and `token` with the values from <Vg k="CONSOLE" />.

4.  Run your <Vpl k="CLIENT" /> using either a physical or a simulator iOS device.

    If this is the first time you run the project, grant microphone and camera access to your <Vpl k="CLIENT" />.

    If you use an iOS simulator, you see the remote video only. You cannot see the local video stream because of [Apple simulator hardware restrictions](https://help.apple.com/simulator/mac/current/#/devb0244142d).

5. Select **Broadcaster** mode. 
</ProductWrapper>

1.  Press **Join** to connect to the same channel as your web demo.

2.  Test processing of raw video data.

    Edit the `AgoraVideoFrameDelegate` definition to add code that processes the raw video data you receive in the following callbacks:
 
    -   `onCapture`: Gets video frame data captured by the local camera

    -   `getVideoFrameProcessMode`: Gets the video frame mode of the observer

    -   `getVideoFormatPreference`: Sets the video frame type preference

    -   `getObservedFramePosition`: Sets the frame position for the video observer

3.  Test processing of raw audio data.

    Edit the `AgoraAudioFrameDelegate` definition to add code that processes the raw audio data you receive in the following callbacks

    -   `onRecord`: Gets the captured audio frame data

    -   `onPlaybackAudioFrame`: Gets the audio frame for playback
