Take the following steps to test the sample code:

1. Update the `APPID`, `token`, and `channel` parameter values in your code. 

1. In the project root directory run the following command in Terminal:

    ```bash
    npm start
    ```

    After the project runs successfully, you see a pop-up window.

1. Invite a friend to run the demo app on a second device. Alternatively, use the [Web demo](https://webdemo.agora.io/basicVideoCall/index.html) to join the same channel.
