## v1.4.8
v1.4.8 was released on August 24, 2021.

This release is the first release of RTM Unity SDK. You can refer to the following documents to integrate the SDK and implement <Vg k="SIG"/> sending and receiving functions:

- [Peer-to-peer or Channel Messaging](https://docs.agora.io/en/Real-time-Messaging/start_messaging_unity?platform=Unity)

- [API Reference](https://docs.agora.io/en/Real-time-Messaging/API%20Reference/RTM_unity/index.html)

Agora provides an open-source sample project [Agora-Unity-RTM-SDK](https://github.com/AgoraIO-Community/Agora-Unity-RTM-SDK/) , you can download and experience it.
