## Overview

Agora Notification Center Service (NCS) is a tool that enables you to subscribe to events across multiple product lines in one place.

### Releases

### 2022.10 Initial release

* After you enable the NCS service, your server receives the events that you subscribe to in the form of HTTPS requests.
* To improve communication security between the NCS server and your server, Agora uses signatures for identity verification.
* As of this release, you can use NCS in conjunction with the RTC products.
