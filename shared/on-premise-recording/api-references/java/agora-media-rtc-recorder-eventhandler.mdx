The `IAgoraMediaRtcRecorderEventHandler` interface defines callback methods that are triggered during various RTC recorder events, such as changes in connection state, user activity, and statistics updates.

### `onConnected`

Occurs when the SDK connects to the channel.

```java
default void onConnected(String channelId, String userId)
```

| Parameter | Type   | Description |
| --------- | ------ | ----------- |
| channelId | String | Channel ID. |
| userId    | String | User ID.    |

### `onDisconnected`

Occurs when the SDK disconnects from the channel.

```java
default void onDisconnected(String channelId, String userId, Constants.ConnectionChangedReasonType reason)
```

| Parameter | Type                                  | Description                   |
| --------- | ------------------------------------- | ----------------------------- |
| channelId | String                                | Channel ID.                   |
| userId    | String                                | User ID.                      |
| reason    | Constants.ConnectionChangedReasonType | Reason for connection change. |

### `onReconnected`

Occurs when the SDK reconnects to the channel.

```java
default void onReconnected(String channelId, String userId, Constants.ConnectionChangedReasonType reason)
```

| Parameter | Type                                  | Description              |
| --------- | ------------------------------------- | ------------------------ |
| channelId | String                                | Channel ID.              |
| userId    | String                                | User ID.                 |
| reason    | Constants.ConnectionChangedReasonType | Reason for reconnection. |

### `onConnectionLost`

Occurs when the SDK loses connection with the channel.

```java
default void onConnectionLost(String channelId, String userId)
```

| Parameter | Type   | Description |
| --------- | ------ | ----------- |
| channelId | String | Channel ID. |
| userId    | String | User ID.    |

default void onUserJoined(String channelId, String userId)

| --------- | ------ | --------------- |
| channelId | String | Channel ID.     |

| Parameter | Type                            | Description              |
| --------- | ------------------------------- | ------------------------ |

| reason    | Constants.UserOfflineReasonType | Reason for user leaving. |

### `onFirstRemoteVideoDecoded`

Occurs when the first frame of remote video is decoded.

```java
default void onFirstRemoteVideoDecoded(String channelId, String userId, int width, int height, int elapsed)
```

| Parameter | Type   | Description                          |
| --------- | ------ | ------------------------------------ |
| channelId | String | Channel ID.                          |
| userId    | String | Remote user ID.                      |
| width     | int    | Width of the video stream (pixels).  |
| height    | int    | Height of the video stream (pixels). |
| elapsed   | int    | Time since user joined (ms).         |
