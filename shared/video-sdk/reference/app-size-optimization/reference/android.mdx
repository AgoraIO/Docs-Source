### Plugin list

### Dependency `implementation`
When integrating the <Vg k="VSDK" /> through Maven Central, you can modify the `dependencies` in the `/Gradle Scripts/build.gradle(Module: <projectname>.app)` file to specify the dynamic libraries that you need to integrate. The relationship between each file and the required `implementation` is detailed in the table below:
