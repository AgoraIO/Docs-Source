1.  **Create an extension for screen sharing**

    iOS uses a separate target for screen sharing that uses `ReplayKit`. You use a new instance of `AgoraRtcEngineKit` to stream to the same channel your user is already connected to. To implement this:

    1. Create a `screenSharer` target with a `SampleHandler` swift file holding a <Vg k="VSDK" /> engine instance as follows:

        <ScreenSharerTarget />

    1. Override [`RPBroadcastSampleHandler.broadcastStarted`](https://developer.apple.com/documentation/replaykit/rpbroadcastsamplehandler/2143170-broadcaststarted) inside `SampleHandler` to stream the screen capture video frames to the channel:

        <OverrideBroadcastStarted />

    For more details, have a look at [`SampleHandler`](https://github.com/AgoraIO/video-sdk-samples-ios/blob/main/Example-App/screenSharer/SampleHandler.swift).
