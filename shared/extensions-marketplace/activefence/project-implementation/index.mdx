Now that  <Vg k="RTEE_AF" /> is activated, to check content sent in the channels in your <Vpl k="CLIENT" />:

1. **Setup your webhook**

      In order to react to calls made by <Vg k="RTEE_AF" /> for moderated content, your need to create a server that
    received and interprets the data sent to you in the webhook you [defined in <Vg k="RTEE_AF" /> ](#define-webhook).
    For example:

    1. In your browser, open [agora-activefence-kicker](https://github.com/AgoraIO-Community/agora-activefence-kicker)
    and use the one-click deployment for the server
    2. Add the deployment link + `/kick/` to the webhook URL for the desired event in <Vg k="RTEE_AF_D" /> .
