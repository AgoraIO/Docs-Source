PlayAI provides fast, reliable text-to-speech (TTS) with customizable voices.

### Sample configuration

The following example shows a starting `tts` parameter configuration you can use when you [Start a conversational AI agent](/conversational-ai/rest-api/agent/join).

```json
"tts": {
    "params": {
        "api_key": "<PLAYHT_TTS_SECRET_KEY>",
        "user_id": "<PLAYHT_TTS_USER_ID>",
        "voice_engine": "Play3.0-mini",
        "voice": "alex",
        "speed": 1.0
    }
}
```

### Key parameters

### Key parameters

<ParameterList title="params" required={true}>
  <Parameter name="api_key" type="string" required={true}>  
  The API key used for authentication. Get your API key from the [PlayHT Dashboard](https://play.ht/studio/api-access).
  </Parameter>
  <Parameter name="user_id" type="string" required={true}>  
  The user ID associated with your PlayHT account. Get your user ID from the [PlayHT Dashboard](https://play.ht/studio/api-access).
  </Parameter>
  <Parameter name="voice_engine" type="string" required={true}>  
  The voice engine to use for speech synthesis. Supported values: `PlayDialog`, `Play3.0-mini`, `PlayHT2.0-turbo`.
  </Parameter>
  <Parameter name="voice" type="string" required={true}>  
  The voice ID to use for speech generation, for example, `alex`. See [available voices](https://docs.play.ht/reference/api-list-voices) for supported voice IDs.
  </Parameter>
  <Parameter name="speed" type="number" required={false}>  
  The speed of speech, where `1.0` is the normal speed. Values less than `1.0` slow down the speech, while values greater than `1.0` speed it up.
  </Parameter>
</ParameterList>
