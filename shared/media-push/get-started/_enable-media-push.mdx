To enable <Vg k="MPUSH" /> in <Vg k="CONSOLE" />:

1. Log in to <Link to="{{Global.AGORA_CONSOLE_URL}}"><Vg k="CONSOLE" /></Link>, and click the **Project Management** icon on the left navigation panel.

2. On the **Project Management** page, find the project for which you want to enable the <Vg k="MPUSH" /> service, and click the edit icon.

    <img src="https://web-cdn.agora.io/docs-files/1637660034129"/>

3. Under **Real-time engagement extensions**, find **<Vg k="MPUSH" />**, and click **Enable**.

    <img src="https://web-cdn.agora.io/docs-files/1661509060968"/>

4. Click the **Status** button to enable the <Vg k="MPUSH" /> **RESTful API** service. Read the pop-up prompt carefully, and then click **Save**. <Admonition type="info">The service cannot be disabled after enabling. </Admonition>

    ![media-push-client-side-api](https://agora-doc.s3.us-east-1.amazonaws.com/images/media-push/media-push-client-side-api.png)

5. The **Enable** button of **<Vg k="MPUSH" />** changes to the **Config** button, which allows you to configure the <Vg k="MPUSH" /> service.
    <img src="https://web-cdn.agora.io/docs-files/1661509305442"/>
