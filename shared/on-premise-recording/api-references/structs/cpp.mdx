### `AgoraServiceConfiguration`

Configures global parameters for [`IAgoraService`](#iagoraservice). Pass this struct when calling [`initialize`](#initialize).

```cpp
struct AgoraServiceConfiguration {
  bool enableAudioProcessor;
  bool enableAudioDevice;
  bool enableVideo;
  void* context;
  const char* appId;
  unsigned int areaCode;
  agora::CHANNEL_PROFILE_TYPE channelProfile;
  const char* license;
  rtc::AUDIO_SCENARIO_TYPE audioScenario;
  commons::LogConfig logConfig;
  bool useStringUid;
  IServiceObserver* serviceObserver;
  Optional<rtc::THREAD_PRIORITY_TYPE> threadPriority __deprecated;
  bool useExternalEglContext;
  bool domainLimit;

  AgoraServiceConfiguration()
    : enableAudioProcessor(true),
      enableAudioDevice(true),
      enableVideo(false),
      context(NULL),
      appId(NULL),
      areaCode(rtc::AREA_CODE_GLOB),
      channelProfile(agora::CHANNEL_PROFILE_LIVE_BROADCASTING),
      license(NULL),
      audioScenario(rtc::AUDIO_SCENARIO_DEFAULT),
      useStringUid(false),
      serviceObserver(NULL),
      useExternalEglContext(false),
      domainLimit(false) {}
};
```

#### Parameters

| Field                  | Type                          | Description                                                                                                                                                                                                                                           |
| ---------------------- | ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `enableAudioProcessor` | `bool`                        | Whether to enable the audio processing module.<br/> - `true` (default): Enabled.<br/> - `false`: Disabled. Disabling prevents creation of audio tracks.                                                                                                       |
| `enableAudioDevice`    | `bool`                        | Whether to enable the audio device module.<br/> - `true` (default): Enabled.<br/>` - false`: Disabled.<br/><br/>**Caution**: In recording scenarios, set this to `false`. Otherwise, the SDK will report an error because the recorder has no audio devices. |
| `enableVideo`          | `bool`                        | Whether to enable video support.<br/> - `true`: Enabled.<br/> - `false` (default): Disabled.                                                                                                                                                                  |
| `context`              | `void*`                       | Reserved for custom context.                                                                                                                                  |
| `appId`                | `const char*`                 | Your project’s App ID.                                                                                                                      |
| `areaCode`             | `unsigned int`                | The region code. Default is `AREA_CODE_GLOB`.                                                                                                                 |
| `channelProfile`       | `agora::CHANNEL_PROFILE_TYPE` | The channel profile. See [`CHANNEL_PROFILE_TYPE`](#channel_profile_type).                                                                                 |
| `license`              | `const char*`                 | The license used for authentication when joining a channel. Charges are based on the license.                                                                                                                                                         |
| `audioScenario`        | `rtc::AUDIO_SCENARIO_TYPE`    | The audio scenario. See [`AUDIO_SCENARIO_TYPE`](#audio_scenario_type).                                                                                                                                                                            |
| `logConfig`            | `commons::LogConfig`          | Log configuration, including log path, size, and level. See [`LogConfig`](#logconfig).                                                                                                                                                          |
| `useStringUid`         | `bool`                        | Whether to use string-based user IDs.<br/> - `true`: Enabled.<br/> - `false` (default): Disabled.                                                                                                                                                             |
| `domainLimit`          | `bool`                        | Whether to enable domain restriction.<br/> - `true`: Only connects to DNS-resolved servers.<br/> - `false` (default): No server restrictions.                                                                                                                 |

### `WatermarkRatio`

Defines the relative position and width of a watermark on the screen.

* `xRatio` and `yRatio` specify the coordinates of the top-left corner of the watermark, relative to the top-left corner of the screen.
* `widthRatio` specifies the relative width of the watermark. The SDK calculates the height proportionally to maintain the aspect ratio.

```cpp
struct WatermarkRatio {
  float xRatio;
  float yRatio;
  float widthRatio;

  WatermarkRatio() : xRatio(0.0), yRatio(0.0), widthRatio(0.0) {}
  WatermarkRatio(float x, float y, float width) : xRatio(x), yRatio(y), widthRatio(width) {}
};
```

#### Parameters

| Field        | Type    | Description                                                                                                                                         |
| ------------ | ------- | --------------------------------------------------------------------------------------------------------------------------------------------------- |
| `xRatio`     | `float` | Horizontal position of the watermark’s top-left corner relative to the screen’s top-left. Range: `[0.0, 1.0]`.                                      |
| `yRatio`     | `float` | Vertical position of the watermark’s top-left corner relative to the screen’s top-left. Range: `[0.0, 1.0]`.                                        |
| `widthRatio` | `float` | Relative width of the watermark. The height is scaled to preserve the aspect ratio. Range: `[0.0, 1.0]`. Default is `0`, which hides the watermark. |

### `EncryptionConfig`

Defines the configuration for Agora’s built-in encryption feature.

```cpp
struct EncryptionConfig {
  ENCRYPTION_MODE encryptionMode;
  const char* encryptionKey;
  uint8_t encryptionKdfSalt[32];
  bool datastreamEncryptionEnabled;

  EncryptionConfig()
      : encryptionMode(AES_128_GCM2),
        encryptionKey(OPTIONAL_NULLPTR),
        datastreamEncryptionEnabled(false) {
    memset(encryptionKdfSalt, 0, sizeof(encryptionKdfSalt));
  }

  const char* getEncryptionString() const {
    switch (encryptionMode) {
      case AES_128_XTS: return "aes-128-xts";
      case AES_128_ECB: return "aes-128-ecb";
      case AES_256_XTS: return "aes-256-xts";
      case SM4_128_ECB: return "sm4-128-ecb";
      case AES_128_GCM: return "aes-128-gcm";
      case AES_256_GCM: return "aes-256-gcm";
      case AES_128_GCM2: return "aes-128-gcm-2";
      case AES_256_GCM2: return "aes-256-gcm-2";
      default: return "aes-128-gcm-2";
    }
  }
};
```

#### Parameters

| Field                         | Type              | Description                                                                                                                                                                               |
| ----------------------------- | ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `encryptionMode`              | `ENCRYPTION_MODE` | Built-in encryption mode. Agora recommends `AES_128_GCM2` or `AES_256_GCM2`, which support a salt for enhanced security. See [`ENCRYPTION_MODE`](#encryption_mode).                   |
| `encryptionKey`               | `const char*`     | The encryption key as a string. No length limit, but Agora recommends 32 bytes.<br/>**Note**: If `NULL`, built-in encryption is disabled, and the SDK returns `ERR_INVALID_ARGUMENT (-2)`. |
| `encryptionKdfSalt`           | `uint8_t[32]`     | A 32-byte salt. Agora recommends generating this with OpenSSL on your server.<br/>**Note**: Required only for `AES_128_GCM2` or `AES_256_GCM2` modes. Must not be all zeros.               |
| `datastreamEncryptionEnabled` | `bool`            | Whether to enable encryption for data streams.<br/> - `true`: Enabled.<br/> - `false` (default): Disabled.                                                                                        |
