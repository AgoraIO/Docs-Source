Call `setAINSMode` to enable the AI noise suppression feature, and select a noise suppression mode:

```csharp
// Enable Audio AI Noise Suppression
bool enableAINS = true;

// Set the mode for Audio AI Noise Suppression
// Choose one of the available modes: AUDIO_AINS_MODE, AINS_MODE_BALANCED, AINS_MODE_AGGRESSIVE, AINS_MODE_ULTRALOWLATENCY
AUDIO_AINS_MODE mode = AUDIO_AINS_MODE.AINS_MODE_BALANCED;

int result = rtcEngine.SetAINSMode(enableAINS, mode);
```
