Use this endpoint to get the current status of the specified <Vpd k="NAME" /> instance.

### API Endpoint

- Method: `GET`
- Endpoint: `https://api.agora.io/cn/api/conversational-ai-agent/v1/projects/{appid}/agents/{agentId}`

### Path parameters

| Parameter   | Type | Required | Description   |
|:--------|:---------|:----|:--------------|
| `appid` | string | Yes | The App ID of the project|
| `agentId` | string | Yes | The agent instance ID you obtained after successfully calling `join` to start the conversational agent. |

### Response

- If the returned status code is `200`, the request was successful. The response body contains the result of the request.

- If the returned status code is not `200`, the request failed. The response body includes the error code and description. Refer to status codes to understand the possible reasons for failure.

### Response body

| Parameter   | Type |  Description   |
|:--------|:---------|:--------------|
| `message`    | string   | Request message |
| `requestResponse`   | object  | Response information |
| ⇒`agent_name` | string   | Agent classification  |
| ⇒`name` | string   | Unique identifier of the agent  |
| ⇒`properties` | object   | Agent properties |
| ⇒⇒`agent_rtc_uid` | integer   | The uid of the agent in the RTC channel.  |
| ⇒⇒`asr` | object   | Automatic speech recognition (ASR) configuration  |
| ⇒⇒⇒`language` | string | Language: <li>`en-US`: English</li> <li>`zh-CN`: (Default) Chinese (supports mixed Chinese and English)</li> |
| ⇒⇒`audiod_scenario` | integer   | RTC audio scenario  |
| ⇒⇒`channel` | string   | RTC channel name  |
| ⇒⇒`custom_llm` | object   | Custom LLM configuration.  |
| ⇒⇒⇒`model` | string | Model ID that may be required by custom LLM.  |
| ⇒⇒⇒`greeting` | string | Agent greeting.  |
| ⇒⇒⇒`url` | string | Custom LLM callback address  |
| ⇒⇒⇒`token` | string | Custom LLM verification token  |
| ⇒⇒`tts` | object | Text-to-speech settings |
| ⇒⇒⇒`voice_id` | string | Tone ID  |
| `statusCode` | string | Status code  |
| `taskId` | string | The task ID |
