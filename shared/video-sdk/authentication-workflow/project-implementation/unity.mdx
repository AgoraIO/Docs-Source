1. **Enable the user to specify a channel**

    Add a text box to the user interface.

    1. In your <Vpl k="NAME" /> project, right-click **Canvas**, then click **UI** > **Input Field - TextMeshPro**. A text box appears in the scene Canvas.

    1. In **Inspector**, rename **InputField(TMP)** to `textbox`, then change **Pos Y** to `100`.

    1. In **Hierarchy**, click **textbox** > **Text Area** > **Placeholder**. Then, in **Inspector**, change the **Text Input** field to `Channel Name`.

2. **Add the required import statements**

    In **Project**, open `Assets/Agora-RTC-Plugin/Agora-Unity-RTC-SDK/Code/NewBehaviourScript.cs`, then add the following to the list of namespace declarations:

    ``` csharp
    using UnityEngine.Networking;
    using System;
    using TMPro;
    ```
