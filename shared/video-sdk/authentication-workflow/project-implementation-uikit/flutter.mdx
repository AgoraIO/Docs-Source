### Authentication using <Vg k="UIK" />

1. **Add a variable for the token server url**

    ```dart
    // The base URL to your token server.
    String serverUrl = "<Token Server URL>"; // For example, "https://agora-token-service-production-92ff.up.railway.app"
    ```

1. **Set the token server url for the `AgoraClient`**

    Under `agoraConnectionData:`, **replace** the line `tempToken: token,` with the following:

    ```dart
    tokenUrl: serverUrl,
    ```
