### Test channel media Relay

1. In **Unity Editor**, in `Assets/Agora-Unity-RTC-SDK/Code/JoinChannelVideo.cs`, update `_appID`, `_channelName` and `destChannelName`.

2. [Generate a temporary token](../reference/manage-agora-account#generate-a-temporary-token) in <Vpl k="CONSOLE" /> using `_appID` and `_channelName`. Use it to update `_token` in `JoinChannelVideo`. Use the same values to generate another token and update `sourceChannelToken`.

3. Generate a third token in <Vpl k="CONSOLE" /> using `_appID` and `destChannelName`. Use it to update `destChannelToken` in `JoinChannelVideo`.

4. In your browser, navigate to the <Link target="_blank" to="{{Global.DEMO_BASIC_VIDEO_CALL_URL}}"><Vg k="COMPANY" /> web demo</Link> and update _App ID_, _Channel_, and _Token_ with the values for your temporary token, then click **Join**.

5. Repeat the previous step on a second device, but this time use `_appID`, `destChannelName`, and `destChannelToken` to **Join** the channel.

6. In **Unity Editor**, click **Play**. A moment later you see the <Vpl k="CLIENT" /> running on your development device.

    If this is the first time you run your app, grant camera and microphone permissions.

7. Select **Host** and press **Join**. You see the video from the web browser demo app connected to `_channelName` in the top frame of your device.

8. Press **Start Channel Media Relay**. You see the video from the web browser demo app connected to `_channelName` relayed to the web browser demo app connected to `destChannelName`.

9. Press **Stop Channel Media Relay**. The media relaying is stopped.
