<Vg k="COMPANY"/> <Vg k="FC"/> <Vg k="FC_UIB"/> is a low code method of generating a customized classroom. It enables you to drag & drop widgets to design the UI and customize the theme of your classroom. You can enable or disable features simply by checking boxes in the scene builder interface. After completing the design, you can deploy the project or download a ZIP file containing customized source code for several platforms.

This page shows you how to create a custom classroom with <Vg k="FC_UIB" />, install it on a device and test it.

## Understand the tech

<Vg k="FC_UIB" /> is a web interface you use to rapidly create and deploy a custom classroom:

![Image](/images/flexible-classroom/uibuilder.png)

You use <Vg k="FC_UIB" /> to:
- Create a classroom from a template for a scenario.
- Customize the scenes in the classroom.
- Drag-and-drop the desired functionality into your scenes.
- Customize the look and feel of your classroom.
- Preview the web version of your classroom.
- Download the code to run this classroom on a device or web server.

## Create and integrate a classroom using <Vg k="FC_UIB" />

UI builder enables you to generate a customized classroom in minutes.

This section shows how to use the <Vg k="FC_UIB" />  to create a custom classroom and deploy it on the platform of your choice.

### Create a classroom

To construct a custom classroom with <Vg k="FC_UIB" />:

1. In your web browser, navigate to the <Vg k="COMPANY "/> <Vg k="FC" /> [<Vg k="FC_UIB" />](https://solutions-apaas.agora.io/scene-builder/index.html) and press **Try it now**.

1. If you are not already logged in to <Vg k="CONSOLE" />, you are asked to login. After logging in, you are asked for the following authorization:

    ![Authorization](/images/flexible-classroom/flexible-classroom-authorization.png)

    Press **Accept** to allow <Vg k="FC_UIB" /> to use your <Vg k="COMPANY "/> account. This takes you back to the main <Vg k="FC_UIB" /> page.

1. Press the **Try it now** button again. You see the following screen:

    ![Create new project](/images/flexible-classroom/create-new-project.png)

    Enter a name for your project and press **Create a project**. <Vg k="FC_UIB" />  creates a new project under **My Projects**.

1. Select the newly created project and press **Edit**. <Vg k="FC_UIB" /> opens a new tab and shows you the scene editor.

    ![Scene Editor](/images/flexible-classroom/scene-editor-layout.png)

1. Press **Create new scene**. You are asked to choose a template based on the number of expected participants in your classroom.

    ![Choose a template](/images/flexible-classroom/choose-a-scene-template.png)

    For the purpose of this demo, select **Interactive Small Classroom**. Enter a name for the scene and press **Pick this scene**. <Vg k="FC_UIB" /> opens the scene editor.

1. Add modules to your scene

    Press the **Module** button in the toolbar on the left to see the Module panel. You are presented with the following module options:

    ![Choose a template](/images/flexible-classroom/scene-editor-add-module.png)

    1. Drag the **Function Bar** module and drop it in the top panel of your scene. The top panel displays the network state icon, room name and schedule time for the classroom. Select the Function Bar, you see configurable options in the panel on the right. Check or uncheck options to configure your function bar.

    1. From the Module panel, drag the **Teacher Video Window** and drop it in the second panel of your scene. Select the Teacher Video module to display configuration options in the panel on the right. Check or uncheck options to configure the module.

        ![Teacher video window](/images/flexible-classroom/scene-editor-teacher-video.png)
    
    1. From the Module panel, drag the **Student Video Window** and drop it in the second panel of your scene. Select the Student Video module to display configuration options in the panel on the right. Check or uncheck the following options to configure the module:

        * Camera
        * Microphone
        * Board Authorization
        * Reward
        * OffStage

    1. Drag the **Engagement Area** module from the Modules panel and drop it in the third panel of your scene. Select the newly created module to display configuration options in the panel on the right. Check or uncheck the following options to configure the Engagement Area module:

        * Netless Board
        * Cloud Storage
        * Roster
        * Counter
        * Popup Quiz
        * Poll
        * Screen Share
        * Breakout Room
        * Laser Pointer
        * Raise Hand
        * Agora Chat

    Your completed scene should look similar to the following:

        ![Final Scene](/images/flexible-classroom/scene-editor-complete-scene.png)

1. Choose your theme

    Press the **Theme** button in the toolbar on the left. You see options to change system colors in the panel on the right. Choose the *Light* or *Dark* theme and desired colors to customize your classroom UI.

### Integrate your classroom

When you have selected the desired features and created the look and feel for your classroom, you use <Vg k="FC_UIB" /> to generate the source code that runs the classroom on your users devices. To do this:

1. Download the code for your customized classroom. Press **Source Code Download** in <Vg k="FC_UIB" />. You are presented with the following options:

    * Styling only
    * Full source code

    Select *Full source code*. <Vg k="FC_UIB" /> takes a few seconds to generate your custom code and then prompts you to download a `zip` file. Save the file on your development machine.

## Test your classroom

<Vg k="COMPANY"/> recommends you run this project on a physical mobile device, as some simulators may not support the full features of this project. To ensure that you have implemented <Vg k="FC" /> in your <Vpl k="CLIENT" />:

1. Press **Preview** in <Vg k="FC_UIB" />.

    <Vg k="FC_UIB" /> prepares a web preview version of the classroom and takes you to the login screen. Fill in the room name and select the *Teacher* role. Press **Enter** to create the classroom.

1. Check the device settings to ensure that your camera and microphone are working properly. Press **Finish** to close the popup window.

1. Press **Begin Class** to start your classroom session.

1. Compile and run the downloaded code on a second device and launch the classroom.

1. Enter the same Room name and join the classroom as a *Student*.

    Now you can see the content from the *Teacher* and participate in the classroom activities.
