This page lists all enums for the <Vg k="OPREC_SDK" />.

## AreaCode

Defines the connection region, which refers to the region of the Agora SDK server.

| Enum Value | Description |
|------------|-------------|
| `AREA_CODE_CN` | (0x00000001): Mainland China. |
| `AREA_CODE_NA` | (0x00000002): North America. |
| `AREA_CODE_EU` | (0x00000004): Europe. |
| `AREA_CODE_AS` | (0x00000008): Asia (excluding Mainland China). |
| `AREA_CODE_JP` | (0x00000010): Japan. |
| `AREA_CODE_IN` | (0x00000020): India. |
| `AREA_CODE_GLOB` | (0xFFFFFFFF): (Default) Global. |

## EncryptionMode

| `AES_256_GCM2` | (8): Requires salt (`encryptionKdfSalt`). |

## WaterMarkFitMode

Defines the fit mode of the watermark.

| Enum Value                 | Description                                                                                                                                                                                              |
| -------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `FIT_MODE_COVER_POSITION`  | (0): Uses the `positionInLandscapeMode` and `positionInPortraitMode` values set in [`WatermarkOptions`](struct#watermarkoptions). The settings in [`WatermarkRatio`](struct#watermarkratio) are ignored. |
| `FIT_MODE_USE_IMAGE_RATIO` | (1): Uses the values set in [`WatermarkRatio`](struct#watermarkratio). The `positionInLandscapeMode` and `positionInPortraitMode` settings in [`WatermarkOptions`](struct#watermarkoptions) are ignored. |
