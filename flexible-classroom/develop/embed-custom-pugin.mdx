<Vg k="EXTAPP"/> enables developers to develop a custom plugin, such as a countdown plugin or a dice, and embed the plugin in the <Vg k="FC"/>. Plugins implemented by <Vg k="EXTAPP"/> can be regarded as an independent applications with its own life cycle and data management, but they also connect with the <Vg k="ACS"/>. Developers can customize the user interfaces of the plugins, pass custom data to the <Vg k="ACS"/>, and also listen for data changes from the <Vg k="ACS"/>.

The source code of <Vg k="EXTAPP"/> is in the `extapp` directory in the [CloudClass-Android](https://github.com/AgoraIO-Community/CloudClass-Android) repository on GitHub (Branch release/apaas/1.1.5.1).

This page introduces the procedure of using <Vg k="EXTAPP"/> to develop and embed a custom plugin in the <Vg k="FC"/>.
