### Beautification

#### Apply preset beauty effects

Use the `setFaceShapeBeautyOptions(_:options:)` method to enable beauty effects. Set `enable` to `true` and configure the beauty style and intensity using `AgoraFaceShapeBeautyOptions`.

```swift
var beautyOptions = AgoraFaceShapeBeautyOptions()
// Set the beauty style to male
beautyOptions.shapeStyle = .male
// Set the intensity
beautyOptions.styleIntensity = 100

agoraKit?.setFaceShapeBeautyOptions(true, options: beautyOptions)
```

#### Fine-tune facial areas

Use the `setFaceShapeAreaOptions(_:)` method to fine-tune specific facial features. This method immediately applies changes to individual areas without affecting the preset values.

```swift
var areaOptions = AgoraFaceShapeAreaOptions()
// Adjust the head size
areaOptions.shapeArea = .headScale
areaOptions.shapeIntensity = 100

agoraKit?.setFaceShapeAreaOptions(areaOptions)
```

#### Disable beauty effects

To turn off beauty effects, call `setFaceShapeBeautyOptions(_:options:)` and pass `false` to the `enable` parameter.

```swift
agoraKit?.setFaceShapeBeautyOptions(false, options: beautyOptions)
```

### Makeup

#### Enable makeup effects

To enable makeup effects, call the `setExtensionPropertyWithVendor(_:extension:key:value:)` method and set the following parameters:

- `vendor`: `"agora_video_filters_clear_vision"`
- `extension`: `"clear_vision"`
- `key`: `"makeup_options"`
- `value`: A JSON string that defines the beauty options. See [Makeup JSON fields](#makeup-json-fields)

```swift
let makeupParams: [String: Any] = [
   "enable_mu": true,
   "blushStyle": 1,
   "blushColor": 2,
   "blushStrength": 0.5
]
guard let json = try? JSONSerialization.data(withJSONObject: makeupParams),
   let jsonString = String(data: json, encoding: .utf8) else { return }

agoraKit?.setExtensionPropertyWithVendor("agora_video_filters_clear_vision",
                                         extension: "clear_vision",
                                         key: "makeup_options",
                                         value: jsonString)
```

#### Modify makeup effects

To update makeup settings, change the JSON values and call `setExtensionPropertyWithVendor(_:extension:key:value:)` again.

```swift
let makeupParams: [String: Any] = [
   "blushColor": 1
]
guard let json = try? JSONSerialization.data(withJSONObject: makeupParams),
   let jsonString = String(data: json, encoding: .utf8) else { return }

agoraKit?.setExtensionPropertyWithVendor("agora_video_filters_clear_vision",
                                         extension: "clear_vision",
                                         key: "makeup_options",
                                         value: jsonString)
```

#### Disable makeup effects

To turn off makeup effects, set `enable_mu` to `false` in the JSON string and apply it:

```swift
let makeupParams: [String: Any] = [
   "enable_mu": false
]
guard let json = try? JSONSerialization.data(withJSONObject: makeupParams),
   let jsonString = String(data: json, encoding: .utf8) else { return }

agoraKit?.setExtensionPropertyWithVendor("agora_video_filters_clear_vision",
                                         extension: "clear_vision",
                                         key: "makeup_options",
                                         value: jsonString)
```
