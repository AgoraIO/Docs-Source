<summary><strong>v2.1.0</strong> - March 6, 2024</summary>

### Improvements

This update optimizes the performance of the MetaKit plugin and adds support for two new `keys` in `setExtensionProperty`: [`requestTexture`](./api-ref#Open-texture-request) and [`switchTextureAvatarMode`](./api-ref#Switch-texture-scene). These replace the original `addSceneView` and `switchAvatarMode` functions, respectively. The new interface supports automatic generation and return of texture data, which is then directly rendered, previewed, encoded, and transmitted through the SDK. This enhancement improves rendering performance, reduces latency, and ensures more efficient operation and a better user experience.

### API Changes

#### New

- [`requestTexture`](./api-ref#Open-texture-request)
- [`switchTextureAvatarMode`](./api-ref#Switch-texture-scene)
