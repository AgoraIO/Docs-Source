1. **Implement a client for <Vpd k="NAME"/>**

    1. Import the AgoraUIKit component from the `agora-rn-uikit` package:

        Open the file `App.js` and replace the contents with the following:

        ```js
        import React, {useState} from 'react';
        import AgoraUIKit from 'agora-rn-uikit';
        import {Text} from 'react-native';

        ```

    1. Connect to and disconnect from a channel

        The App component uses the `videoCall` state variable. To join a call your app sets `videoCall` to `true`, passes the standard Agora `appID`, `channel` and `token` to an AgoraUIKit instance, then connects to a channel.

        To set the variables for this logic, add the following code to `App.js` after the import statements:

        ```js
        const App = () => {
        const [videoCall, setVideoCall] = useState(true);
        const connectionData = {
            appId: '<app-id>',
            channel: '<channel-name>',
            token: '<token>',
        };

        export default App;
        ```

    1. Setup callbacks for <Vg k="UIK" /> events

        The callbacks object contains Agora Engine events as keys and callback functions as their values. You use the `EndCall` event to update the state and unmount the AgoraUIKit component. To do this add the following code to `App.js` after the `connectionData` channel declaration:

        ```js
        const callbacks = {
            EndCall: () => setVideoCall(false),
        };
        ```

    1. Display the user interface

        If `videoCall` is `true`, show the AgoraUIKit interface. If it is `false` show the Join button. To do this, add the following code after the callbacks declaration:

        ```js
        return videoCall ? (
            <AgoraUIKit connectionData={connectionData} rtcCallbacks={rtcCallbacks} />
        ) : (
            <Text onPress={() => setVideoCall(true)}>Start Call</Text>
        );
        ```
