import CodeBlock from '@theme/CodeBlock';

1. In your IDE, create a new Java project.  
2. Set the project name, save location, and choose **Maven** as the build system.  
3. Finish creating the project.  
4. Open the `pom.xml` file and add the following dependency inside the `<dependencies>` section to include the <Vg k="COMPANY" /> REST client:

    <CodeBlock language="xml" showLineNumbers>
    {`<dependencies>
       <dependency>
           <groupId>io.agora</groupId>
           <artifactId>agora-rest-client-core</artifactId>
           <version>0.3.0</version>
       </dependency>
   </dependencies>`}
    </CodeBlock>
