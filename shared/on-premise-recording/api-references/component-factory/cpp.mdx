## IAgoraMediaComponentFactory

This class is used to create media components.

| API | Description |
|-----|-------------|
| [`createMediaRtcRecorder`](#createmediartcrecorder) | Creates an [`IAgoraMediaRtcRecorder`](#iagoramediartcrecorder) media recorder instance. |
| [`createAgoraMediaComponentFactory`](#createagoramediacomponentfactory) | Creates an [`IMediaComponentFactory`](#iagoramediacomponentfactory) object and returns a pointer. |

### `createMediaRtcRecorder`

Creates an instance of [`IAgoraMediaRtcRecorder`](#createmediartcrecorder).

```cpp
virtual agora_refptr<IAgoraMediaRtcRecorder> createMediaRtcRecorder() = 0;
```

#### Return Values

- On success: Returns an [`IAgoraMediaRtcRecorder`](#iagoramediartcrecorder) object.
- On failure: Returns a null pointer.

### `createAgoraMediaComponentFactory`

Creates and returns a pointer to an `IMediaComponentFactory` object.

```cpp
AGORA_API agora::rtc::IMediaComponentFactory* AGORA_CALL createAgoraMediaComponentFactory();
```

#### Return Values

- On success: Returns a pointer to an `IMediaComponentFactory`.
- On failure: Returns a null pointer.
