1. **Enable the extension**

    After creating and initializing Agora engine, you call `enableExtension` to enable the <Vg k="RTEE_FU" /> extension. For Android, you need to load the extension before enabling it. TO do this, add the following code to `setupVideoSDKEngine` after `await agoraEngine.initialize(...)`:

    ```dart
    if (Platform.isAndroid) {
        await agoraEngine.loadExtensionProvider(path: 'AgoraFaceUnityExtension');
    }

    await agoraEngine.enableExtension(
        provider: "FaceUnity", extension: "Effect", enable: true);
    ```

1. **Handle the Agora engine extension callbacks**

    The `RtcEngineEventHandler` fires callbacks when an extension is successfully started or fails to start. The `onExtensionEvent` callback listens for extension events while the extension is running. After receiving the `onExtensionStarted` callback you initialize the extension. To handle the Agora engine callbacks, add the following code after `agoraEngine.registerEventHandler( RtcEngineEventHandler(`:

    ```dart
    onExtensionStarted: (provider, extension) {
        debugPrint(
            '[onExtensionStarted] provider: $provider, extension: $extension');
        if (provider == 'FaceUnity' && extension == 'Effect') {
            initializeFaceUnityExt();
        }
    },
    onExtensionError: (provider, extension, error, message) {
        debugPrint(
            '[onExtensionError] provider: $provider, '
                'extension: $extension, error: $error, message: $message');
    },
    onExtensionEvent: (String provider, String extName, String key, String value) {
        debugPrint(
            '[onExtensionEvent] provider: $provider, '
                'extension: $extName, key: $key, value: $value');
    }
    ```

1. **Initialize the extension**

    To initialize and configure the extension, you call `setExtensionProperty` and pass in the corresponding `key` and `value` pairs. The specific call order is as follows:

    1. To initialize the extension and authenticate the user, use the `fuSetup` key. The corresponding value contains the `authdata` JSON parameter with the path to the certificate file.

    2. To preload AI capabilities, use the `fuLoadAIModelFromPackage` key. The value contains the `data` and `type` JSON parameters with the path to the AI capability model file `ai_xxx.bundle`, and the AI capability type, respectively.

    To do this, add the following method to the `_MyAppState` class:

    ```dart
    Future<void> initializeFaceUnityExt() async {
        // Initialize the extension and authenticate the user
        await agoraEngine.setExtensionProperty(
            provider: 'FaceUnity',
            extension: 'Effect',
            key: 'fuSetup',
            value: jsonEncode({'authdata': authpack.gAuthPackage}));

        // Load the AI model
        final aiFaceProcessorPath =
            await _copyAsset('Resource/model/ai_face_processor.bundle');

        await agoraEngine.setExtensionProperty(
            provider: 'FaceUnity',
            extension: 'Effect',
            key: 'fuLoadAIModelFromPackage',
            value: jsonEncode({'data': aiFaceProcessorPath, 'type': 1 << 10}));
    }
    ```

1. **Set the AR effect and enable AI recognition features**

    To load props, adjust beauty intensity, or enable face, gesture and human recognition, call `setExtensionProperty` and pass in the corresponding `key` and `value`.

    In this example, you set the cat ears effect. Add the following method to the `_MyAppState` class:

    ```dart
    Future<void> setupCatSparksEffect() async {
        final aiFaceProcessorPath =
            await _copyAsset('Resource/model/ai_face_processor.bundle');
            
        await agoraEngine.setExtensionProperty(
            provider: 'FaceUnity',
            extension: 'Effect',
            key: 'fuLoadAIModelFromPackage',
            value: jsonEncode({'data': aiFaceProcessorPath, 'type': 1 << 10}));

        final catSparksPath =
            await _copyAsset('Resource/items/ItemSticker/CatSparks.bundle');

        await agoraEngine.setExtensionProperty(
            provider: 'FaceUnity',
            extension: 'Effect',
            key: 'fuCreateItemFromPackage',
            value: jsonEncode({'data': catSparksPath}));
    }
    ```

    You can use a combination of calls according to the needs of your scene. 
    To implement other effects or to enable AI recognition features, refer to the [<Vg k="RTEE_FU" /> Key-Value overview](#faceunity-key-value-overview) to see the detailed interface description.

1. **Release resources**

    When you no longer need to use the <Vg k="RTEE_FU" /> extension, call `setExtensionProperty`, and pass in the `fuDestroyLibData` key to release the resources occupied by the extension. Add the following code to the `dispose()` method of the `_MyAppState` class:

    ```dart
    await agoraEngine.setExtensionProperty(
        provider: 'FaceUnity',
        extension: 'Effect',
        key: 'fuDestroyLibData',
        value: jsonEncode({}));
    ```

    The `fuDestroyLibData` key in the `onExtensionEvent` callback confirms the success of this call.
