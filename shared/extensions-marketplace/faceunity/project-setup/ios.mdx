<PlatformWrapper platform="ios">

1. Create a `Resource` folder under your project folder, as follows:
    ```shell
    .
    ├── <ProjectName>
    ├── <ProjectName>.xcodeproj
    └── Resource
    ```
1. Unzip the downloaded package, and import all `.framework` files to your project. Ensure that you select **Embed & Sign** instead of **Embed**.
1. Import the certificate file `authpack.h` to your project.
1. Save the model and prop files you need from the resource package to the `Resource` path. For details of files provided in the resource pack, see [Resource package structure](#resource-package-structure).
1. To import the required header files, add the following statements:

   ``` objective-c
    #import <AgoraRtcKit/AgoraRtcEngineKit.h>
    #import "authpack.h"
   ```
