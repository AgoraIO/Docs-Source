The `IVideoFrameObserver` class.

### `on_frame`

```python
def on_frame(self, channel_id, remote_uid, frame:VideoFrame)
```

Occurs when a video frame is received.

#### Parameters

| Parameter | Description |
|---------------|----------------------------------------------------------------------|
| `channel_id` | The channel name.             |
| `remote_uid` | The remote user ID.             |
| `frame`| The video data. See [`VideoFrame`](./apidatatype#videoframe)         |
