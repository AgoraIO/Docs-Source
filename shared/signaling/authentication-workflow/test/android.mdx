<PlatformWrapper platform="android">

1. **Configure your project**

    In `<samples-root>/signaling-manager/src/main/res/raw/config.json`, replace:
    - `appId` - with the value for your project from <Link target="_blank" to="{{global.AGORA_CONSOLE_URL}}"><Vg k="CONSOLE" /></Link>.
    - `serverUrl` - with the base URL for your token server. For example: `https://agora-token-service-production-yay.up.railway.app`.


1.  **Run the <Vpl k="NAME" /> reference app**:

    1. In Android Studio, connect a physical Android device to your development machine.
    1. Click Run to start the app.
    1. A moment later you see the project installed on your device. 
    1. From the main screen of the app, select **Authentication workflow**.

1. **Test the authentication server functionality**:

    For each user you want to communicate between:

    1. Launch an instance of the <Vpl k="CLIENT"/> on a separate device.
    1. Enter a numeric user ID.
    1. Press **Login**.
    1. Enter a channel name.
    1. Press **Join** to join a channel.
    1. Send messages between users.
