<PlatformWrapper platform="web">

1.  **Enable the connection to <Vg k="CP"/>**

    To access <Vg k="AGORA_BACKEND" /> in a restricted network environment, set `cloudProxy` to `true`:

    ``` javascript
    "cloudProxy": true,
    ```

2.  **Pass the configuration to the <Vg k="MESS_ENGINE" />**

    In your `rtmConfig` object, update `cloudProxy` then set this configuration to the signalingEngine.

    ``` javascript
      // Set cloud proxy
      const rtmConfig = {
        token: config.token,
        cloudProxy: config.cloudProxy,
      };

      // Extend the SignalingManager by importing it
      const signalingManager = await SignalingManager(
        messageCallback,
        eventsCallback,
        rtmConfig
      );
    ```
