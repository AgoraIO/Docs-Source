You call the `enableContentInspect` method to enable the video screenshot upload function, and set the `type` in `AgoraContentInspectConfig` to `.supervise`. After the video screenshot upload function is successfully enabled, your <Vpl k="CLIENT"/> takes screenshots according to the frequency specified by the `interval` and uploads the screenshots to your third-party storage. The sample code is as follows:

``` swift
let inspectExtraConfig = AgoraContentInspectConfig()
inspectExtraConfig.extraInfo = "YourExtraInfo"
var modules = [AgoraContentInspectModule]()
let module1 = AgoraContentInspectModule()
module1.type = .supervise
// Set the upload frequency of screenshots in seconds
module1.interval = 2
modules.append(module1)
inspectExtraConfig.modules = modules
// Enable screenshot upload
agoraKit.enableContentInspect(true, config:inspectExtraConfig)
```
