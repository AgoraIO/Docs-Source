Get a list of <Vpd k="NAME" />s.

### API Endpoint

- Method: `GET`
- Endpoint: `https://api.agora.io/cn/api/conversational-ai-agent/v1/projects/{appid}/agents`

### Path parameters

| Parameter   | Type | Required | Description   |
|:--------|:---------|:----|:--------------|
| `appid` | string | Yes | The App ID of the project|

### Response

- If the returned status code is `200`, the request was successful. The response body contains the result of the request.

- If the returned status code is not `200`, the request failed. The response body includes the error code and description. Refer to status codes to understand the possible reasons for failure.

### Response body

| Parameter   | Type |  Description   |
|:--------|:---------|:--------------|
| `data` | object | Agent data |
| ⇒`count` | integer | Number of lists |
| ⇒`list` | array | Array |
| ⇒⇒`startTs` | integer | Start timestamp |
| ⇒⇒`stateName` | string | Current status |
| ⇒⇒`taskId` | integer | Task ID |
| `meta` | object |  |
| ⇒`cursor` | string | Paging cursor |
| ⇒`total` | integer | Total quantity |
| `status` | string | Request status |
