<PlatformWrapper platform="ios">

1. Create an <Vpl k="CLIENT" /> instance for the first user:

    1. [Register a user](enable#register-a-user) in <Vg k="CONSOLE" /> and [Generate a user token](enable#generate-a-user-token).

    1. In the `ViewController` class, update `userId`, `token`, and `appKey` with values from <Vg k="CONSOLE" />.  To get your app key, see [Get Chat project information](enable#get-chat-project-information).

    1. Connect a physical <Vpl k="NAME"/> device to your development device.

    1. In Xcode, click **Run app**. A moment later you see the project installed on your device.

1. Create an <Vpl k="CLIENT" /> instance for the second user:

    1. Register a second user in <Vg k="CONSOLE" /> and generate a user token.

    1. In the `ViewController` class, update `userId` and `token` with values for the second user. Make sure you use the same `appKey` as for the first user.

    1. Run the modified app on a device emulator or a second physical <Vpl k="NAME"/> device.

1. On each device, click **Join** to log in to the <Vg k="CHAT_SERVER" />.

1. Edit the recipient name on each device to show the user ID of the user logged in to the other device.

1. Type a message in the **Message** box of either device and press **`>>`**.

    The message is sent and appears on the other device.

1. Press **Leave** to log out of the <Vg k="CHAT_SERVER" />.

</PlatformWrapper>
