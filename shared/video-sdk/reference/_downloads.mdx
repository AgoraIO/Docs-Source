<PlatformWrapper platform="web">

To manually install <Vpd k="SDK" />:

- **From CDN**

    In the file housing your code, add the following code to the line before `<style>`.

    ``` html
    <script src="https://download.agora.io/sdk/release/AgoraRTC_N-<the.version.number>.js"></script>
    ```

- **Using a download**
    1. Extract the files from: https://download.agora.io/sdk/release/Agora_Web_SDK_v4_12_2_FULL.zip to a local directory.
    1. Copy the `.js` file to the folder holding your project files.
    1. In the files housing your code, add the following code to the line before `<style>`.

        ```js
        <script src="./AgoraRTC_N-<the.version.number>.js"></script>
        ```

</PlatformWrapper>

<PlatformWrapper platform="android">

To manually install <Vpd k="SDK" /> from a download:

1. Extract the files from: https://download.agora.io/sdk/release/Agora_Native_SDK_for_Android_v4.0.0-beta.2_FULL.zip to a local directory, `<unzipped_package>`.

2.  Copy the following files and folders from `<unzipped_package>/rtc/sdk` to your project:

    | File or folder           | Path in your project     |
    |--------------------------|--------------------------|
    | Files                    |                          |
    | `agora-rtc-sdk.jar`      | `/app/libs/`             |
    | Folders                  |                          |
    | `arm-v8a`                | `/app/src/main/jniLibs/` |
    | `armeabi-v7a`            | `/app/src/main/jniLibs/` |
    | `x86`                    | `/app/src/main/jniLibs/` |
    | `x86_64`                 | `/app/src/main/jniLibs/` |
    | `high_level_api/include` | `/app/src/main/jniLibs/` |

3.  Select `Project Files/app/libs/agora-rtc-sdk.jar` file on navigation bar, right-click and then select `add as a library`.

</PlatformWrapper>

<PlatformWrapper platform="ios">

To manually install <Vpd k="SDK" />:

1. [Ensure that you have installed **CocoaPods**](https://guides.cocoapods.org/using/getting-started.html#getting-started).

2.  In Terminal, navigate to the project path, and run the `pod init` command to create a `Podfile` in the project folder.

3.  Open the `Podfile`, delete all contents, and input the following codes.

    ``` terminal
    platform :ios, '9.0'
        target 'Your App' do
        // For version, fill in a specific SDK version number. For example, 4.0.0-beta.1
        pod 'AgoraRtcEngine_iOS_Beta', 'version'
    end
    ```
1. Replace `Your App` with the target name of your project and replace `version` with the version of the SDK that you want to integrate.

4.  Return to Terminal, and run the `pod install` command to install <Vpd k="SDK" />. Once you successfully install the SDK, it shows `Pod installation complete!` in Terminal, and you can see an `xcworkspace` file in the project folder.

5.  Open the generated `xcworkspace` file.
