By default, the SDK connects to the nearest Agora server. Set the `areaCode` parameter when calling the `initialize` method to specify the access area. After you set the access area, the SDK only connects to Agora servers in the specified area.

You can set the area to:

* `AreaCodeGlob`: Global (Default) 
* `AreaCodeCn`  : Mainland China
* `AreaCodeNa`  : North America
* `AreaCodeEu`  : Europe
* `AreaCodeAs`  : Asia excluding Mainland China
* `AreaCodeJp`  : Japan
* `AreaCodeIn`  : India

<Admonition title="information" type="info">
Area codes support bitwise operations.
</Admonition>

Refer to the following sample code:

```javascript
// Only connect to the Agora SD-RTN located in North America.
agoraEngine.initialize(
{
    appId: appID,
    areaCode: AreaCode.AreaCodeNa
});
```
