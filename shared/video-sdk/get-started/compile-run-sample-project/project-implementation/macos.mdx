<PlatformWrapper platform="macos">

1. Open the file `APIExample.xcworkspace` in Xcode. Check **Automatically manage signing** in **Signing & Capabilities** under **TARGETS**, and then configure your Apple developer account and **Bundle Identifier** under each target.

    ![](/images/video-sdk/run-compile-macos-step-1.jpeg)

1. Click **Start compiling**.

1. After successful compilation, the sample project window pops up.

1. Select an example you want to try, then enter a channel name, such as *test*, and click the **Join Channel**.

    ![](/images/video-sdk/run-compile-macos-step-4.png)

1. To test various audio and video interaction scenarios, connect to the [Agora web demo](https://docs.agora.io/en/video-calling/develop/receive-notifications?platform=web), or install and run the sample project on a second device. Make sure you use the same app ID on both devices. When you join the same channel from two devices, you can see and hear each other.

    ![](/images/video-sdk/run-compile-macos-step-5.png)
