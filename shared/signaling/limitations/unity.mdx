## Call limit

The call limit is for one `IRtmService` instance. If an operation corresponds to multiple methods, the number of the method calls of an operation equals the sum of the method calls of all corresponding methods in a specific time frame.

<div class="alert note">You can increase the call limit of an API by creating multiple <code>IRtmService</code> instances.</div>

## String size

- The maximum size of a peer-to-peer or channel message is 32 KB. See  <Link to="{{Global.SIGNALING_API_REF_UNITY_ROOT}}/classagora__rtm_1_1_i_message.html#a5676ab94aed61a0164b9441923b434db">IMessage.SetText</Link>.
- The maximum size of the content in a call invitation is 8 KB. See <Link to="{{Global.SIGNALING_API_REF_UNITY_ROOT}}/classagora__rtm_1_1_local_invitation.html#ae7a9afa8629181f69e29ee7ac692e409">ILocalCallInvitation.SetContent</Link>.
- The maximum size of the response in a call invitation is 8 KB. See <Link to="{{Global.SIGNALING_API_REF_UNITY_ROOT}}/classagora__rtm_1_1_local_invitation.html#aeaa157785381a35f37289c1e12dbb342">IRemoteCallInvitation.GetResponse</Link>.
