Agora provides basic beauty effects as part of the <Vg k="VSDK" />. You can also add third-party effects by integrating extensions from vendors like FaceUnity and Banuba. Visit the [Extensions Marketplace](https://www.agora.io/en/agora-extensions-marketplace/) for information on third-party extensions.

## Basic Beauty Effects

Basic beauty effects are free of charge and included in the native and web SDKs. 
<PlatformWrapper notAllowed="web">
Agora native <Vg k="VSDK" />s provide the `setBeautyEffectOptions` API to implement basic beauty effects.

The following table shows the available effects and parameter options:

| Effect | Description |
| :----------- | :---------- |
| Contrast adjustment for light and dark | Choose from the following values: <ul><li>`LIGHTENING_CONTRAST_LOW` (0): Low contrast level.</li><li>`LIGHTENING_CONTRAST_NORMAL` (1): Normal contrast level.</li><li>`LIGHTENING_CONTRAST_HIGH` (2): High contrast level.</li></ul> The larger the value, the greater the contrast between light and dark. |
| Brightness adjustment (whitening) | Whitening with the value range of [0.0 ~ 1.0], where 0.0 means the original brightness. The default value is 0.6. The higher the value, the greater the degree of brightness. |
|Smoothness adjustment (skin smoothing) | Skin Smoothing with the value range of [0.0 ~ 1.0], where 0.0 means the original smoothness. The default value is 0.5. The greater the value, the greater the smoothness level. |
| Color saturation adjustment (for a more rosy complexion) | Cheek blushing with the value range of [0.0 ~ 1.0], where 0.0 means the original redness. The default value is 0.1. The larger the value, the greater the redness level. |
| Detail clarity (sharpening) | Sharpening with the value range of [0.0 ~ 1.0],where 0.0 means the original sharpness. The default value is 0.3. The larger the value, the greater the degree of sharpness. |

### Sample Code

<PlatformWrapper platform="android">
```java
BeautyOptions options = new BeautyOptions();
options.lighteningContrastLevel = LIGHTENING_CONTRAST_NORMAL;
options.smoothnessLevel = 0.5;
options.lighteningLevel = 0.6;
options.rednessLevel = 0.1;
options.sharpnessLevel = 0.3;
mRtcEngine.setBeautyEffectOptions(true, options);
```
</PlatformWrapper>

<PlatformWrapper platform="web">
For the Web SDK, <Vg k="COMPANY" /> provides the [`agora-extension-beauty-effect`](https://www.npmjs.com/package/agora-extension-beauty-effect?activeTab=versions) extension. Use it together with the Agora Web SDK v4.12.0 or later to implement the basic beauty effects feature. This extension gives you granular control over the degree of enhancement for effects such as skin lightening, skin smoothing, and red saturation.
Check out the [web demo](https://webdemo-global.agora.io/example/plugin/beauty/index.html) to explore the effects.
</PlatformWrapper>
