#### Test your Blueprint implementation

To test your Blueprint implementation:

1. In the **Load Agora Config** function, update `AppId`, `ChannelId` and `Token` with the values for your temporary token.

1. In the Unreal Editor, click **Play** button to run your project and click **JoinChannel** to join the channel.

1. Now, you can see yourself and the remote user on the test device and talk to the remote user using your <Vpl k="CLIENT" />.
