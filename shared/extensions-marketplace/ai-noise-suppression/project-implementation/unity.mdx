**Enable <Vg k="RTEE_NS" /> into your <Vpl k="CLIENT" />**

    Call `SetParameters` to enable the audio noise suppression feature and select a noise suppression mode. In your script file, locate `SetupVideoSDKEngine` and add the following after `RtcEngine.Initialize(context);`:

        ```csharp
        // Enable the AI noise reduction feature.
        RtcEngine.SetParameters("{\"che.audio.enable.nsng\":true}");
        // Set the mode of the AI noise reduction feature to hybrid mode('2'), which uses both the traditional and deep-learning noise reduction. 
        // You might pass in '1' instead of '2' to change the mode to use deep-learning noise reduction only. 
        // To disable the deep-learning noise reduction, pass in '0'.
        RtcEngine.SetParameters("{\"che.audio.ns.mode\":2}");
        ```
    The default noise reduction effect is soft. To make the reduction effect more aggressive, add the following after `agoraEngine.setParameters("{\"che.audio.ns.mode\":2}");`:

        ```csharp
        // The limit of noise suppression gain mapping index. The default value is 80.
        RtcEngine.SetParameters("{\"che.audio.nsng.lowerBound\":10}");
        // The limit of noise suppression gain mapping value. The default value is 50.
        RtcEngine.SetParameters("{\"che.audio.nsng.lowerMask\":10}");
        // The limit of statistical noise suppression gain. The default value is 5.
        RtcEngine.SetParameters("{\"che.audio.nsng.statisticalbound\":0}");
        // The gain limit of the whole noise suppression framework. The default value is 30
        RtcEngine.SetParameters("{\"che.audio.nsng.finallowermask\":10}");
        ```
