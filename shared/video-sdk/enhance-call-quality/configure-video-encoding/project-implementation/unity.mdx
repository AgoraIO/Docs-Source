Use the `SetVideoEncoderConfiguration` method to configure video encoding parameters. Best practice is to call this method before calling `EnableVideo` to speed up the time required for the first frame to appear. Alternatively, call this method within the channel to flexibly adjust video encoding properties.
To set video encoder configuration, refer to the following code:

```csharp
// Video encoding parameter configuration
VideoEncoderConfiguration config = new VideoEncoderConfiguration
{
    // Set the video encoding resolution (px)
    dimensions = _dimensions[dim],
    // Set the frame rate to 15 fps
    frameRate = 15,
    // Set the bitrate mode to STANDARD_BITRATE
    bitrate = 0,
    // Set the minimum encoding bitrate to 1 Kbps
    minBitrate = 1,
    // Set the orientation mode to ADAPTIVE
    orientationMode = ORIENTATION_MODE.ORIENTATION_MODE_ADAPTIVE,
    // Set the degradation preference to MAINTAIN_FRAMERATE
    degradationPreference = DEGRADATION_PREFERENCE.MAINTAIN_FRAMERATE,
    // Set the mirror mode for encoding video to AUTO
    mirrorMode = VIDEO_MIRROR_MODE_TYPE.VIDEO_MIRROR_MODE_AUTO
};
// Apply the video encoding configuration
RtcEngine.SetVideoEncoderConfiguration(config);
```
