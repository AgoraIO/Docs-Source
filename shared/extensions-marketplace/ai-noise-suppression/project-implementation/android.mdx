<PlatformWrapper platform="android">

1. **Integrate <Vg k="RTEE_NS" /> into your <Vpl k="CLIENT" />**

    If you installed the `io.agora.rtc:full-sdk` when you setup your  <Vpl k="CLIENT" />,  <Vg k="RTEE_NS" /> is already included. If not, add the latest version of https://search.maven.org/search?q=a:ains as an implementation in `/Gradle Scripts/build.gradle (Module: <project name>.app)`.


1.  **Import the necessary classes**

    In `app/src/main/java/com.example.<projectname>/MainActivity`, add the following lines to import the Android classes used by the extension:

    ``` java
    import io.agora.rtc2.video.VideoDenoiserOptions;
    ```

1. **Initialize the variable to set <Vg k="RTEE_NS" /> options**

    Add the following line to the top of the `MainActivity` class:
    ``` java
    private VideoDenoiserOptions denoiserOptions;
    ```

1. **Set the <Vg k="RTEE_NS" /> mode**

    Auto mode automatically enables or disables the video noise reduction feature according to the ambient light. To set the <Vg k="RTEE_NS" /> mode to `auto`, in `setupVideoSDKEngine`, add the following code after `config.mEventHandler = mRtcEventHandler;`:

    ```java
    denoiserOptions.denoiserMode = VideoDenoiserOptions.VIDEO_DENOISER_AUTO;
    ```

1. **Adjust the <Vg k="RTEE_NS" /> level**

    By default, <Vg k="RTEE_NS" /> is set to 'HighQuality' to balance the <Vg k="RTEE_NS" /> quality and performance.

    - To set the value to 'Fast' and prioritize <Vg k="RTEE_NS" /> performance over quality, in `setupVideoSDKEngine`, put this line after `denoiserOptions.denoiserMode = VideoDenoiserOptions.VIDEO_DENOISER_AUTO;`:

        ```java
            denoiserOptions.denoiserLevel = VideoDenoiserOptions.VIDEO_DENOISER_LEVEL_FAST;
        ```

    - To give more focus to <Vg k="RTEE_NS" /> quality rather than performance, change the level to 'Strength':

        ```swift
        denoiserOptions.denoiserLevel = VideoDenoiserOptions.VIDEO_DENOISER_LEVEL_STRENGTH;
        ```

6. **Set <Vg k="RTEE_NS" /> on <Vg k="ENGINE" />**

    Put the following code after `agoraEngine = RtcEngine.create(config);`:

    ```swift
    agoraEngine.setVideoDenoiserOptions(true,denoiserOptions );
    ```
