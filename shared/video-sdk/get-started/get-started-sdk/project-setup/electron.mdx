### Create a project

Create a new folder for your Electron project. Add the following files to the project root folder:

* `package.json`: Used to install and manage project dependencies.
* `index.html`: Specifies the user interface of the app.
* `main.js`: The main process file.
* `renderer.js`: Rendering process file, used to interact with <Vpd k="SDK"/>.

### Integrate the SDK

To integrate the <Vpd k="SDK"/> in your project using npm, take the following steps:

1. Add the following code to `package.json`:

    * macOS

        ```json
        {
            "name": "electron-demo-app",
            "version": "0.1.0",
            "author": "your name",
            "description": "My Electron app",
            "main": "main.js",
            "scripts": {
                "start": "electron ."
            },
            "agora_electron": {
                "platform": "darwin",
                "prebuilt": true
            },
            "dependencies": { "agora-electron-sdk": "latest" },
            "devDependencies": {"electron": "latest" }
        }
        ```
    
    * Windows

        ```json
        {
            "name": "electron-demo-app",
            "version": "0.1.0",
            "author": "your name",
            "description": "My Electron app",
            "main": "main.js",
            "scripts": {
                "start": "electron ."
            },
            "agora_electron": {
                "platform": "win32",
                "prebuilt": true,
                "arch": "ia32"
            },
            "dependencies": { "agora-electron-sdk": "latest" },
            "devDependencies": {"electron": "latest" }
        }
        ```

    For a description of the parameters, refer to the following:

    * `agora-electron-sdk`: The version number of Agora Electron SDK. Set it to `latest` to get the latest version of the SDK. For other versions, see [`agora-electron-sdk`](https://www.npmjs.com/package/agora-electron-sdk).
    * `electron`: The version number of Electron, supports version 5.0.0 and above (including the latest version). If you use a macOS device equipped with an M1 chip, set the version number to 11.0.0 or above.
    * `platform`: (Optional) The default is selected according to the system. For macOS use `darwin`, for Windows use `win32`.
    * `prebuilt`: (Optional) The default setting is `true` to prevent incompatibility between the Electron or Node.js version and the SDK.
    * `arch`: (Optional) Automatically selected by system by default.

    <Admonition title="Note" style="info">Different versions of Electron have different environment requirements. If you see errors due to a mismatched environment when running the project, refer to the official Electron documentation to choose the appropriate Electron and Node.js versions.</Admonition>

1. To install the dependencies, open Terminal in the project root folder and run the following commands:

    * macOS
        ```bash
        npm install
        ```

    * Windows
        ```bash
        npm install -D --arch=ia32 electron
        npm install
        ```

### Create a user interface

Add the following code to `index.html` to create a basic user interface that contains a local video window and a remote video window:

<details>
<summary>Sample code to create the user interface</summary>
```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>Electron Quickstart</title>
  </head>
  <body>
    <h1>Electron Quickstart</h1>
    <!--Add a local video window to the interface -->
    <div
      id="join-channel-local-video"
      style="width: 300px; height: 300px; float: left"
    ></div>
    <!--Add a remote video window to the interface -->
    <div
      id="join-channel-remote-video"
      style="width: 300px; height: 300px; float: left"
    ></div>
  </body>
  <script src="./renderer.js"></script>
</html>
```
</details>
