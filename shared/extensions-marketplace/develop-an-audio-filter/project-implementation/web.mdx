- Required basic classes:
 - `AudioExtension`: Implements initialization-related functions, including creating an extension, setting log reports, and setting event reports.
 - `AudioProcessor`: Implements audio processing capabilities, including receiving, processing, and delivering audio data.
- Optional auxiliary classes:
 - `Ticker`: Helps manage periodic tasks.
 - `Logger`: Uploads logs to the Agora SDK.
 - `Reporter`: TReports events to the Agora SDK.
