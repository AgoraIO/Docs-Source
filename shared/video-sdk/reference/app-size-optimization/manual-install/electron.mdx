
<PlatformWrapper platform="electron">

You use NPM to install <Vpd k="SDK" />. To do this:

1. Open a terminal window in your project folder and execute the following command:

    ``` sh
    npm i agora-electron-sdk
    ```

See [SDKs](../../sdks) for the latest Electron downloads.
</PlatformWrapper>

<PlatformWrapper platform="windows">

To install <Vpd k="SDK" /> in your Windows Forms `.net core` or `.net framework` project:

1. In Solution Explorer, right click **Dependencies**. Click **Manage Nuget Packages** > **Browse**, then type *agora_rtc_sdk*.
1. Select the Latest stable version, then click **Install**.

The <Vpd k="SDK" /> is now added to your project.

See [SDKs](../../sdks) for the latest Windows downloads.

</PlatformWrapper>
