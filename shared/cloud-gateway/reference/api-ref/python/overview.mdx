## RTCConnection Class

| Method | Description |

| [`create_data_stream`](./rtcconnection#createdatastream) | Creates a data stream. |

## IRTCConnectionObserver Class

| Callback | Description |
| --- | --- |
| [`on_connected`](./irtcconnectionobserver#onconnected) | Occurs when the SDK connection state changes to `3`. |
| [`on_disconnected`](./irtcconnectionobserver#ondisconnected) | Occurs when the SDK connection state changes to `1`. |
| [`on_token_privilege_will_expire`](./irtcconnectionobserver#ontokenprivilegewillexpire) | Occurs when the token will expire in 30 seconds. |
| [`on_token_privilege_did_expire`](./irtcconnectionobserver#ontokenprivilegedidexpire) | Occurs when the token has expired. |
| [`on_user_joined`](./irtcconnectionobserver#onuserjoined) | Occurs when a remote user joins the channel. |
| [`on_user_left`](./irtcconnectionobserver#onuserleft) | Occurs when a remote user leaves the channel. |
| [`on_stream_message_error`](./irtcconnectionobserver#onstreammessageerror) | Occurs when there is an error receiving a data stream message. |
| [`on_error`](./irtcconnectionobserver#onerror) | Occurs when an error occurs. |
