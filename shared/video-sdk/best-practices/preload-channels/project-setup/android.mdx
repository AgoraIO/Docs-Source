### Integrate the VideoLoader API

Integrate the <Vg k="COMPANY" /> VideoLoader API into your project. Add the files in the [VideoLoaderAPI](https://github.com/AgoraIO-Community/VideoLoaderAPI/tree/main/Android/lib_videoloaderapi/src/main/java/io/agora/videoloaderapi) directory to your project as follows:

    - `OnLiveRoomItemTouchEventHandler`
    - `OnPageScrollEventHandler.kt`
    - `OnPageScorllEventHandler2.kt`
    - `OnRoomListScrollEventHandler.kt`
    - `VideoLoader.kt`
    - `VideoLoaderImpl.kt`

<Admonition type="info" title="Note">To facilitate subsequent code upgrades, do not modify the names and paths of the files you added.</Admonition>
