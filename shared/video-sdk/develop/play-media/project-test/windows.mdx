3.  In Visual Studio, in `AgoraImplementation.h`, update `appId`, `channelName` and `token` with the values for your temporary token.

4.  In Visual Studio, click **Local Windows Debugger**. A moment later you see the project running on your development device.

    If this is the first time you run the project, you need to grant microphone and camera access to your <Vpl k="CLIENT"/>.

<ProductWrapper product="interactive-live-streaming">
5. Select an option and click **Join** to start a session.
    - When you join as a **Host**, the local video is published and played in the <Vpl k="CLIENT"/>.
    - When you join as **Audience**, the remote stream is subscribed and played.
</ProductWrapper>

<ProductWrapper product="video-calling">
5. . Click **Join** to start a call. Now, you can see yourself on the device screen and talk to the remote user using your <Vpl k="CLIENT"/>.
</ProductWrapper>

6.  Press **Open Media File**.

    After a short while, you see the button text changes confirming that the media file is opened successfully.

7.  Press **Play Media File**

    You see the media file played both locally and in the web demo app. The progress bar indicates the play progress.

8.  Press **Pause**

    Media publishing is paused and the camera and microphone publishing is resumed.

9.  Press **Resume**

    You see that the media file resumes playing.

10.  Wait for the media file to finish playing. When the progress bar reaches the end, the media player publishing is stopped and camera and microphone publishing is resumed.

</PlatformWrapper>
