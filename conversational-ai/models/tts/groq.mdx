Groq provides fast, reliable text-to-speech (TTS) with customizable voices.

### Sample configuration

The following example shows a starting `tts` parameter configuration you can use when you [Start a conversational AI agent](/conversational-ai/rest-api/agent/join).

```json
"tts": {
  "vendor": "groq",
  "params": {
    "api_key": "<groq_key>",
    "model": "playai-tts",
    "voice": "Arista-PlayAI"
  }
}
```

<Admonition type="caution">
The parameters listed on this page are validated for use with <Vpd k="NAME" />. To avoid unpredictable behavior, <Vg k="COMPANY" /> strongly recommends using only the supported parameters. For a complete reference, consult the [Groq documentation](https://console.groq.com/docs/api-reference#models-list).
</Admonition>

### Key parameters

<ParameterList title="tts" required={true}>
  <Parameter name="api_key" type="string" required={true}>
    The Groq API key used to authenticate requests. You must provide a valid key for the service to function.
  </Parameter>
  <Parameter name="model" type="string" required={true}>
    The Groq TTS model to use. For example, `playai-tts`.
  </Parameter>
  <Parameter name="voice" type="string" required={true}>
    The voice preset to use for speech synthesis. For example, `Arista-PlayAI`.
  </Parameter>
</ParameterList>
