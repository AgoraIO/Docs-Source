## 1.8.0

v1.8.0 was released on March xx, 2023 with the following features:

- Audio/video two-way communication between IoT SDK instances.
- Audio/video two-way communication between IoT SDK instances and RTC Native SDK instances.
- Audio/video two-way communication between IoT SDK instances and RTC Web SDK instances.
- Providing a target bitrate per bandwidth estimation (BWE) algorithms.
- Requesting key frames from remote users.
- Muting and unmuting local and remote media streams.
- Monitoring the transmission state of remote media streams.
