    ``` bash
    git clone https://github.com/AgoraIO/Tools <your download directory>
    ```

2.  **Add the <Vg k="COMPANY" /> IO tools code to your project**

    Copy the `DynamicKey/AgoraDynamicKey/python3/src` folder from the repository into the `src` folder of your Python project.

    ``` bash
    cp "<your download directory>/DynamicKey/AgoraDynamicKey/python3/src/*" "<your project folder>/src"
    ```

3.  **Import the <Vg k="COMPANY" /> token builder classes into your project**

    Add the following `import` statements at the beginning of the Python source file where you wish to generate the authentication token:

    ``` python
    from src.RtcTokenBuilder import RtcTokenBuilder, Role_Publisher, time
    ```

### Implement token generation

<Vg k="AGORA_BACKEND" /> supports both integer user Ids and string user accounts for token generation. To ensure smooth communication, all the users in a channel must use the same type of Id, that is, either the integer `uid`, or a string `userAccount`. Depending on user type, you generate a token using either:

-   `buildTokenWithUid` for an RTC token based on `uid`

-   `buildTokenWithAccount` for an RTC token based on `userAccount`

To generate tokens based on user ID and user account, add the following after the import statements of the Python source file where you wish to generate the authentication token:

``` python
appId = "Your app Id"
appCertificate = "Your app certificate"
channelName = "Your channel name"
uid = 0  # The integer uid, required for an RTC token
userAccount = "User account"  # The user account string for an RTC token
expirationTimeInSeconds = 3600  # The time after which the token expires


def generate_token():
    #  Calculate the time expiry timestamp
    current_timestamp = int(time.time())
    privilege_expired_ts = current_timestamp + expirationTimeInSeconds

    print("UID token:")
    token = RtcTokenBuilder.buildTokenWithUid(appId, appCertificate, channelName, uid, Role_Publisher, privilege_expired_ts)
    print(token)

    print("User account token:")
    token = RtcTokenBuilder.buildTokenWithAccount(appId, appCertificate, channelName, userAccount, Role_Publisher, privilege_expired_ts)
    print(token)


generate_token()
```
