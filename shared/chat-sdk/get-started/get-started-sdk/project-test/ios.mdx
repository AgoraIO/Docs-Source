Use Xcode to compile and run the project on an iOS device or emulator. The following user interface appears.

![1641976387226](https://web-cdn.agora.io/docs-files/1641976387226)

To try sending and receiving text messages, follow the steps on two clients:

1. Enter a username (such as `Localuser` and `Remoteuser`) and a password (such as `123456`) and click `Create user` on each client to create two Chat users.

2. Log into Chat as `Localuser`, type `Remoteuser` for `Peer username`, and send a text message.

   ![1641976375607](https://web-cdn.agora.io/docs-files/1641976375607)

3. Log into Chat as `Remoteuser` and check logs to see whether this message is received.

   ![1641976362744](https://web-cdn.agora.io/docs-files/1641976362744)
