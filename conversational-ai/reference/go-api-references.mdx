### `Join`

Creates an agent instance and joins the specified RTC channel.

```go
func (c *Client) Join(ctx context.Context, name string, payload *req.JoinPropertiesReqBody) (*resp.JoinResp, error)
```

#### Parameters

| Name     | Type                            | Description |
|----------|----------------------------------|-------------|
| `ctx`    | `Context`                        | Request context for controlling the lifecycle. |
| `name`   | `string`                         | Unique identifier for the agent. Must not be reused. |
| `payload`| `*req.JoinPropertiesReqBody`     | Agent configuration, including channel details, tokens, LLM and TTS settings. See [`JoinPropertiesReqBody`](#joinpropertiesreqbody). |

#### Returns

- `*resp.JoinResp`: Response data on successful call. See [`JoinResp`](#joinresp).
- `error`: An error object if the call fails.

### `JoinPropertiesCustomLLMBody`

Configure custom language model (LLM) settings when an agent joins an RTC channel.

```go
type JoinPropertiesCustomLLMBody struct {
	Url string `json:"url"`
	APIKey string `json:"api_key"`
	SystemMessages []map[string]any `json:"system_messages"`
	Params map[string]any `json:"params"`
	MaxHistory *int `json:"max_history,omitempty"`
	GreetingMessage string `json:"greeting_message,omitempty"`
	InputModalities []string `json:"input_modalities,omitempty"`
	OutputModalities []string `json:"output_modalities,omitempty"`
	FailureMessage string `json:"failure_message,omitempty"`
}
```

#### Parameters

| Parameter         | Type                  | Description |
|------------------|-----------------------|-------------|
| `Url`            | `string`              | (Required) LLM callback URL. Must be compatible with the OpenAI protocol. |
| `APIKey`         | `string`              | (Required) LLM API key for authentication. Default value is empty, please make sure to enable API key in production environment. |
| `SystemMessages` | `[]map[string]any`    | (Optional) A set of predefined information appended at the beginning of each LLM call to control the LLM output. Can be role settings, prompts and answer examples, must be compatible with the OpenAI protocol. |
| `Params`         | `map[string]any`      | (Optional) Additional information transmitted in the LLM message body, such as the model used, maximum token limit, etc. Different LLM vendors support different configurations, see their respective LLM documentation for details. |
| `MaxHistory`     | `*int`                | (Optional) Number of short-term memory entries to cache in the LLM. Default is 10. Pass 0 to not cache short-term memory. Agents and subscribers will log entries separately. |
| `GreetingMessage`| `string`              | (Optional) Agent greeting message. If filled in, when there is no user in the `remote_rtc_uids` list, the agent will automatically send a greeting message to the first subscribed user who joins the channel. |
| `InputModalities`| `[]string`            | (Optional) The input mode for the LLM. <br/> - `["text"]`: Text only (default). <br/> - `["text", "image"]`: Text and images, requires the selected LLM to support visual mode input. |
| `OutputModalities`| `[]string`           | (Optional) The output mode of the LLM. <br/> - `["text"]`: Text only (default), the output text will be converted to speech by the TTS module and published to the RTC channel. <br/> - `["audio"]`: Audio only. Audio will be published directly to the RTC channel. <br/> - `["text", "audio"]`: Text and audio. You can write your own logic to process LLM output as needed. |
| `FailureMessage` | `string`              | (Optional) Agent failure message. If filled in, when the LLM call fails, it will be returned through the TTS module. |

### `JoinPropertiesVadBody`

The Voice Activity Detection (VAD) configuration for the agent to join the RTC channel.

```go
type JoinPropertiesVadBody struct {
	InterruptDurationMs *int `json:"interrupt_duration_ms,omitempty"`
	PrefixPaddingMs *int `json:"prefix_padding_ms,omitempty"`
	SilenceDurationMs *int `json:"silence_duration_ms,omitempty"`
	Threshold *float64 `json:"threshold,omitempty"`
}
```

#### Parameters

| Parameter            | Type      | Description |
|----------------------|-----------|-------------|
| `InterruptDurationMs` | `int`    | (Optional) Voice duration threshold (in milliseconds). The range is `[120, 1200]`. <br/><br/><Admonition type="Notice" title="Notice">Minimum duration of detected vocal signal to avoid false interruptions.</Admonition> |
| `PrefixPaddingMs`     | `int`    | (Optional) Prefix fill threshold in milliseconds, in the range `[0, 5000]`. <br/><br/><Admonition type="Caution" title="Notice">Minimum duration of continuous speech required to start a new speech segment, to avoid very short sounds triggering voice activity detection.</Admonition> |
| `SilenceDurationMs`   | `int`    | (Optional) Silence duration threshold in milliseconds, in the range `[0, 2000]`. <br/><br/><Admonition type="Caution" title="Notice">Minimum silence duration to ensure that a short pause at the end of speech does not prematurely end the speech segment.</Admonition> |
| `Threshold`           | `float64`| (Optional) Speech recognition sensitivity, the range is `(0.0, 1.0)`. <br/><br/><Admonition type="Caution" title="Notice">Determines the degree to which an audio signal is considered "voice activity". Lower values make it easier for the agent to detect speech, higher values may ignore faint sounds.</Admonition> |

### `JoinPropertiesReqBody`

The body of the request to call the conversational agent engine's [`Join`](#join) API.

```go
type JoinPropertiesReqBody struct {
	Token string `json:"token"`
	Channel string `json:"channel"`
	AgentRtcUId string `json:"agent_rtc_uid"`
	RemoteRtcUIds []string `json:"remote_rtc_uids"`
	EnableStringUId *bool `json:"enable_string_uid,omitempty"`
	IdleTimeout *int `json:"idle_timeout,omitempty"`
	AgentRtmUId *string `json:"agent_rtm_uid,omitempty"`
	AdvancedFeatures *JoinPropertiesAdvancedFeaturesBody `json:"advanced_features,omitempty"`
	LLM *JoinPropertiesCustomLLMBody `json:"llm,omitempty"`
	TTS *JoinPropertiesTTSBody `json:"tts,omitempty"`
	Vad *JoinPropertiesVadBody `json:"vad,omitempty"`
	Asr *JoinPropertiesAsrBody `json:"asr,omitempty"`
}
```

#### Parameters

| Parameter           | Type                                  | Description |
|---------------------|----------------------------------------|-------------|
| `Token`             | `string`                               | Token used to join the RTC channel. If App Certificate is enabled, provide the project's dynamic key. |
| `Channel`           | `string`                               | (Required) Name of the RTC channel that the agent joins. |
| `AgentRtcUId`       | `string`                               | (Required) Agent user ID in the RTC channel. Use `"0"` for random assignment (Token must match). |
| `RemoteRtcUIds`     | `[]string`                             | (Required) User IDs the agent subscribes to in the RTC channel. `"*"` subscribes to all users. |
| `EnableStringUId`   | `*bool`                                | (Optional) Enable string UIDs:<br/>- `true`: Enable<br/>- `false`: Disable (default) |
| `IdleTimeout`       | `*int`                                 | (Optional) Max idle time (in seconds). After all `remote_rtc_uids` leave, the agent exits. `0` disables auto-exit. |
| `AgentRtmUId`       | `*string`                              | RTM user ID for the agent. Valid only if `advanced_features.enable_rtm` is `true`. |
| `AdvancedFeatures`  | `*JoinPropertiesAdvancedFeaturesBody`  | (Optional) Advanced feature config. See [`JoinPropertiesAdvancedFeaturesBody`](#joinpropertiesadvancedfeaturesbody). |
| `LLM`               | `*JoinPropertiesCustomLLMBody`         | (Required) Custom LLM config. See [`JoinPropertiesCustomLLMBody`](#joinpropertiescustomllmbody). |
| `TTS`               | `*JoinPropertiesTTSBody`               | (Required) TTS module config. See [`JoinPropertiesTTSBody`](#joinpropertiesttsbody). |
| `Vad`               | `*JoinPropertiesVadBody`               | (Optional) VAD config. See [`JoinPropertiesVadBody`](#joinpropertiesvadbody). |
