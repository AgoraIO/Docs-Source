### Create a project

To create a new ReactJS project with the necessary dependencies:

1. Ensure that you have installed [Node.js LTS](https://nodejs.org/en) and `npm`.

1. Open a terminal and execute:

    ```bash
    npm create vite@latest agora-sdk-quickstart — — template react-ts
    ```

    This creates a folder named `agora-sdk-quickstart`.

### Integrate the SDK

Use one of the following methods to integrate the project dependencies:

<Tabs>
<TabItem value="npm" label="NPM" default>

    To install the <Vg k="COMPANY" /> SDK using `npm`, navigate to the project folder and run the following command:

<CodeBlock language="bash">
{`npm i agora-rtc-react`}
</CodeBlock>
</TabItem>

<TabItem value="cdn" label="CDN">

To integrate React JS dependencies and the <Vg k="COMPANY" /> SDK using CDN, add the following code to your HTML file:

<CodeBlock language="html" showLineNumbers>
{`<!-- Include the React development libraries (must be introduced in this order) -->
<script crossorigin src="https://unpkg.com/react@18/umd/react.development.js"></script>
<script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.development.js"></script>

<!-- If the above React libraries have already been integrated through npm, skip them and include only the Agora RTC React SDK -->
<script src="https://download.agora.io/sdk/release/agora-rtc-react.2.3.0.js"></script>`}
</CodeBlock>

</TabItem>    
</Tabs>

### Create a root element

In the project root folder, update `index.html` to create a root element:

```html
<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Agora React Quickstart</title>
  </head>
  <body>
    <div id="root"></div>
    <script type="module" src="/src/index.jsx"></script>
  </body>
</html>
```
