This page shows you how to build a server that starts and stops a voice AI agent using the Agora Conversational AI SDKs. The SDKs wrap the Conversational AI REST API with typed vendor configuration, automatic token generation, and a session lifecycle API.

## Prerequisites

Before you begin, make sure you have: 

- [Enabled Agora Conversational AI](../get-started/manage-agora-account) for your project.
- Completed the [Voice Calling](/voice-calling/get-started/get-started-sdk) or [Video Calling](/video-calling/get-started/get-started-sdk) quickstart. 
- Your **App ID** and **App Certificate** from [Agora Console](https://console.agora.io). 
- API keys for your chosen STT, LLM, and TTS providers. **Not required if using Agora-managed models.**

## Implementation

The following steps set up a client, configure an agent, and manage session lifecycle.

### Create a client

<Tabs groupId="language">
<TabItem value="typescript" label="Node">

`AgoraClient` holds your credentials and handles authentication. Pass your App ID and App Certificate. The SDK generates a channel-scoped token automatically before each session starts.

```typescript
import { AgoraClient, Area } from 'agora-agent-server-sdk';

const client = new AgoraClient({
  area: Area.US,
  appId: process.env.AGORA_APP_ID!,
  appCertificate: process.env.AGORA_APP_CERT!,
});
```

Use the region closest to your users: `Area.US`, `Area.EU`, `Area.AP`, or `Area.CN`.
</TabItem>
<TabItem value="python" label="Python">

`Agora` holds your credentials and handles authentication. Pass your App ID and App Certificate. The SDK generates a channel-scoped token automatically before each session starts.

<Admonition type="info">
For async usage, use `AsyncAgora` in place of `Agora`. All session methods have equivalent async signatures. See the [Python SDK reference](/conversational-ai/reference/SDK/python) for details.
</Admonition>

```python
import os
from agora_agent import Agora, Area

client = Agora(
    area=Area.US,
    app_id=os.environ['AGORA_APP_ID'],
    app_certificate=os.environ['AGORA_APP_CERT'],
)
```

Use the region closest to your users: `Area.US`, `Area.EU`, `Area.AP`, or `Area.CN`.
</TabItem>
<TabItem value="go" label="Go">

In Go, the client handles authentication for REST API calls, while your App ID and App Certificate are passed separately when creating a session. The session uses them to auto-generate a temporary token for the Agora channel.


Use the region closest to your users: `option.AreaUS`, `option.AreaEU`, `option.AreaAP`, or `option.AreaCN`.

### Define an agent

The SDK supports two approaches for vendor configuration:

- **Agora-managed models**: Agora provides the credentials automatically. You do not need to sign up for vendor accounts or provide API keys. To use Agora managed models, omit the vendor API key. 
- **BYOK**: Supply your own vendor API keys to use any model with your own vendor accounts.

<Tabs groupId="language">
<TabItem value="typescript" label="Node">

```typescript
import { Agent, DeepgramSTT, OpenAI, MiniMaxTTS } from 'agora-agent-server-sdk';

const agent = new Agent({
  instructions: 'You are a helpful voice assistant. Keep responses concise.',
  greeting: 'Hello! How can I help you today?',
})
  .withStt(new DeepgramSTT({
    // apiKey: process.env.ASR_API_KEY!, // Uncomment to use your own Deepgram key
    model: 'nova-3',
    language: 'en',
  }))
  .withLlm(new OpenAI({
    // apiKey: process.env.LLM_API_KEY!, // Uncomment to use your own OpenAI key
    model: 'gpt-4o-mini',
  }))
  .withTts(new MiniMaxTTS({
    // key: process.env.TTS_API_KEY!,    // Uncomment to use your own MiniMax key
    model: 'speech-2.6-turbo',
    voiceId: 'English_captivating_female1',
  }));
```

For the full list of `Agent` options and vendor classes, see the [SDK reference](/conversational-ai/reference/SDK/node).

### Start a session

Bind the agent to a specific RTC channel and start the session. The SDK generates a channel-scoped token automatically from your App ID and App Certificate.

<Tabs groupId="language">
<TabItem value="typescript" label="Node">

```typescript
import { ExpiresIn } from 'agora-agent-server-sdk';

const session = agent.createSession(client, {
  channel: 'your-channel-name',
  agentUid: '1',
  remoteUids: ['100'],
  idleTimeout: 120,
  expiresIn: ExpiresIn.hours(1),
});

const agentId = await session.start();
console.log('Agent started:', agentId);
```

`start()` transitions the session through `idle → starting → running` states and returns the `agentId`. You can check the session state at any time with `session.status`.

</TabItem>
<TabItem value="python" label="Python">

```python
from agora_agent.agentkit import expires_in_hours

session = agent.create_session(
    client,
    channel='your-channel-name',
    agent_uid='1',
    remote_uids=['100'],
    idle_timeout=120,
    expires_in=expires_in_hours(1),
)

session.on('started', lambda data: print('Agent is live:', data['agent_id']))
session.on('stopped', lambda data: print('Agent has left:', data['agent_id']))
session.on('error', lambda err: print('Session error:', err))

agent_id = session.start()
```

Register event handlers before calling `start()`. The session state transitions through `idle → starting → running` and returns the agent ID. You can check the current session state at any time with `session.status`.

</TabItem>
<TabItem value="go" label="Go">

```go
import (
    "context"
    "fmt"
    "log"
    "os"

    "github.com/AgoraIO-Conversational-AI/agent-server-sdk-go/agentkit"
)

session := agentkit.NewAgentSession(agentkit.AgentSessionOptions{
    Client:         c.Agents,
    Agent:          agent,
    AppID:          os.Getenv("AGORA_APP_ID"),
    AppCertificate: os.Getenv("AGORA_APP_CERT"),
    Channel:        "your-channel-name",
    AgentUID:       "1",
    RemoteUIDs:     []string{"100"},
})

session.On("started", func(data interface{}) {
    info := data.(map[string]string)
    fmt.Println("Agent is live:", info["agent_id"])
})
session.On("stopped", func(data interface{}) {
    fmt.Println("Agent has left")
})
session.On("error", func(data interface{}) {
    log.Println("Session error:", data)
})

ctx := context.Background()
agentId, err := session.Start(ctx)
if err != nil {
    log.Fatalf("Failed to start session: %v", err)
}
```

Register event handlers before calling `Start()`. The session state transitions through `idle → starting → running` and returns the agent ID. You can check the current session state at any time with `session.Status()`.

</TabItem>

### Interact with a running session

Once a session is in the `running` state, you can instruct the agent or update its configuration mid-session.

<Tabs groupId="language">
<TabItem value="typescript" label="Node">

```typescript
// Make the agent speak a specific message
await session.say('One moment while I look that up for you.');

// Interrupt the agent mid-speech
await session.interrupt();

// Update agent configuration without restarting the session.
// Accepts a partial properties object in REST API format.
await session.update({
  llm: {
    system_messages: [{ role: 'system', content: 'You are now in concise mode.' }],
  },
});
```

</TabItem>
<TabItem value="python" label="Python">

```go
if err != nil {
    log.Printf("Update failed: %v", err)
}
```

## Complete example

The following is a minimal server your client app can call to manage agent sessions.

<Tabs groupId="language">
<TabItem value="typescript" label="Node">

<details>
<summary>Express server example</summary>

```typescript
import 'dotenv/config';
import express from 'express';
import {
  AgoraClient,
  Agent,
  Area,
  AgoraError,
  DeepgramSTT,
  OpenAI,
  MiniMaxTTS,
  ExpiresIn,
} from 'agora-agent-server-sdk';

const app = express();
app.use(express.json());

const client = new AgoraClient({
  area: Area.US,
  appId: process.env.AGORA_APP_ID!,
  appCertificate: process.env.AGORA_APP_CERT!,
});

// Agora-managed models: Omit API keys for supported models and Agora provides
// credentials automatically. To switch to BYOK, uncomment the API key lines.
const agent = new Agent({
  instructions: 'You are a helpful voice assistant. Keep responses concise.',
  greeting: 'Hello! How can I help you today?',
})
  .withStt(new DeepgramSTT({
    // apiKey: process.env.ASR_API_KEY!, // Uncomment to use your own Deepgram key
    model: 'nova-3',
    language: 'en',
  }))
  .withLlm(new OpenAI({
    // apiKey: process.env.LLM_API_KEY!, // Uncomment to use your own OpenAI key
    model: 'gpt-4o-mini',
  }))
  .withTts(new MiniMaxTTS({
    // key: process.env.TTS_API_KEY!,    // Uncomment to use your own MiniMax key
    model: 'speech-2.6-turbo',
    voiceId: 'English_captivating_female1',
  }));

const sessions = new Map<string, ReturnType<typeof agent.createSession>>();

app.post('/sessions/start', async (req, res) => {
  const { channel, userId } = req.body;

  const session = agent.createSession(client, {
    channel,
    agentUid: '1',
    remoteUids: [userId],
    idleTimeout: 120,
    expiresIn: ExpiresIn.hours(1),
  });

  session.on('error', (err) => console.error(`Session error [${channel}]:`, err));

  try {
    const agentId = await session.start();
    sessions.set(agentId, session);
    res.json({ agentId });
  } catch (err) {
    if (err instanceof AgoraError) {
      res.status(err.statusCode).json({ error: err.message });
    } else {
      res.status(500).json({ error: 'Failed to start agent' });
    }
  }
});

app.post('/sessions/stop', async (req, res) => {
  const { agentId } = req.body;
  const session = sessions.get(agentId);

  if (!session) {
    res.status(404).json({ error: 'Session not found' });
    return;
  }

  try {
    await session.stop();
    sessions.delete(agentId);
    res.json({ status: 'stopped' });
  } catch (err) {
    if (err instanceof AgoraError) {
      res.status(err.statusCode).json({ error: err.message });
    } else {
      res.status(500).json({ error: 'Failed to stop agent' });
    }
  }
});

app.listen(3000, () => console.log('Server listening on port 3000'));
```

```python
# Requires Flask: pip install flask
import os
from flask import Flask, request, jsonify
from agora_agent import Agora, Agent, Area
from agora_agent.agentkit import expires_in_hours
from agora_agent.agentkit.vendors import DeepgramSTT, OpenAI, MiniMaxTTS

app = Flask(__name__)

client = Agora(
    area=Area.US,
    app_id=os.environ['AGORA_APP_ID'],
    app_certificate=os.environ['AGORA_APP_CERT'],
)

# Agora-managed models: Omit API keys for supported models and Agora provides
# credentials automatically. To switch to BYOK, uncomment the API key lines.
agent = (
    Agent(
        name='support-assistant',
        instructions='You are a helpful voice assistant. Keep responses concise.',
        greeting='Hello! How can I help you today?',
    )
    .with_stt(DeepgramSTT(
        # api_key=os.environ['ASR_API_KEY'],  # Uncomment to use your own Deepgram key
        model='nova-3',
        language='en',
    ))
    .with_llm(OpenAI(
        # api_key=os.environ['LLM_API_KEY'],  # Uncomment to use your own OpenAI key
        model='gpt-4o-mini',
    ))
    .with_tts(MiniMaxTTS(
        # key=os.environ['TTS_API_KEY'],       # Uncomment to use your own MiniMax key
        model='speech-2.6-turbo',
        voice_id='English_captivating_female1',
    ))
)

sessions = {}

@app.post('/sessions/start')
def start_session():
    body = request.get_json()
    channel = body['channel']
    user_id = body['userId']

    session = agent.create_session(
        client,
        channel=channel,
        agent_uid='1',
        remote_uids=[user_id],
        idle_timeout=120,
        expires_in=expires_in_hours(1),
    )
    session.on('error', lambda err: print(f'Session error [{channel}]:', err))

    try:
        agent_id = session.start()
        sessions[agent_id] = session
        return jsonify({'agentId': agent_id})
    except Exception as err:
        return jsonify({'error': str(err)}), 500

@app.post('/sessions/stop')
def stop_session():
    body = request.get_json()
    agent_id = body['agentId']
    session = sessions.get(agent_id)

    if not session:
        return jsonify({'error': 'Session not found'}), 404

    try:
        session.stop()
        sessions.pop(agent_id)
        return jsonify({'status': 'stopped'})
    except Exception as err:
        return jsonify({'error': str(err)}), 500

if __name__ == '__main__':
    app.run(port=3000)
```

```go
// Requires: go get github.com/AgoraIO-Conversational-AI/agent-server-sdk-go
package main

import (
    "context"
    "encoding/json"
    "fmt"
    "log"
    "net/http"
    "os"
    "sync"

    agora "github.com/AgoraIO-Conversational-AI/agent-server-sdk-go"
    "github.com/AgoraIO-Conversational-AI/agent-server-sdk-go/agentkit"
    "github.com/AgoraIO-Conversational-AI/agent-server-sdk-go/agentkit/vendors"
    "github.com/AgoraIO-Conversational-AI/agent-server-sdk-go/client"
    "github.com/AgoraIO-Conversational-AI/agent-server-sdk-go/option"
)

var (
    sessions   = make(map[string]*agentkit.AgentSession)
    sessionsMu sync.Mutex
)

func main() {
    c := client.NewClient(
        option.WithArea(option.AreaUS),
    )

    // Use ASR_API_KEY, LLM_API_KEY, and TTS_API_KEY environment variables.
    agent := agentkit.NewAgent(
        agentkit.WithName("support-assistant"),
        agentkit.WithInstructions("You are a helpful voice assistant. Keep responses concise."),
        agentkit.WithGreeting("Hello! How can I help you today?"),
    ).WithStt(
        vendors.NewDeepgramSTT(vendors.DeepgramSTTOptions{
            APIKey:   os.Getenv("ASR_API_KEY"),
            Model:    "nova-3",
            Language: "en",
        }),
    ).WithLlm(
        vendors.NewOpenAI(vendors.OpenAIOptions{
            APIKey: os.Getenv("LLM_API_KEY"),
            Model:  "gpt-4o-mini",
        }),
    ).WithTts(
        vendors.NewOpenAITTS(vendors.OpenAITTSOptions{
            APIKey: os.Getenv("TTS_API_KEY"),
            Voice:  "shimmer",
            Model:  "tts-1",
        }),
    )

    http.HandleFunc("/sessions/start", func(w http.ResponseWriter, r *http.Request) {
        var body struct {
            Channel string `json:"channel"`
            UserID  string `json:"userId"`
        }
        if err := json.NewDecoder(r.Body).Decode(&body); err != nil {
            http.Error(w, `{"error":"invalid request"}`, http.StatusBadRequest)
            return
        }

        session := agentkit.NewAgentSession(agentkit.AgentSessionOptions{
            Client:         c.Agents,
            Agent:          agent,
            AppID:          os.Getenv("AGORA_APP_ID"),
            AppCertificate: os.Getenv("AGORA_APP_CERT"),
            Channel:        body.Channel,
            AgentUID:       "1",
            RemoteUIDs:     []string{body.UserID},
            IdleTimeout:    agora.Int(120),
            ExpiresIn:      3600,
        })
        session.On("error", func(data interface{}) {
            log.Printf("Session error [%s]: %v", body.Channel, data)
        })

        agentId, err := session.Start(context.Background())
        if err != nil {
            http.Error(w, fmt.Sprintf(`{"error":"%v"}`, err), http.StatusInternalServerError)
            return
        }

        sessionsMu.Lock()
        sessions[agentId] = session
        sessionsMu.Unlock()

        w.Header().Set("Content-Type", "application/json")
        fmt.Fprintf(w, `{"agentId":"%s"}`, agentId)
    })

    http.HandleFunc("/sessions/stop", func(w http.ResponseWriter, r *http.Request) {
        var body struct {
            AgentID string `json:"agentId"`
        }
        if err := json.NewDecoder(r.Body).Decode(&body); err != nil {
            http.Error(w, `{"error":"invalid request"}`, http.StatusBadRequest)
            return
        }

        sessionsMu.Lock()
        session, ok := sessions[body.AgentID]
        sessionsMu.Unlock()

        if !ok {
            http.Error(w, `{"error":"session not found"}`, http.StatusNotFound)
            return
        }

        if err := session.Stop(context.Background()); err != nil {
            http.Error(w, fmt.Sprintf(`{"error":"%v"}`, err), http.StatusInternalServerError)
            return
        }

        sessionsMu.Lock()
        delete(sessions, body.AgentID)
        sessionsMu.Unlock()

        w.Header().Set("Content-Type", "application/json")
        fmt.Fprint(w, `{"status":"stopped"}`)
    })

    log.Println("Server listening on port 3000")
    log.Fatal(http.ListenAndServe(":3000", nil))
}
```
