6. **Test joining multiple channels**

    1. In `src/agora-manager/config.ts`, update `appId`, `channelName`, and `secondChannel`.

    2. [Generate a temporary token](../reference/manage-agora-account#generate-a-temporary-token) in <Vpl k="CONSOLE" /> using `appId` and `channelName`. Use it to update `rtcToken` in `config.ts`.

    3. Generate a second token in <Vpl k="CONSOLE" /> using `appId` and `secondChannel`. Use it to update `secondChannelToken` in `config.ts`.

    4. In your browser, navigate to the <Link target="_blank" to="{{Global.DEMO_BASIC_VIDEO_CALL_URL}}"><Vg k="COMPANY" /> web demo</Link> and update _App ID_, _Channel_, and _Token_ with the values for your temporary token, then click **Join**.

    5. Repeat the previous step on a second device, but this time use `appID` `secondChannel`, and `secondChannelToken` to **Join** the channel.

    1. Start the proxy server:

        In Terminal run the following command:
        ```bash
        node ./utils/proxy.js
        ```
    1. Start the dev server:

        Execute the following command in the terminal:
        ```bash
        yarn dev
        ```
        Use the URL displayed in the terminal to open the <Vpl k="CLIENT" /> in your browser.

    1. Select **Audience** 

        1. Press **Join**. 
        
            If this is the first time you run the project, you need to grant microphone and camera access to your <Vpl k="CLIENT" />. You see the video from the web browser demo app connected to `channel` in the remote view of your development device.

        2. Press **Join Second Channel**. 
        
            You see the video from the web browser demo app connected to `secondChannel` in the bottom container of your development device.
        3. Press **Leave Second Channel** and then **Leave** to exit both channels.

    1. Select **Host** 

        1. Press **Join**.

        2. Press **Join Second Channel**.
            
            You see the web browser demo app connected to `secondChannel` shows the video from your development device.

        3. Press **Leave Second Channel** and then **Leave** to exit both channels.
