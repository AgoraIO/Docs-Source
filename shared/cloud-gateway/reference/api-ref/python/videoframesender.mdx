The `VideoFrameSender` class.

### `send_video_frame`

Sends a YUV format video frame to the video track.

```python
def send_video_frame(self, frame:ExternalVideoFrame)
```

#### Parameters

| Parameter | Description |
|------|------|
| `frame` | The video frame to send, see [`ExternalVideoFrame`](#externalvideoframe). |

#### Returns

- 0: Success.
- < 0: Failure.
