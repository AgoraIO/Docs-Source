The <Vg k="CHAT_SDK"/> provides the message read receipt feature that allows the user, after sending a message, to know whether the message is read. The feature is available to both one-to-one chats and group chats.

- Message delivery receipt: Available only to <Vg k="CHAT_ONE_LOWER"/>s.
- Message read receipt: Available to both <Vg k ="CHAT_ONE_LOWER"/>s and <Vg k ="CHAT_GROUP_LOWER"/>s.

## Prerequisites

Before proceeding, ensure that you meet the following requirements:

- You have integrated the <Vg k="CHAT_SDK"/>, initialized the SDK and implemented the functionality of registering accounts and login. For details, see <Link to="/agora-chat/get-started/get-started-sdk">Chat <Vg k="GET_STARTED"/></Link>.
- You understand the API call frequency limits as described in [Limitations](/agora-chat/reference/limitations).
- Message read receipts for chat groups are not enabled by default. To use this feature, contact support@agora.io.
