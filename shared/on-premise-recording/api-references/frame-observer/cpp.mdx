## IRecorderVideoFrameObserver

Handles video frame callbacks for the recorder.

| API | Description |
|---|---|
|[`onYuvFrameCaptured`](#onyuvframecaptured) | Occurs when a YUV video frame is captured. |
|[`onEncodedFrameReceived`](#onencodedframereceived) | Callback for receiving an encoded video frame. |
|[`onJPGFileSaved`](#onjpgfilesaved) | Occurs when a captured JPG file is successfully saved. |

### onYuvFrameCaptured

```cpp
virtual void onYuvFrameCaptured(const char* channelId, user_id_t userId, const agora::media::base::VideoFrame *frame) = 0;
```

Triggered when a YUV video frame is captured. This occurs after calling [`enableRecorderVideoFrameCapture`](iagoramediartcrecorder#enablerecordervideoframecapture) with `VideoFrameCaptureType` set to `VIDEO_FORMAT_YUV_FRAME_TYPE`.

| Parameter | Data Type | Description |
|-----------|-----------|-------------|
| `channelId` | `const char*` | Channel ID. |
| `userId` | `user_id_t` | Remote user ID. |
| `frame` | `const agora::media::base::VideoFrame *` | The captured video frame. See [`VideoFrame`](struct#videoframe). |


### onEncodedFrameReceived

```cpp
virtual void onEncodedFrameReceived(const char* channelId, user_id_t userId, const uint8_t* imageBuffer, size_t length, agora::rtc::EncodedVideoFrameInfo videoEncodedFrameInfo) = 0;
```

Callback for receiving an encoded video frame. This occurs after calling [`enableRecorderVideoFrameCapture`](iagoramediartcrecorder#enablerecordervideoframecapture) with `VideoFrameCaptureType` set to either `VIDEO_FORMAT_ENCODED_FRAME_TYPE` or `VIDEO_FORMAT_JGP_FORMAT_TYPE`.

- For encoded frame type, triggered each captured frame.
- For JPG format type, triggered at the interval set in `RecorderVideoFrameCaptureConfig::jpgCaptureIntervalInSec`.

You can obtain the codec format from `EncodedVideoFrameInfo::codecType`.

| Parameter | Data Type | Description |
|-----------|-----------|-------------|
| `channelId` | `const char*` | Channel ID. |
| `userId` | `user_id_t` | Remote user ID. |
| `imageBuffer` | `const uint8_t*` | Pointer to the video image buffer. |
| `length` | `size_t` | Length of the video image data. |
| `videoEncodedFrameInfo` | `agora::rtc::EncodedVideoFrameInfo` | Information about the encoded video frame. See [`EncodedVideoFrameInfo`](struct#encodedvideoframeinfo). |



### onJPGFileSaved

```cpp
virtual void onJPGFileSaved(const char* channelId, user_id_t userId, const char* filename) = 0;
```

Triggered when a captured JPG file is successfully saved. This occurs at the interval set in `RecorderVideoFrameCaptureConfig::jpgCaptureIntervalInSec` after calling [`enableRecorderVideoFrameCapture`](iagoramediartcrecorder#enablerecordervideoframecapture) with `VideoFrameCaptureType` set to `VIDEO_FORMAT_JPG_FILE_TYPE`.

| Parameter | Data Type | Description |
|-----------|-----------|-------------|
| `channelId` | `const char*` | Channel ID. |
| `userId` | `user_id_t` | Remote user ID. |
| `filename` | `const char*` | Name of the saved JPG file in the format `$jpgFileStorePath/channelId_userId_YYYYMMDDHHMMSS_MS.jpg`. |
