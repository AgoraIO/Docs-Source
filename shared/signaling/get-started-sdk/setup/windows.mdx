To integrate the <Vpl k="NAME" /> <Vpd k="SDK" /> into your project:

1. [Download](/sdks?platform=windows) the latest version of the SDK and unzip it. 

1. Open Visual Studio and create a new **Console Application** in C++.

1. Copy the SDK files to your project:

    1. Copy the `sdk/high_level_api/include` folder into your project directory. For example, `MyProject/sdk/include`.
    1. Place the `.so` library files in a suitable directory, such as `MyProject/sdk/lib`.

1. Include the header files:

    1. Open your project in Visual Studio.
    1. Go to **Project Properties → C/C++ → General → Additional Include Directories**.
    1. Add the path to the `sdk/include` folder you just copied.

1. Link the libraries:

    1. Go to **Project Properties → Linker → General → Additional Library Directories**.
    1. Add the path to the `sdk/lib` folder containing the `.so` files.
    1. Under **Linker → Input → Additional Dependencies**, add the library names.
