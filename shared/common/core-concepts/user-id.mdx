#### User ID

<ProductWrapper notAllowed="signaling">
A User ID (UID) identifies a user in a channel. Each user in a channel should have a unique user ID. If you do not specify a user ID when the user joins a channel, a UID is automatically generated and assigned to the user.
</ProductWrapper>

<ProductWrapper product="signaling">
A user ID (UID) identifies a user in a channel. A user is a person or entity that logs into your <Vpl k="CLIENT" />.
Each user in a [project](#app-id) must have a globally unique UID.

The same UID cannot log in to <Vg k="AGORA_BACKEND" /> from multiple devices at the same time. If <Vpl k="CLIENT"
/>s  with the same UID logs in to <Vg k="AGORA_BACKEND" />, the <Vpl k="CLIENT" /> that logged in first
is disconnected and sent an event notification.

The UID is used for billing and online status notifications.
</ProductWrapper>
