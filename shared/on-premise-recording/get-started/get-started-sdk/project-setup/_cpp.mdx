### Integrate the SDK

1. Download the latest <Vg k="OPREC_SDK" /> package from the [SDKs](/sdks?platform=linux) page.

2. After extracting the package, the directory structure looks like this:

    ```
    .
    ├── agora_sdk
    │   ├── include
    │   ├── libagora-fdkaac.so
    │   ├── libagora_rtc_sdk.so
    │   └── libaosl.so
    └── example
        ├── CMakeLists.txt
        ├── build.sh
        ├── out
        ├── recorder
        ├── recorder.json
        ├── singleVideo.json
        ├── scripts
        └── third-party
    ```

3. Integrate the SDK into your project:

    * Import the header files from the `agora_sdk/include` directory into your development project.
    * Link the following dynamic library files from the `agora_sdk` directory:

        * `libagora-fdkaac.so`
        * `libagora_rtc_sdk.so`
        * `libaosl.so`

    The `example` directory in the SDK package includes a complete local recording sample project. To run these examples, see [Test the project](#test-the-project).
