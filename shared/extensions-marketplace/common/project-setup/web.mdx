To integrate an extension:

1. Use npm to install ([agora-rte-extension ](https://www.npmjs.com/package/agora-rte-extension)) extension development module:

   ```
   npm install --save agora-rte-extension
   ```

2. Import the required module. Add the following code to your `.js` file:

   ```
   import {AudioExtension, AudioProcessor} from 'agora-rte-extension'
   ```
