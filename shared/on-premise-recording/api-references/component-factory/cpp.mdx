## IAgoraMediaComponentFactory

| API | Description |
|---|---|
|[`createMediaRtcRecorder`](#createmediartcrecorder) | Creates an [`IAgoraMediaRtcRecorder`](#iagoramediartcrecorder) media recorder instance. |
|[`createAgoraMediaComponentFactory`](#createagoramediacomponentfactory) | Creates an `IMediaComponentFactory` object and returns a pointer. |

### createMediaRtcRecorder

```cpp
virtual agora_refptr<IAgoraMediaRtcRecorder> createMediaRtcRecorder() = 0;
```

Creates an [`IAgoraMediaRtcRecorder`](iagoramediartcrecorder) media recorder instance.

#### Return value

- If the method call succeeds, returns an [`IAgoraMediaRtcRecorder`](iagoramediartcrecorder) object.
- If the method call fails, returns a null pointer.

### createAgoraMediaComponentFactory

```cpp
AGORA_API agora::rtc::IMediaComponentFactory* AGORA_CALL createAgoraMediaComponentFactory();
```

Creates an `IMediaComponentFactory` object and returns a pointer.

#### Return value

- If the method call succeeds, returns a pointer to an `IMediaComponentFactory` object.
- If the method call fails, returns a null pointer.
