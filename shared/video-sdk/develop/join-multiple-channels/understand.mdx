<Vg k="VSDK" />'s multi-channel functionality is based on two key components: 

* <code>{props.rtcConnection}</code>

    The <code>{props.rtcConnection}</code> object identifies a connection. It contains the following information:

    - Channel name
    - User ID of the local user

    You create multiple <code>{props.rtcConnection}</code> objects, each with a different channel name and user ID. Each <code>{props.rtcConnection}</code> instance can independently publish multiple audio streams and a single video stream.

* <code>{props.rtcEngineEx}</code> 

    The class contains methods tailored for interacting with a designated <code>{props.rtcConnection}</code> object.

To join multiple channels, you call <code>{props.joinChannelEx}</code> method in the <code>{props.rtcEngineEx}</code> class multiple times, using a different <code>{props.rtcConnection}</code> instance each time. 

When joining multiple channels:

- Ensure that the user ID for each <code>{props.rtcConnection}</code> object is unique and nonzero.
- Configure publishing and subscribing options for the <code>{props.rtcConnection}</code> object in <code>{props.joinChannelEx}</code>.
