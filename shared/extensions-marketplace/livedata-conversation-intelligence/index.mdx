## Understand the tech

<PlatformWrapper platform="android">
The <Vg k="RTEE_LDCI" /> extension is an encapsulation of the real-time speech recognition and translation core API on the cloud. You can quickly integrate the <Vg k="RTEE_LDCI" /> capabilities by passing in the specified `key` and `value` parameters to the <Link to="{{Global.API_REF_ANDROID_ROOT}}/class_irtcengine.html#api_irtcengine_setextensionproperty">setExtensionProperty</Link> method provided in <Vg k="COMPANY" /> SDK v<Vg k="VSDK_RELEASE_API" />.

The key parameter of the `setExtensionProperty` method corresponds to the name of the API, and the value parameter wraps some or all of the parameters of the API in JSON format. Therefore, by passing in the specified `key` and `value` parameters, you can call the corresponding cloud API to realize the functions of <Vg k="RTEE_LDCI" />.
</PlatformWrapper>

## Prerequisites

Ensure that your development environment meets the following requirements:

<PlatformWrapper platform="android">
- Android Studio 4.1 or later.
- A physical device (not an emulator) running Android 5.0 or later.
</PlatformWrapper>

<PlatformWrapper platform="ios">
- Xcode 9.0 or later.
- A physical device (not an emulator) running iOS 9.0 or later.
</PlatformWrapper>
- <Vg k="RTEE_LDCI" /> is used with <Vg k="COMPANY" /> <Vg k="VSDK" /> v<Vg k="VSDK_RELEASE_API" />. 

    Refer to the [SDK quickstart](/video-calling/get-started/get-started-sdk) to integrate <Vg k="VSDK" /> v<Vg k="VSDK_RELEASE_API" /> and implement basic video calling.

## Project Setup

The <Vg k="RTEE_LDCI" /> extension provides a sample project on GitHub to help you get started quickly.

<ProjectSetup />

2. **Test translation and transcription features**

    Once you have installed the sample project on your device, follow these steps to test the translation and transcription features:

    1. Start the <Vpl k="CLIENT" />. Fill in the **channel name** in the input box, and click **Join**. 
    1. Click **Start Translation** to begin the translation. Speak into the device. You see the transcription and translation on the screen in real-time.
    1. Click **End Translation** to end transcription and translation.
    1. Click **End Plug-in** to stop using the extension.
