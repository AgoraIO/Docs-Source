<PlatformWrapper platform='electron'>

1. [Generate a temporary token](../get-started/manage-agora-account#generate-a-temporary-token) in <Vg k="CONSOLE" /> .

2. In your browser, navigate to the <Link target="_blank" to="{{Global.DEMO_BASIC_VIDEO_CALL_URL}}"><Vg k="COMPANY" /> web demo</Link> and update _App ID_, _Channel_, and _Token_ with the values for your temporary token, then click **Join**.

1. In **preload.js**, update `appID`, `channel` and `token` with your values.

1. In `setVirtualBackground` modify the value of `virtualBackgroundSource.source` to the absolute path of a `.jpg` or `.png` image file on your test device.

1. Run the app
    
    Execute the following command in the terminal:
    ```bash
    npm start
    ```
    You see your app opens a window named **Get started with <Vpd k="PRODUCT" />**.

1. Press **Join** to connect to the same channel as your web demo.

1. Press **Set virtual background**. The background is blurred in the local video of your development device and the remote video of the web demo. You see a message in the console, confirming that background blur has been enabled.

1. Press **Set virtual background** again. The background changes color to a solid blue. You see a message in the console, confirming that color background has been enabled.

1. Press the button again. You see an image in the background. You also see a message in the console, confirming that background image has been enabled.

1. Press the button again. Virtual background is disabled and the actual background is restored.
