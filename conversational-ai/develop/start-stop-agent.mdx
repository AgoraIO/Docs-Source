This page describes how to call the <Vpd k="NAME" /> RESTful APIs to start and stop an AI agent.

## Understand the tech

A user joins a channel and triggers your business server to start an agent via the <Vpd k="NAME" /> API. The agent joins the same channel and communicates with the user through voice, using the specified LLM, TTS service, and Agora's low-latency <Vg k="BACKEND_NAME"/>. When the conversation ends, the business server stops the agent and the user leaves the channel.

## Prerequisites

- [Enabled Agora conversational AI](/conversational-ai/reference/enable-conversational-ai#enable-conversational-ai) for your project.
- The following information from <Vg k="CONSOLE" />:
    - [App ID](/conversational-ai/reference/enable-conversational-ai#get-the-app-id): The string identifier for your project.
    - [App Certificate](/conversational-ai/reference/enable-conversational-ai#get-the-app-certificate): Required by the SDK to generate authentication tokens automatically.
    - For Curl only: [Customer ID and Customer secret](/conversational-ai/rest-api/restful-authentication#generate-customer-id-and-customer-secret) and a [temporary token](/conversational-ai/reference/enable-conversational-ai#generate-temporary-tokens).
- API keys for a supported LLM provider such as [OpenAI](https://openai.com/index/openai-api/) and a supported TTS provider such as [Microsoft Azure](https://azure.microsoft.com/en-us/products/ai-services/ai-speech).

- Implemented the [Voice Calling](/voice-calling/get-started/get-started-sdk) or [Video Calling](/video-calling/get-started/get-started-sdk) quickstart.

## Implementation

This section introduces how to use the Agora Conversational AI SDKs to start and stop a <Vpd k="AGENT" />. In a production environment, implement this logic on your business server.


### Start a conversational AI agent

Call the SDK method or the `join` endpoint to create an agent instance that joins an Agora channel. For the Curl request, pass in the `channel` name and `token` for agent authentication. To generate your base64-encoded credentials, see [RESTful authentication](/conversational-ai/rest-api/restful-authentication.mdx).

<Tabs groupId="language">

<TabItem value="python" label="Python SDK">

```python
from agora_agent import Agent, Agora, Area, DeepgramSTT, OpenAI, MicrosoftTTS

client = Agora(
    area=Area.US,
    app_id='your-app-id',
    app_certificate='your-app-certificate',
)

agent = (
    Agent(name='unique_name')
    .with_stt(DeepgramSTT(model='nova-3', language='en-US'))
    .with_llm(OpenAI(
        api_key='your-llm-api-key',
        base_url='https://api.openai.com/v1/chat/completions',
        model='gpt-4o-mini',
        system_messages=[{'role': 'system', 'content': 'You are a helpful chatbot.'}],
        greeting_message='Hello, how can I help you?',
        failure_message="Sorry, I don't know how to answer this question.",
        max_history=10,
    ))
    .with_tts(MicrosoftTTS(
        key='your-tts-api-key',
        region='eastus',
        voice_name='en-US-AndrewMultilingualNeural',
    ))
)

session = agent.create_session(
    client,
    channel='your_channel_name',
    agent_uid='0',
    remote_uids=['1002'],
    idle_timeout=120,
)

agent_id = session.start()
print(f'Agent started: {agent_id}')
```
</TabItem>

import { Agent, AgoraClient, Area, DeepgramSTT, OpenAI, MicrosoftTTS } from 'agora-agents';

<TabItem value="go" label="Go SDK">

```go
package main

import (
    "context"
    "fmt"
    "log"

    "github.com/AgoraIO/agora-agents-go/v2/agentkit"
    "github.com/AgoraIO/agora-agents-go/v2/agentkit/vendors"
    "github.com/AgoraIO/agora-agents-go/v2/option"
)

func main() {
    ctx := context.Background()
    idleTimeout := 120

    client := agentkit.NewAgoraClient(agentkit.AgoraClientOptions{
        Area:           option.AreaUS,
        AppID:          "your-app-id",
        AppCertificate: "your-app-certificate",
    })

    agent := agentkit.NewAgent(
        agentkit.WithName("unique_name"),
    ).WithStt(
        vendors.NewDeepgramSTT(vendors.DeepgramSTTOptions{
            Model:    "nova-3",
            Language: "en-US",
        }),
    ).WithLlm(
        vendors.NewOpenAI(vendors.OpenAIOptions{
            APIKey: "your-llm-api-key",
            URL:    "https://api.openai.com/v1/chat/completions",
            Model:  "gpt-4o-mini",
            SystemMessages: []map[string]interface{}{
                {"role": "system", "content": "You are a helpful chatbot."},
            },
            GreetingMessage: "Hello, how can I help you?",
            FailureMessage:  "Sorry, I don't know how to answer this question.",
            MaxHistory:      intPtr(10),
        }),
    ).WithTts(
        vendors.NewMicrosoftTTS(vendors.MicrosoftTTSOptions{
            Key:       "your-tts-api-key",
            Region:    "eastus",
            VoiceName: "en-US-AndrewMultilingualNeural",
        }),
    )

    session := agent.CreateSession(client, agentkit.CreateSessionOptions{
        Channel:     "your_channel_name",
        AgentUID:    "0",
        RemoteUIDs:  []string{"1002"},
        IdleTimeout: &idleTimeout,
    })

    agentID, err := session.Start(ctx)
    if err != nil {
        log.Fatal(err)
    }
    fmt.Println("Agent started:", agentID)
}
```

<TabItem value="curl" label="Curl">

```bash
curl --request POST \
  --url https://api.agora.io/api/conversational-ai-agent/v2/projects/:appid/join \
  --header 'Authorization: Basic <your_base64_encoded_credentials>' \
  --header 'Content-Type: application/json' \
  --data '
{
  "name": "unique_name",
  "properties": {
    "channel": "<your_channel_name>",
    "token": "<your_rtc_token>",
    "agent_rtc_uid": "0",
    "remote_rtc_uids": ["1002"],
    "enable_string_uid": false,
    "idle_timeout": 120,
    "llm": {
      "url": "https://api.openai.com/v1/chat/completions",
      "api_key": "<your_llm_api_key>",
      "system_messages": [
        {
          "role": "system",
          "content": "You are a helpful chatbot."
        }
      ],
      "greeting_message": "Hello, how can I help you?",
      "failure_message": "Sorry, I don\'t know how to answer this question.",
      "max_history": 10,
      "params": {
        "model": "gpt-4o-mini"
      }
    },
    "asr": {
      "language": "en-US"
    },
    "tts": {
      "vendor": "microsoft",
      "params": {
          "key": "<your_tts_api_key>",
          "region": "eastus",
          "voice_name": "en-US-AndrewMultilingualNeural"
      }
    }
  }
}'
```

If the request is successful, the agent ID is returned. Store the agent ID to manage the agent in subsequent calls.

### Stop the conversational AI agent

To end the conversation with the AI agent, call the SDK method or the leave endpoint. This causes the agent to leave the Agora channel.

<Tabs groupId="language">
<TabItem value="python" label="Python SDK">

```python
session.stop()
```

</TabItem>
<TabItem value="typescript" label="TypeScript SDK">

```typescript
await session.stop();
```
</TabItem>

<TabItem value="go" label="Go SDK">

```go
if err := session.Stop(ctx); err != nil {
    log.Fatal(err)
}
```
</TabItem>

<TabItem value="curl" label="Curl">

```bash
curl --request POST \
  --url https://api.agora.io/api/conversational-ai-agent/v2/projects/:appid/agents/:agentId/leave \
  --header 'Authorization: Basic <your_base64_encoded_credentials>'
```

If the request is successful, the server responds with a `200 OK` status and an empty JSON object.

```json
// 200 OK
{}
```

</TabItem>
</Tabs>


<Admonition type="info">
The number of Peak Concurrent Users (PCU) allowed to call the server API under a single App ID is limited to 20. If you need to increase this limit, please [contact technical support](mailto:support@agora.io).
</Admonition>
