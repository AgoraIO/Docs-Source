Real-Time Transcription supports 2 transcription modes:

- Channel-based transcription for all active hosts in a channel. To enable it, see [REST Quickstart](../get-started/quickstart).
- Individual transcription for the specified hosts audio only.

This page explains how to implement transcription of individual hosts only.

## Implementation

To transcribe a specific host, follow the API call sequence from the [REST Quickstart](../get-started/quickstart) and modify the `start` request as follows:

```shell
curl --location -g 'https://api.agora.io/v1/projects/{{appId}}/rtsc/speech-to-text/tasks?builderToken={{tokenName}}' \
--header 'Content-Type: application/json' \
--data '{
{{
     "languages": [
         "<YourTranscribeLanguages>"
    ],
    "maxIdleTime": 50,
    "rtcConfig": {
        "channelName": "<YourChannelName>",        // The RTC channel name.
        "subBotUid": "<YourSubscribeUid>",         // The unique UID of the audio stream puller bot.
        "subBotToken": "<YourSubscribeToken>",     // The RTC token for the audio stream puller bot to join the RTC channel. Optional, role set to audience.
        "pubBotUid": "<YourPublishUid>",           // The unique UID of the text publisher bot.
        "pubBotToken": "<YourPublishToken>",       // The RTC token for the text publisher bot to join the RTC channel. Optional, role set to host.
        "subscribeAudioUids": [uint],              // The UIDs of the hosts whose audio needs to be transcribed, max is 3.
    },
```
