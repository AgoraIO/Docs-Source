Add the screen sharing module to your Android project.

1. Download the <Link to="https://download.agora.io/sdk/release/Agora_Native_SDK_for_Android_v4.0.1_FULL.zip">Latest Android SDK</Link> and unzip the folder.

1. Navigate to the `Agora_Native_SDK_for_Android_FULL\rtc\sdk` folder.

1. Copy the file named `AgoraScreenShareExtension.aar` and paste it into the `<project-name>/libs` folder.

1. In `/Gradle Scripts/build.gradle (Module: <project name>.app)`, add the following line under `dependencies`:

``` text
implementation files('../libs/AgoraScreenShareExtension.aar')
```
