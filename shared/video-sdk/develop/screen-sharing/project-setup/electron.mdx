Windows or macOS system assigns a unique **Display ID** for each screen and a unique **Window ID** for each window. <Vg k="COMPANY" /> provides the following two <Vg k="SCR_SHR" /> solutions:

    - Share a specific screen or part of a specified screen using **Display ID**.
    - Share a specific window or part of a specified window using **Window ID**.
