This number is typically different from your DAU/MAU. If an end user (with one UID or UUID) logs into <Vg k="COMPANY"/> Chat, via x number of devices, they are counted as 1 MAU but x PCC.

**PCC overage fee**: We measure the aggregated number of PCCs and MAUs across all your projects at the end of each billing cycle, which ends at every calendar month and charge an overage fee, if applicable. You can find the overage fee schedule by visiting <Link to="{{Global.AGORA_CONSOLE_URL}}"><Vg k="CONSOLE" /></Link>. Paid packages of <Vg k="COMPANY"/> Chat by default allow for a PCC smaller or equal to 10% of your total MAU. Exceeding the 10% of MAU won’t impact your end users or disrupt services, as we meter the PCC only at the end of each calendar month. <Vg k="COMPANY"/> Chat Starter/Pro/Enterprise package subscribers are entitled to at least 500, 1K, and 1K PCC respectively, for free in each month. For example, if a Starter package subscriber has 1K MAU and 500 PCC in a month, there won’t be any PCC overage fee billed.
