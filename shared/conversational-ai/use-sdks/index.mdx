This page explains how to build a <Vpd k="AGENT" /> using <Vg k="COMPANY" /> SDKs.

## Understand the tech

<Vg k="COMPANY" /> SDKs provide the following advantages when working with the <Vpd k="NAME" /> RESTful API:

- **Simplified communication**: Handle request and response processing for the RESTful API to simplify communication.

- **High availability**: Automatically maintain service continuity by switching to the best available domain when network issues such as DNS failures or timeouts occur. 

- **User-friendly APIs**: Provide clear and concise methods for common tasks such as creating or stopping a <Vpd k="AGENT" />.

- **Optimized performance**: Designed for efficiency and scalability, with support for high concurrency and broad platform compatibility.


## Prerequisites

Before you begin, make sure that you have:

<Tabs groupId="platform">
<TabItem value="go" label="GO" default>
- [Go SDK](https://go.dev/dl/) 1.18 or above.
</TabItem>
<TabItem value="java" label="Java" default>
- [Java SDK](https://www.oracle.com/java/technologies/downloads/) 1.8 or above.
</TabItem>
</Tabs>

- [Enabled <Vg k="COMPANY" /> conversational AI](../get-started/manage-agora-account#enable-conversational-ai) for your project.
- The following information from <Vg k="CONSOLE" />:
    - [App ID](../get-started/manage-agora-account#get-the-app-id): The string identifier for your project used to call the <Vpd k="NAME" /> RESTful API.
    - [Customer ID and Customer secret](../rest-api/restful-authentication#generate-customer-id-and-customer-secret): Used for HTTP authentication when calling the RESTful APIs.
    - A [temporary token](../get-started/manage-agora-account#generate-temporary-tokens): The token is used by the agent for authentication when joining an <Vg k="COMPANY" /> channel. 
- Obtained an API key and callback URL from a Large Language Model (LLM) provider such as [OpenAI](https://openai.com/index/openai-api/).
- Obtained an API key from a text-to-speech (TTS) provider such as [Microsoft Azure](https://azure.microsoft.com/en-us/products/ai-services/ai-speech).
- Implemented the [voice](/voice-calling/get-started/get-started-sdk) or [video](/video-calling/get-started/get-started-sdk) calling quickstart using SDK v4.5.1 or above.
