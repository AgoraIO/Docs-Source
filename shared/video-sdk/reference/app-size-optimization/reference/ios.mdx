### `subspecs` field

When integrating the iOS/macOS SDK through CocoaPods, you can use the `subspecs` field to specify the dynamic library you need to integrate, so as to exclude the plugins you don't need to use. The corresponding relationship between the dynamic library and the `subspecs` field is shown in the table below:
