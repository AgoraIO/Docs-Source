1.  **Create a new React project**

    In a terminal window, execute the following command:

    ``` bash
    npm create vite@latest <app-name> --template react-ts
    ```

    This installs [Vite](vitejs.dev) and scaffolds a new React project with Typescript.

2.  **Install the <Vg k="UIK" />**

    In the terminal window, navigate to the `<app-name>` directory:

    ```bash
    cd <app-name>
    ```

    Execute the following command to install the <Vg k="UIK" />:

    ```bash
    npm i agora-react-uikit
    ```
