### Test channel media Relay

1.  In **App.tsx**, update `appId`, `channelName` and `destChannelName`.

2. [Generate a temporary token](../reference/manage-agora-account#generate-a-temporary-token) in <Vpl k="CONSOLE" /> using `appId` and `channelName`. Use it to update `token` in `App.tsx`.

3. Generate another token in <Vpl k="CONSOLE" /> using `appId` and `destChannelName`. Use it to update `destChannelToken` in `App.tsx`.

4. In your browser, navigate to the <Link target="_blank" to="{{Global.DEMO_BASIC_VIDEO_CALL_URL}}"><Vg k="COMPANY" /> web demo</Link> and update _App ID_, _Channel_, and _Token_ with the values for your temporary token, then click **Join**.

5. On another device, update App ID, Channel, and Token with the `appId`, `destChannelName`, and `destChannelToken` values.

6.  Run the app

    - *Android*

        1. Enable the Developer options on your Android device, and then connect it to your computer using a USB cable.
        1. Run npx react-native run-android in the project root directory.

    - *iOS:*

        1. Run your app in Xcode.

8. Select **Host** and press **Join**. You see the video from the web browser demo app connected to `channelName` in your development device.

9. Press **Start Channel Media Relay**. You see the video from the web browser demo app connected to `channelName` relayed to the web browser demo app connected to `destChannelName`.

10. Press **Stop Channel Media Relay**. The media relaying is stopped.

### Test joining multiple channels

1.  In **App.tsx**, update `appId`, `channelName` and `secondChannelName`.

2. [Generate a temporary token](../reference/manage-agora-account#generate-a-temporary-token) in <Vpl k="CONSOLE" /> using `appId` and `channelName`. Use it to update `token` in `App.tsx`.

3. Generate another token in <Vpl k="CONSOLE" /> using `appId` and `secondChannelName`. Use it to update `secondChannelToken` in `App.tsx`.

4. In your browser, navigate to the <Link target="_blank" to="{{Global.DEMO_BASIC_VIDEO_CALL_URL}}"><Vg k="COMPANY" /> web demo</Link> and update _App ID_, _Channel_, and _Token_ with the values for your temporary token, then click **Join**.

5. Repeat the previous step on a second device, but this time use `appId`, `secondChannelName`, and `secondChannelToken` to **Join** the channel.

6.  Run the app

    - *Android*

        1. Enable the Developer options on your Android device, and then connect it to your computer using a USB cable.
        1. Run npx react-native run-android in the project root directory.

    - *iOS:*

        1. Run your app in Xcode.

8. Select **Audience** 

    1. Press **Join**. You see the video from the web browser demo app connected to `channelName` in your development device.
    2. Press **Join Second Channel**. You see the video from the web browser demo app connected to `secondChannelName` in your development device.
    3. Press **Join Second Channel** again and then **Leave** to exit both channels.

9. Select **Host** 

    1. Press **Join**. You see the local video in your development device. The web browser demo app connected to `channelName` shows the video from your development device.
    2. Press **Join Second Channel**. You see the video from the web browser demo app connected to `secondChannelName` in your development device.
    3. Press **Join Second Channel** again and then **Leave** to exit both channels.
