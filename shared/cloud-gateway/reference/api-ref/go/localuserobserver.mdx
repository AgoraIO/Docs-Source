The `LocalUserObserver` class.

### `OnUserAudioTrackStateChanged`

Triggered when the state of a remote audio track changes.

```go
OnUserAudioTrackStateChanged func(localUser *LocalUser, uid string, remoteAudioTrack *RemoteAudioTrack, state int, reason int, elapsed int)
```

#### Parameters

| Parameter | Description |
|-----------|-------------|
| `localUser` | `LocalUserObserver` class. |
| `uid` | Remote user ID. |
| `remoteAudioTrack` | Remote audio track. |
| `state` | Current state of the audio track. |
| `reason` | Reason for the state change. |
| `elapsed` | Time elapsed (in milliseconds) since the remote user joined the channel. |

### `OnStreamMessage`

Callback for receiving stream messages. This callback indicates that the local user has received a stream message sent by a remote user using the `SendStreamMessage` method.

```go
OnStreamMessage func(localUser *LocalUser, uid string, streamId int, data []byte)
```

#### Parameters

| Parameter | Description |
|-----------|-------------|
| `localUser` | `LocalUserObserver` class. |
| `uid` | User ID of the sender. |
| `streamId` | Stream ID of the received message. |
| `data` | Received data. |

### `OnUserInfoUpdated`

Triggered when user media information is updated.

```go
OnUserInfoUpdated func(localUser *LocalUser, uid string, userMediaInfo int, val int)
```

#### Parameters

|Parameter|Description|
|---------|-----------|
|`localUser`|[`LocalUser`](#localuser) class.|
|`uid`|User ID.|
|`userMediaInfo`|User media information. See [`UserMediaInfo`](#usermediainfo) for details.|
|`val`| Whether the user has stopped sending streams. - `0`: Not stopped. - `1`: Stopped.|
