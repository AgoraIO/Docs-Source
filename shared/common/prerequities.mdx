If your target platform is iOS, your development environment must meet the following requirements:
- React Native 0.60 or later
- macOS
- Node 10 or later
- Xcode 10 or later
- CocoaPods
- A physical or virtual mobile device running iOS 9.0 or later. If you use React Native 0.63 or later, ensure your iOS version is 10.0 or later.

If your target platform is Android, your development environment must meet the following requirements:
- React Native 0.60 or later
- macOS, Windows, or Linux
- Node 10 or later
- Java Development Kit (JDK) 11 or later
- Android Studio (latest version recommended)
- A physical or virtual mobile device running Android 5.0 or later

For more information, see [Setting up the development environment](https://reactnative.dev/docs/environment-setup).
