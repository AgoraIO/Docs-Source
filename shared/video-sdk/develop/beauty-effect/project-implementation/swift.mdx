
### Beautiful

#### Achieve preset beauty effects

The `setFaceShapeBeautyOptions:options:` method is the master switch of the beauty effect. Call this method and pass `true` in the `enable` parameter to turn on the beauty effect. You can specify the beauty style and style strength in `FaceShapeBeautyOptions` and use the preset parameters to quickly achieve the beauty effect.

```swift
var beautyOptions = AgoraFaceShapeBeautyOptions()
// Set the beauty style type to male
beautyOptions.shapeStyle = AgoraFaceShapeStyle.male
// Set style intensity
beautyOptions.styleIntensity = 100
agoraKit?.setFaceShapeBeautyOptions(true, options: beautyOptions)
```

#### Fine-tune individual parts

After turning on the beautify function, you can call the `setFaceShapeAreaOptions:` method to fine-tune each area of the face individually. This method takes effect immediately when adjusting the modification strength of a single part, and does not modify the preset value.

```swift
var areaOptions = AgoraFaceShapeAreaOptions()
// Specify a fine-tuned part, e.g. head size
areaOptions.shapeArea = AgoraFaceShapeArea.headScale
// Strength of adjustments in this area
areaOptions.shapeIntensity = 100
agoraKit?.setFaceShapeAreaOptions(areaOptions)
```

#### Close Beauty

Call the `setFaceShapeBeautyOptions:options:` method and pass `false` in the `enable` parameter to turn off the beauty effect.

```swift
agoraKit?.setFaceShapeBeautyOptions(false, options: beautyOptions)
```

### Beauty care

#### Start beauty

Call the `setExtensionPropertyWithVendor:extension:key:value:` method to enable the beauty effect and fill in the parameters as follows:

    - `provider`: `agora_video_filters_clear_vision`
    - `extension`: `clear_vision`
    - `key`: `makeup_options`
    - `value`: Packaged in JSON format where different fields correspond to different beauty effects.

Refer to the following code to enable the beauty effect:

```swift
let makeupParams: [String: Any] = [
   "enable_mu": true,
   "blushStyle": 1,
   "blushColor": 2,
   "blushStrength": 0.5
]
guard let json = try? JSONSerialization.data(withJSONObject: makeupParams, options: []),
   let jsonString = String(data: json, encoding: .utf8) else {
   return
}
        
self.agoraKit.setExtensionPropertyWithVendor("agora_video_filters_clear_vision",
                                              extension: "clear_vision",
                                              key: "makeup_options",
                                              value: jsonString)
```

#### Modify beauty effect options

After modifying the value of the beauty effect field in JSON, call the `setExtensionPropertyWithVendor:extension:key:value:` method to modify the beauty effect options. Refer to the following code to modify the beauty effect:

```swift
let makeupParams: [String: Any] = [
   "blushColor": 1,
]
guard let json = try? JSONSerialization.data(withJSONObject: makeupParams, options: []),
   let jsonString = String(data: json, encoding: .utf8) else {
   return
}
        
self.agoraKit.setExtensionPropertyWithVendor("agora_video_filters_clear_vision",
                                              extension: "clear_vision",
                                              key: "makeup_options",
                                              value: jsonString)
```

#### Turn off beauty effects

Specify the `enable_mu` node as `false` in the JSON and call the `setExtensionPropertyWithVendor:extension:key:value:` method to turn off the beauty effect:

```swift
let makeupParams: [String: Any] = [
   "enable_mu": false,
]
guard let json = try? JSONSerialization.data(withJSONObject: makeupParams, options: []),
   let jsonString = String(data: json, encoding: .utf8) else {
   return
}
        
self.agoraKit.setExtensionPropertyWithVendor("agora_video_filters_clear_vision",
                                              extension: "clear_vision",
                                              key: "makeup_options",
                                              value: jsonString)
```
