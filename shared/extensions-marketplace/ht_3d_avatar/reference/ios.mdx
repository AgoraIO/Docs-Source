### Sample project

To run the complete sample project available on Github, take the following steps:

1. Clone the sample code repository:
    
    ```bash
    git clone https://github.com/Texeljoy/AgoraHTAvatarExtensionDemo/tree/master/iOS
    ```

1. Download the [iOS plug-in package](https://github.com/Texeljoy/AgoraHTAvatarExtension/tree/master/iOS/ALL_ARCHITECTURE/Release) for <Vg k="RTEE_HT3D" />. Save all `.framework` files to the `agora-htavatar-extension-demo/iOS/ExtensionExample` folder.

1. [Contact <Vg k="COMPANY" />](mailto:support@agora.io) to obtain a certificate file and the <Vg k="RTEE_HT3D" /> resource package.

1. Save the certificate file `HTKey.h` and the resource package `HTEffect.bundle` to the `agora-htavatar-extension-demo/iOS/ExtensionExample` folder.

1. To install project dependencies using CocoaPods, in the terminal, navigate to the `agora-htavatar-extension-demo/iOS` folder and run the following command:
    
    ```bash
    pod install
    ```

1. Open the project `ExtensionExample.xcworkspace` in Xcode.

1. Open the file `Config.h`:

    ```c
    NSString *const appID = @"<YOUR_APP_ID>";
    NSString *const token = nil;
    ```

    Replace `<YOUR_APP_ID>` with your Agora app ID. You don't need to specify the `token` value when running the demo.

1. Open the file `agora-htavatar-extension-demo/iOS/ExtensionExample/HTKey.h`:

    ```c
    NSString *const HTKey = @"<YOUR_HT_SDK_KEY>";
    ```

    Replace `<YOUR_HT_SDK_KEY>` with the test key provided by <Vg k="COMPANY" />.

1. Connect a physical iOS device and run the project.

1. Click `htARSetModelto` button to set the 3D AR prop effect.
