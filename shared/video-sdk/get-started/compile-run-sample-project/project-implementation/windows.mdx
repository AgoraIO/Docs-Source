<PlatformWrapper platform="windows">

1. Open the file `/APIExample/APIExample.sln` in Visual Studio.

1. Click **Start compiling**.

1. After compilation is complete, you see the following window on your device:

    ![](https://agora-doc.s3.us-east-1.amazonaws.com/images/video-sdk/run-compile-windows-step-3.png)

1. Choose any demo you want to experience. Taking the **Live Broadcasting** as an example, enter the channel name you used when generating the temporary token in **Channel Name**, then click **Join Channel**.

1. To test various audio and video interaction use-cases, use the [Agora web demo](https://webdemo.agora.io/basicVideoCall/index.html), or install and run the sample project on a second device. Make sure you use the same app ID on both devices. When you join the same channel from two devices, you can see and hear each other.

    ![](https://agora-doc.s3.us-east-1.amazonaws.com/images/video-sdk/run-compile-windows-step-5.png)
