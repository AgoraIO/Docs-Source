To enable <Vg k="MPUSH" /> in <Vg k="CONSOLE" />:

1. In [Agora Console](https://console.agora.io/), open **Project Management**.

2. On the **Project Management** page, click **Config** for the project you want to enable <Vg k="MPUSH" /> in.

3. Find **<Vg k="MPUSH" />**, and click **Enable**.

4. In **Media push configuration** enable **Server-side RESTful API**.

1. In **Enable Media push Server-side RESTful API**, if you agree with the conditions in the prompt, click **Save**.

    **Note**: <Vg k="MPUSH" />  cannot be disabled after enabling.


The **Enable** button changes to **Config** button. You can now configure <Vg k="MPUSH" />.
