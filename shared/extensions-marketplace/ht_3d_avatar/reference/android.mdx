### Sample project

To run the complete sample project available on Github, take the following steps:

1. Clone the sample code repository:
    
    ```bash
    git clone https://github.com/Texeljoy/AgoraHTAvatarExtensionDemo/tree/master/Android
    ```

1. Download the [Android plug-in package](https://github.com/Texeljoy/AgoraHTAvatarExtension/tree/master/Android/ALL_ARCHITECTURE) for <Vg k="RTEE_HT3D" />. Save all `.aar` files to the `agora-htavatar-extension-demo/Android/app/libs` folder. 

1. [Contact <Vg k="COMPANY" />](mailto:support@agora.io) to obtain a certificate file and the <Vg k="RTEE_HT3D" /> resource package.

1. Save the required model and prop files from the resource package to the  `agora-htavatar-extension-demo/Android/app/src/main/assets` folder. 

1. Open the project folder `agora-htavatar-extension-demo/Android` in Android Studio and sync project with Gradle files.

1. Open the file `agora-htavatar-extension-demo/Android/app/src/main/java/io/agora/rte/extension/hteffectar/example/Constants.java`:
    
    ```java
    public interface Constants {
        String mAppId = "<YOUR_APP_ID>";
        String mToken = null;
        String mLicenseName = "<YOUR_LICENSE_NAME>";
    }
    ```
    
    Replace `<YOUR_APP_ID>` with your Agora app ID. You don't need to specify `mToken` and `mLicenseName` values when running the demo.
    

1. Open the file `agora-htavatar-extension-demo/Android/app/src/main/java/io/agora/rte/extension/hteffectar/example/HtKey.java`:
    
    ```java
    public interface HtKey {
        String HTKey = "<YOUR_HT_SDK_KEY>";
    }
    ```

    Replace `<YOUR_HT_SDK_KEY>` with the test key provided by <Vg k="COMPANY" />

1. Connect a physical Android device and run the project.

1. Click `htARSetModelto` button to set the 3D AR prop effect.
