## 1.8.0

v1.8.0 was released on May 1, 2023.

#### New features

1. Two-way audio and video communication between IoT SDK instances, as well as between
IoT SDK and Video SDK for Android, iOS, macOS, Windows, and Web.

2. Providing a target bitrate per bandwidth estimation (BWE) algorithms.

3. Requesting key frames from remote users.

4. Muting and unmuting local and remote media streams.

5. Monitoring the transmission state of remote media streams.
