    * Sets the audio profile.
    * Starts the network probe test.

1. Click **Join** to start a call.

1. After joining a channel, you receive toast messages informing you of some selected call statistics, including:

    * The number of users in the channel
    * Packet loss rate
    * Local audio stats
    * Remote audio state changes

1. You see the network status indicator updated periodically based on the result of the `onNetworkQuality` callback.
