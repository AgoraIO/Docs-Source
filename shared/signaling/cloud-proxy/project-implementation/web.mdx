1.  **Enable the connection to <Vg k="CP"/>**

    To access <Vg k="AGORA_BACKEND" /> in a restricted network environment, set `cloudProxy` to `true` in your
  `rtmConfig` object:

    ``` javascript
      // Set cloud proxy
      const rtmConfig = {
        token: config.token,
        logLevel: "debug",
        useStringUserId: true,
        cloudProxy: true,
      };
    ```

2.  **Pass the configuration to the <Vg k="MESS_ENGINE" /> **

    ``` javascript
      const signalingManager = await SignalingManager(
        messageCallback,
        eventsCallback,
        rtmConfig
      );
    ```
