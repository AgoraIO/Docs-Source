<PlatformWrapper platform="web">
1.  **Create a new React project**

    In a terminal window, execute the following command:

    ``` terminal
    npx create-react-app <app-name>
    ```

    If `create-react-app` is not present on your device, it is downloaded and installed.

2.  **Download and install <Vg k="UIK" />**

    In the terminal window, navigate to the `<app-name>` directory, then execute the following command:

    ``` terminal
    npm i agora-react-uikit
    ```
