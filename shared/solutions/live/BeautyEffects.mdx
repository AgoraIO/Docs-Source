## Basic Beauty Effects


Agora provides the basic beauty effects in Native SDK and Web SDK as well. And it is free of charge.

Agora Native RTC SDK provides API [setBeautyEffectOptions](https://api-ref.agora.io/en/voice-sdk/android/4.x/API/class_irtcengine.html#api_irtcengine_setbeautyeffectoptions) to have the basic Beauty effects.


below table shows the functionalities and detailed parameters
| Functionality | Description |
| :----------- | :---------- |
| Contrast adjustment for lightness and darkness | contrast with the value range of [0,1,2]. The larger the value, the greater the contrast between light and dark.<ul><li>LIGHTENING_CONTRAST_LOW(0): Low contrast level.</li><li>LIGHTENING_CONTRAST_NORMAL(1): Normal contrast level.</li><li>LIGHTENING_CONTRAST_HIGH(2): High contrast level.</li></ul> |
| Brightness adjustment (whitening) | Whitening with the value range of [0.0~1.0], where 0.0 means the original brightening. The default value is 0.6. The higher the value, the greater the degree of brightening. |
|Smoothness adjustment (skin smoothing) | Skin Smoothening with the value range of [0.0~1.0], where 0.0 means the original smoothness. The default value is 0.5. The greater the value, the greater the smoothness level. |
| Color saturation adjustment (for a more rosy complexion) | Cheek blushing with the value range of [0.0 ~1.0], where 0.0 means the original redness. The default value is 0.1. The larger the value, the greater the redness level. |
| Detail clarity (sharpening) | sharpening with the value range of [0.0 ~ 1.0],where 0.0 means the original sharpness. The default value is 0.3. The larger the value, the greater the sharpness level. |


### Sample Codes

**Java**
```Java
// Java
BeautyOptions options = new BeautyOptions();
options.lighteningContrastLevel = LIGHTENING_CONTRAST_NORMAL;
options.smoothnessLevel = 0.5;
options.lighteningLevel = 0.6;
options.rednessLevel = 0.1;
options.sharpnessLevel = 0.3;
mRtcEngine.setBeautyEffectOptions(true, options);

```

**Swift**
```Swift
// Swift
let options = AgoraBeautyOptions()
options.lighteningContrastLevel = .normal
options.smoothnessLevel = 0.5
options.lighteningLevel = 0.6
options.rednessLevel = 0.1
options.sharpnessLevel = 0.3
agoraKit.setBeautyEffectOptions(true, options: options)

```

**Objective-C**
```Objective-C
// Objective-C
AgoraBeautyOptions *options = [[AgoraBeautyOptions alloc] init];
options.lighteningContrastLevel = AgoraLighteningContrastNormal;
options.smoothnessLevel = 0.5;
options.lighteningLevel = 0.6;
options.rednessLevel = 0.1;
options.sharpnessLevel = 0.3;
[self.agoraKit setBeautyEffectOptions:YES options:options];

```


**C++**
```C++
// C++
bool enabled = true;
agora::rtc::BeautyOptions options;
options.lighteningContrastLevel = BeautyOptions::LIGHTENING_CONTRAST_NORMAL;
options.smoothnessLevel = 0.5;
options.lighteningLevel = 0.6;
options.rednessLevel = 0.1;
options.sharpnessLevel = 0.3;
m_lpAgoraEngine->setBeautyEffectOptions(enabled, options);
```

In the Web SDK, we provides Web SDK Extension "[agora-extension-beauty-effect](https://www.npmjs.com/package/agora-extension-beauty-effect?activeTab=versions)", and it is used together with the Agora Web SDK (v4.12.0 or later) to implement the basic beauty effects feature. This extension gives you granular control over the degree of enhancement effects, such as skin lightening, skin smoothing, and red saturation.
you can access this [web demo](https://webdemo-global.agora.io/example/plugin/beauty/index.html) to review its behaviors.
