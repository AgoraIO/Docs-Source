<PlatformWrapper platform='linux-cpp'>
    Install the cURL library using the following command:

        ```bash
        sudo apt-get install libcurl4-openssl-dev
        ```

    To build the project, open Terminal and run the following command in the project folder:

        ```bash
        sh +x clean_build.sh
        ```
    To run the project, execute:

        ```bash
        ./build/SignalingDemo
        ```
</PlatformWrapper>
