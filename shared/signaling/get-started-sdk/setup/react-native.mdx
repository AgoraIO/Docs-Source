Follow these steps to set up your project and add the Signaling SDK.

### Create a Project

Create a new React Native application using the React Native Community CLI:

```bash
# Create a new React Native project
npx @react-native-community/cli@latest init SignalingQuickstart

# Navigate to your project directory
cd SignalingQuickstart
```

For detailed React Native environment setup, see [Get Started with React Native](https://reactnative.dev/docs/environment-setup).

### Install the SDK

To download and add the <Vg k="COMPANY" /> <Vpd k="SDK"/> to your React Native project:  

1. In your project directory, run one of the following commands:

   **npm:**

   ```bash
   npm install agora-react-native-rtm
   ```

   **Yarn:**

   ```bash
   yarn add agora-react-native-rtm
   ```
   
2. **For iOS projects**, navigate to the `ios` directory and install pods:

   ```bash
   cd ios && pod install && cd ..
   ```

3. Test the installation by running your project:

   ```bash
   # For Android
   npm run android

   # For iOS
   npm run ios
   ```
