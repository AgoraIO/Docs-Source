This class is used to create media components.

## `createMediaRtcRecorder`

Creates an instance of [`IAgoraMediaRtcRecorder`](agora-media-rtc-recorder).

```cpp
virtual agora_refptr<IAgoraMediaRtcRecorder> createMediaRtcRecorder() = 0;
```

#### Return Values

- On success: Returns an [`IAgoraMediaRtcRecorder`](agora-media-rtc-recorder) object.
- On failure: Returns a null pointer.

## `createAgoraMediaComponentFactory`

Creates and returns a pointer to an `IMediaComponentFactory` object.

```cpp
AGORA_API agora::rtc::IMediaComponentFactory* AGORA_CALL createAgoraMediaComponentFactory();
```

#### Return Values

- On success: Returns a pointer to an `IMediaComponentFactory`.
- On failure: Returns a null pointer.
