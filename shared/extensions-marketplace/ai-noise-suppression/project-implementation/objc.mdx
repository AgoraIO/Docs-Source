

1. **Integrate <Vg k="RTEE_NS" /> into your app**

    Use the XCode package manager to add _AgoraAINoiseSuppressionExtension_ from the  **<Vpl k="PACKAGE_NAME"/>** package.

2. **Enable <Vg k="RTEE_NS" />**

    In `ViewController`, add the following line in `initializeAgoraEngine` after `config.appId = appID`:

    ```swift
    agoraEngine.enableExtension(withVendor: "Agora", extension: "AINS", enabled: true)
    ```

3. **Initialize variables to set <Vg k="RTEE_NS" /> options**

    Add the following line to the top of the `ViewController` class:

    ```swift
    let agoraVideoDenoiserOptions: AgoraVideoDenoiserOptions = AgoraVideoDenoiserOptions()
    ```

4. **Set the <Vg k="RTEE_NS" /> mode**

    Auto mode automatically enables or disables the video noise reduction feature according to the ambient light. Set to '1' so the current user enables or disables <Vg k="RTEE_NS" /> manually.

    To set the <Vg k="RTEE_NS" /> mode to `auto`, in `setupLocalVideo`, add the following code after `enableVideo`:

     ```swift
    agoraVideoDenoiserOptions.mode = .auto
    ```

5. **Adjust the <Vg k="RTEE_NS" /> level**

    By default, <Vg k="RTEE_NS" /> is set to 'HighQuality' (0) to balance the <Vg k="RTEE_NS" /> quality and performance.

    - To set the value to 'Fast' and prioritize <Vg k="RTEE_NS" /> performance over quality, in `setupLocalVideo`, put this line after `agoraVideoDenoiserOptions.mode = .auto`:

        ```swift
        agoraVideoDenoiserOptions.level = .fast
        ```

    - To give more focus to <Vg k="RTEE_NS" /> quality rather than performance, change the level to 'Strength':

        ```swift
        agoraVideoDenoiserOptions.level = .strength
        ```

6. **Set <Vg k="RTEE_NS" /> on <Vg k="ENGINE" />**

    Put the following code after `agoraVideoDenoiserOptions.level = .strength`:

    ```swift
    agoraEngine.setVideoDenoiserOptions(true, options: agoraVideoDenoiserOptions)
    ```
