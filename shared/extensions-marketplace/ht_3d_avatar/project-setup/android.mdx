<PlatformWrapper platform="android">
1. Download the [<Vg k="RTEE_HT3D" /> Android package](https://gitee.com/htai-open/agora-htavatar-extension/tree/master/Android). 

1. Copy all the `.aar` files from the package to the `/app/libs` folder of your Android project.

1. [Contact Agora](mailto:support@agora.io) to obtain the <Vg k="RTEE_HT3D" /> resource package. Save the required model and prop files in the `/app/src/main/assets` folder of your project.

1. Open the `app/build.gradle` file and under `dependencies`, add the following:
    
    ```
    implementation fileTree(dir: "libs", include: ["*.jar", "*.aar"])
    ```

1. Import the classes you need:

    ```java
    import io.agora.rtc2.Constants;
    import io.agora.rtc2.IMediaExtensionObserver;
    import io.agora.rtc2.IRtcEngineEventHandler;
    import io.agora.rtc2.RtcEngine;
    import io.agora.rtc2.RtcEngineConfig;
    import io.agora.rtc2.video.VideoCanvas;
    ```

1. Add in the xml layout file corresponding to the Java file that uses the 3D effects:

    ```xml
    <RelativeLayout
        android:id="@+id/unityView"
        android:layout_width="1dp"
        android:layout_height="1dp">
    </RelativeLayout>
    ```

1. In the Java file that uses 3D effects, add:

    ```java
    unityView = findViewById(R.id.unityView);
    unityView.addView(mUnityPlayer);
    ```
