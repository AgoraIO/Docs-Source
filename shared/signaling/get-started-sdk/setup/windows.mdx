This section shows you how to set up a new Windows project and install the Agora <Vpd k="SDK"/>.

### Create a project

To create a new Console Application in C++, see Create a [Create a C++ console app project](https://learn.microsoft.com/en-us/cpp/build/vscpp-step-1-create?view=msvc-170). If your project is already set up, skip to the next section.

### Integrate the SDK

1. [Download](/sdks?platform=windows) the latest version of the SDK and unzip it.

1. Copy the `sdk` folder from the downloaded package to the root of your project `<project directory>`.

### Configure the project

In **Visual Studio**, right-click your project in the **Solution Explorer**, select **Properties**, and configure the following:

    1. **Choose the target platform architecture** 

        In **Configuration Properties**, go to **General** and select **Win32** or **x64**.

    1. **Include the header files** 

        Go to **Configuration Properties** > **VC++ Directories** > **Include Directories**, click **Edit**, and add the following path:
         ```bash
         ;$(SolutionDir)sdk\high_level_api\include;
         ```

    1. **Set the library directories**
    
        Navigate to **Configuration Properties** > **VC++ Directories** > **Library Directories**, click **Edit**, and add the appropriate path based on your target architecture:

        - For 32-bit applications:
            ```
            ;$(SolutionDir)sdk\x86;
            ```
        - For 64-bit applications:
            ```
            ;$(SolutionDir)sdk\x86_64;
            ```

    1. **Specify the library file** 

        Navigate to **Configuration Properties** > **Linker** > **Input** > **Additional Dependencies**, click the drop-down menu, select **Edit**, and add the following library:
 
         ```bash
         agora_rtm_sdk.dll.lib;
         ```

    1. **Copy the required `.dll` files**  
        
        Copy the following files from the SDK to your project's build directories:

        | Architecture  | Source File | From | To |
        |---------------|-------------|--------|-------------|
        | 32-bit        | `agora_rtm_sdk.dll`, `libaosl.dll`   | `/sdk/x86`        | `<project directory>/Debug`                |
        | 64-bit        | `agora_rtm_sdk.dll`, `libaosl.dll`   | `/sdk/x86_64`     | `<project directory>/x64/Debug`           |
