### Middleware token generation API

This section provides details about the middleware token generation API endpoints.

#### Generate Token

Generates a <Vg k="VSDK" />, <Vg k="SIG" />, or <Vg k="CHAT" /> token based on the provided parameters.

#### Endpoint

`POST /token/getNew`

#### Request Body

```json
{
  "tokenType": "rtc|rtm|chat",
  "channel": "string",
  "uid": "string",
  "role": "publisher|subscriber",
  "expire": int
}
```

#### Response

```json
{
  "token": "string"
}
```
