
3. **Configure the project**:

    In `DocsAppConfig.swift`, update the following:

     - `appId` - the AppID of your project.
     - `rtcToken` - an empty string.
     - `serverUrl` - the base URL of your authentication server. For example, `https://agora-token-service-production-1234.up.railway.app`.
     - `tokenExpiryTime` - with an integer denoting the number of seconds after which a token expires.
