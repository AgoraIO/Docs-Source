<PlatformWrapper platform="ios">

* {data.DOWNLOAD_URL}/{props.DOWNLOAD_ANDROID}


1. Manual {data.VSDK} integration using CocoaPods

2. [Ensure that you have installed CocoaPods](https://guides.cocoapods.org/using/getting-started.html#getting-started).

3. In Terminal, navigate to the project path, and run the `pod init` command to create a `Podfile` in the project folder.

4. Open the `Podfile`, delete all contents, and input the following codes. Remember to replace `Your App` with the target name of your project and replace `version` with the version of the SDK that you want to integrate.
  

  ```html
  # platform :ios, '9.0'
  target 'Your App' do
    pod 'AgoraRtcEngine_iOS', 'version'
  end
  ```

5. Return to Terminal, and run the `pod install` command to install {data.VSDK}. Once you successfully install the SDK, it shows `Pod installation complete!` in Terminal, and you can see an `xcworkspace` file in the project folder.

6. Open the generated `xcworkspace` file.
