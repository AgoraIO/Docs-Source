In the Unreal Editor, you can compile and run the sample project at the main level or any sublevel.

<Tabs>
<TabItem value="Main level" label="Main Level">
1. In the path of the C++ API example, double-click **AgoraExample.uproject** to open the project.

1. Double-click **MainLevel**, then click **Compile** and run the sample project. You see the following interface:

    ![](/images/video-sdk/run-compile-unreal-tab-1-step-3.png)

1. Fill in the text box with the App ID and temporary token you obtained from the <Vg k="CONSOLE" />, and the channel name you used to generate the token.

1. Click the button on the left to navigate from the main level to the sub-levels of the API example to experience the selected demo. Choose `BasicVideoCallScene` as an example, click **JoinChannel** to join the channel. You see yourself in the local view.

    ![](/images/video-sdk/run-compile-unreal-tab-1-step-4.png)

1. To test various audio and video interaction scenarios, connect to the [Agora web demo](https://webdemo.agora.io/basicVideoCall/index.html), or install and run the sample project on a second device. Make sure you use the same App ID on both devices. When you join the same channel from two devices, you can see and hear each other.

</TabItem>
<TabItem value="sub level" label="Sub Level">
1. In the path of the C++ API example, double-click **AgoraExample.uproject** to open the project.

1. Choose **BasicVideoCallScene** as an example, double-click the sub-level in the Content Browser, and then click the  **BP_AgoraActor** blueprint in the **World Outliner panel**. Fill in the App ID, temporary token, and channel name of your project in the **Details panel**.

    ![](/images/video-sdk/run-compile-unreal-tab-2-step-2.png)

1. Click **Compile** and run the sample project at the sublevel. You see the following interface:

    ![](/images/video-sdk/run-compile-unreal-tab-2-step-4.png)

1. Click **JoinChannel** to join the channel. You see yourself in the local view.   

1. To test various audio and video interaction scenarios, connect to the [Agora web demo](https://webdemo.agora.io/basicVideoCall/index.html), or install and run the sample project on a second device. Make sure you use the same App ID on both devices. When you join the same channel from two devices, you can see and hear each other.

</TabItem>
</Tabs>
