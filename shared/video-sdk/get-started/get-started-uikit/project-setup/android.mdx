<PlatformWrapper platform="android">

1.  In **Android Studio**, create a new **Phone and Tablet** [Android project](https://developer.android.com/studio/projects/create-project) with an **Empty Activity** using Java. Set **Minimum SDK** to **API 24**.

    After creating the project, Android Studio automatically starts gradle sync. Ensure that the sync succeeds before you continue.

2.  Integrate <Vg k="UIK" /> into your project:

    In `Gradle Scripts/settings.gradle (Project Settings)`, reference the `jitpack` repository by adding the following line to the `repositories` section inside `dependencyResolutionManagement`:

    ```json
    repositories {
        ...
        maven { setUrl("https://jitpack.io") }
    }
    ```

3.  In `Gradle Scripts/build.gradle (Module: <projectname>)`, add the <Vg k="UIK" /> dependency. The `dependencies` section should look like the following:

    ```groovy
    dependencies {
        ...
        implementation 'com.github.agoraio-community:videouikit-android:final:<version>'
        ...
    }
    ```

    Find the [Latest version of the <Vg k="UIK" />](https://jitpack.io/#AgoraIO-Community/VideoUIKit-Android) and replace `<version>` with the one you want to use. For example: _<Vg k="ANDROID_UIKIT_VERSION" />_.

4.  Sync Gradle and import the <Vg k="UIK" />.

5.  Add permissions for network and device access.

    In `/app/Manifests/AndroidManifest.xml`, add the following permissions after `</application>`:

    ```json
    <uses-permission android:name="android.permission.READ_PHONE_STATE"/>
    <uses-permission android:name="android.permission.INTERNET" />
    <uses-permission android:name="android.permission.RECORD_AUDIO" />
    <uses-permission android:name="android.permission.CAMERA" />
    <uses-permission android:name="android.permission.MODIFY_AUDIO_SETTINGS" />
    <uses-permission android:name="android.permission.ACCESS_WIFI_STATE" />
    <uses-permission android:name="android.permission.ACCESS_NETWORK_STATE" />

    <!-- The Agora SDK requires Bluetooth permissions in case users are using Bluetooth devices. -->
    <uses-permission android:name="android.permission.BLUETOOTH" />
    <!-- For Android 12 and above devices, the following permission is also required. -->
    <uses-permission android:name="android.permission.BLUETOOTH_CONNECT" />
    ```
