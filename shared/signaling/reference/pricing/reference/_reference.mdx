### Clean up storage
Depending on the features you enable for your project and your actual usage, <Vg k="SIG"/> stores the relevant data and you incur the corresponding storage charges. To avoid unintended charges, delete stored data that is no longer needed as follows:

| Function | Storage API | Stored Data | How to delete |
|:----------|:----------------------|:-------------|:----------------------------|
| Presence | <code>{presence.setstate[props.ag_platform]}</code> | Temporary user status | <li>Call <code>{presence.removestate[props.ag_platform]}</code> </li>  |
| Storage | <code>{storage.setchannel[props.ag_platform]}</code> <br/> <code>{storage.setuser[props.ag_platform]}</code> | Channel Metadata and User Metadata | <ul><li> Call <code>{storage.removechannel[props.ag_platform]}</code> or <code>{storage.removeuser[props.ag_platform]}</code> </li><li>Disable Storage from the project configuration page of <Vg k="CONSOLE" />. </li><li> Contact [Agora support](mailto:support@agora.io) </li></ul> |
| Lock | <code>{lock.set[props.ag_platform]}</code> | Lock settings and status | <ul><li> Call <code>{lock.remove[props.ag_platform]}</code> </li><li>Disable Locks functionality from the project configuration page of <Vg k="CONSOLE" />. </li></ul>|
