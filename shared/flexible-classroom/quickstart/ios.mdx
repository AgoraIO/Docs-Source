* Installed [CocoaPods](https://guides.cocoapods.org/using/getting-started.html#getting-started) version 1.10 or later.

* If you use Swift, make sure you have version `5.0` or later.

## Project setup

To set up your <Vg k="FC" /> project:

1. Clone the repository:

    ```bash
    git clone https://github.com/AgoraIO-Community/flexible-classroom-ios.git
    ```

1. Update to the supported version of <Vg k="FC"/>:

      ```bash
      cd flexible-classroom-ios
      git checkout release/2.8.11
      ```

## Implement a flexible classroom

To configure and launch your <Vg k="FC" /> from the downloaded source code:

1. Navigate to the `CloudClass-iOS/App` directory:

   ```bash
   cd App
   ```

1. Install project dependencies using the following command:

   ```bash
   pod install
   ```

   You see the dependency package install.

   ![](https://agora-doc.s3.us-east-1.amazonaws.com/images/flexible-classroom/pod-install-demo.png)

1. To open the project in Xcode, use the following command:

    ```bash
    open AgoraEducation.xcworkspace
    ```
   You see the following:

   ![](https://web-cdn.agora.io/docs-files/1648725725804)

1. In Signing & Capabilities under the project **TARGETS**, check **Automatically manage signing**, and configure your Apple developer account and Bundle Identifier.

   ![](https://web-cdn.agora.io/docs-files/1648725848162)

## Test your implementation

1. Connect a physical iOS device to your development device.

1. In Xcode, click **Run**. A moment later, the project is installed on your device. 

1. On the login screen, type in a room name and user name. Select a class type, set the user role to `Teacher` and then press **Enter**. 

   ![](https://web-cdn.agora.io/docs-files/1648726024179)

   You are logged in to <Vg k="FC" /> and presented with the following UI:

   ![](https://web-cdn.agora.io/docs-files/1648635720196)

1. Install <Vg k="FC" /> on a second iOS device. Login with the same credentials but this time set the user role to `Student`.

1. The `Teacher` and `Student` can now interact and communicate using <Vg k="FC" /> features.
