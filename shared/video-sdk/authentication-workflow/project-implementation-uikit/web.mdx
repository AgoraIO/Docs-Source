To integrate authentication into an <Vpl k="CLIENT" /> using <Vg k="UIK" />:

1.  **Set the URL of the server your <Vpl k="CLIENT" /> retrieves tokens from**

    In `<project_name>/src/App.js`, locate the `rtcProps` declarion and replace `token : ''` with the following code:

    ``` javascript
    // The base URL to your token server. For example, https://app-name.herokuapp.com"
    tokenUrl : ''
    ```

<Vg k="COMPANY" /> <Vg k="UIK" /> automatically fetches a token from the token server at `tokenUrl`.
