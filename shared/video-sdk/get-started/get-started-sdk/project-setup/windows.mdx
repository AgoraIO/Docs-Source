1. **Create an MFC dialog-based application**

    1. From the main menu, choose **File** > **New** > **Project**.

    1. Enter **MFC** into the search box and then choose **MFC App** from the result list.
    
    1. In **Project Name**, input `AgoraImplementation` and press **Create** to open the **MFC Application Wizard**.

    1. In **MFC Application Wizard**, select the **Dialog based** option under **Application type**, and then click **Finish**. You see, your projects opens in the **Visual Studio Editor**.


1. **Integrate <Vpd k="SDK" />**

    To integrate the <Vpd k="SDK" /> into your project.

    1. Download [Agora Video SDK](https://download.agora.io/sdk/release/Agora_Native_SDK_for_Windows_v4.0.1_FULL.zip) and unzip the downloaded SDK package.

    1. Copy **sdk** folder of the downloaded SDK package to your project folder.

1. **Configure the project properties**

    Right-click the project name in the **Solution Explorer** window, click **Properties** to configure the following project properties, and click **OK**.

    1. Go to the **C/C++ > General > Additional Include Directories** menu, click **Edit**, and input the following string in the pop-up window:
       
       ```
       ;$(SolutionDir)sdk\high_level_api\include;
       ```

    1. Go to the **Linker > General > Additional Library Directories** menu, click **Edit**, and input the following string in the pop-up window:

       ```
       ;$(SolutionDir)sdk;$(SolutionDir)ThirdParty\libyuv\debug;$(SolutionDir)ThirdParty\DShow
       ```

    1. Go to the **Linker > Input > Additional Dependencies** menu, click **Edit**, and input the following dll in the pop-up window:

       ```
       ;agora_rtc_sdk.dll.lib
       ```

    1. Go to the **C/C++ > Preprocessor > Preprocessor Definitions menu, click **Edit**, and input the following path in the pop-up window:

       ```
       ;_CRT_SECURE_NO_WARNINGS
       ```
