```python
def on_encoded_video_frame(self, uid, image_buffer, length, video_encoded_frame_info:EncodedVideoFrameInfo)
```

<span className="index-desc-onencodedvideoframe">Occurs when an encoded video frame is received.</span>

#### Parameters

| 参数 | 描述 |
|---------------|----------------------------------------------------------------------|
| `uid` | The remote user ID.             |
| `image_buffer` | The video frame buffer.             |
| `length` | The length of the video frame data.             |
| `video_encoded_frame_info`| The encoded video frame information. See [`EncodedVideoFrameInfo`](./apidatatype#encodedvideoframeinfo)         |

#### Returns

Whether to accept the encoded video frame:

- `0`: Do not accept.
- `1`: Accept.
