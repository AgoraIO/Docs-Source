To run the demo code:

1. Navigate to the project directory and install the dependencies:
    
    ```bash
    cd agora-sdk-quickstart
    npm install
    ```

1. Start the development server by executing the following command:

    ```bash
    npm run dev
    ```

1. Open the project in your browser
    
    Launch your browser and navigate to the `localhost` address shown in your terminal.

1. Fill in the app ID and the temporary token you obtained from <Vg k="CONSOLE" />. Use the same channel name you used to generate the token.
1. Click the **Join Channel** button. 
1. Ask a friend to run the same demo in their browser and join the channel using the same app ID, channel name, and temporary token as yours. After successfully joining the channel, you can talk to each other.
1. Click the microphone button at the bottom to switch the microphone on or off.
1. Click the hang-up button to end the call.
