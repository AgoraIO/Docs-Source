1. **Import Magic Leap SDK**
   
   To import the Magic Leap SDK in your Unity Project, follow <Link to = "https://developer-docs.magicleap.cloud/docs/guides/unity/getting-started/import-unity-packages/">Import Unity Tools</Link>.

        1. A warning might appear regarding a new input system and ask you to enable the backends. Click **Yes** which will restart the Unity project.

        1. Proceed to **Add package from Disk** for `com.magicleap.unitysdk` that you downloaded from ML Hub. This might prompt another warning asking if you want to proceed. Click **I Made a Backup, Go Ahead!**. 

1. **Configure Magic Leap SDK Path**

    In **Unity Editor**, go to **Edit** > **Preferences** > **Magic Leap** and set **Lumin Path** to the path where your `…/MLSDK/<Version>` resides.

1. **Configure Project Settings**

    Consult Magic Leap’s <Link to = "https://developer-docs.magicleap.cloud/docs/guides/unity/getting-started/configure-unity-settings/">Configure Project Settings</Link> and configure the Project Settings accordingly: 
    
     1. Enable **Magic Leap** in **XR Plug-in Management** for the Android Platform.

     1. Make sure that **Use ML Audio** is enabled in **XR Plug-in Management** > **Magic Leap Settings**. 

     1. Enable camera and audio recording permissions in **MagicLeap** > **Manifest Settings**. 

     1. Configure **Player** settings (refer to the guide). 

     1. Configure **Universal Rendering Pipeline (URP) Global Settings** (refer to the guide). 

1. **Add Agora Engine**

     1. Download the Agora Engine package for Unity from the <Link to = "https://github.com/AgoraIO-Extensions/Agora_MagicLeap2_Plugin/releases"> Agora Extensions Release</Link> page. 

     1. Double-click or drag the downloaded package into the Unity **Project Asset** window. Click **Import**. Note that the package includes supporting assets from the MagicLeap example, you may exclude them if you are testing with the existing MagicLeap Unity example project to avoid overwriting these files.
