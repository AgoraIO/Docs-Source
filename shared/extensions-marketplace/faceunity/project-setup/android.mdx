<PlatformWrapper platform="android">

1. Unzip the <Vg k="RTEE_FU" /> AR Filter package, and save all `.aar` files to the `/app/libs` folder in your <Vpl k="NAME"/> project.
1. Save the certificate file `authpack.java` to the folder where the app module is located. For example, if the package name is `io.agora.rte.extension.faceunity.example`, the certificate file should be saved to `/app/src/main/java/io/agora/rte/extension/faceunity/example`).
1. Save the model and prop files you need from the resource package to the `/app/src/main/assets` folder in your project. For details of files provided in the resource pack, see [Resource package structure](https://github.com/Faceunity/FULiveDemoDroid/blob/master/doc/Android_Nama_Demo_Guide.md#2-file-structure).

1. In the `app/build.gradle` file, add the following line under `dependencies`:
   
   ```java
   implementation fileTree(dir: "libs", include: ["*.jar", "*.aar"])
   ```

1. To use the required classes, ensure that the following libraries are included in the list of `import` statements in your <Vpl k="NAME"/> activity:
   
    ```java
    import io.agora.rtc2.Constants;
    import io.agora.rtc2.IMediaExtensionObserver;
    import io.agora.rtc2.IRtcEngineEventHandler;
    import io.agora.rtc2.RtcEngine;
    import io.agora.rtc2.RtcEngineConfig;
    import io.agora.rtc2.video.VideoCanvas;

    import io.agora.rte.extension.faceunity.ExtensionManager;
    ```
