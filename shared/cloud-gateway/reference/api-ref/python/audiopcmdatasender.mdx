The `AudioPcmDataSender` class.

### `send_audio_pcm_data`

Sends PCM audio data to the local audio track.

```python
def send_audio_pcm_data(self, frame: PcmAudioFrame)
```

#### Parameters

| Parameter | Description |
|-----------|-------------|
| `frame` | The [`PcmAudioFrame`](#pcmaudioframe) object, containing the PCM audio data to send. |

#### Returns

- 0: Success.
- < 0: Failure.
