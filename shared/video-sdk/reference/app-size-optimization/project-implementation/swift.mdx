#### Delete plugins when integrating using CocoaPods

When integrating the iOS or macOS SDK through CocoaPods, use the `subspecs` field in the Podfile to specify the dynamic libraries you need to integrate. This excludes plug-ins you do not need. For details about the corresponding relationship between dynamic libraries and `subspecs` field, see [subspecs field](#subspecs-field).

Refer to the following samples to use all, none, or some plugins:

<PlatformWrapper platform="ios">
<Tabs>
<TabItem value="all" label="Use all plugins" default>
<CodeBlock language="shell" >
{`# platform :ios, '9.0'
target 'Your App' do
    pod 'AgoraRtcEngine_iOS', '4.3.1'
end`}
</CodeBlock>
</TabItem>

<TabItem value="none" label="Don't use any plugin" default>
<CodeBlock language="shell" >
{`# platform :ios, '9.0'
target 'Your App' do
    pod 'AgoraRtcEngine_iOS', '4.3.1', :subspecs => ['RtcBasic']
end`}
</CodeBlock>
</TabItem>
<TabItem value="some" label="Use some plugins" default>
<CodeBlock language="shell" >
{`# platform :ios, '9.0'
target 'Your App' do
    pod 'AgoraRtcEngine_iOS', '4.3.1', :subspecs => ['RtcBasic', 'AINS', 'AudioBeauty']
end`}
</CodeBlock>
</TabItem>
</Tabs>
</PlatformWrapper>

<PlatformWrapper platform="macos">
<Tabs>
<TabItem value="all" label="Use all plugins" default>
<CodeBlock language="shell" >
{`# platform :ios, '9.0'
target 'Your App' do
    pod 'AgoraRtcEngine_macOS', '4.3.1'
end`}
</CodeBlock>
</TabItem>

<TabItem value="none" label="Don't use any plugin" default>
<CodeBlock language="shell" >
{`# platform :ios, '9.0'
target 'Your App' do

<TabItem value="some" label="Use some plugins" default>
<CodeBlock language="shell" >
{`# platform :ios, '9.0'
target 'Your App' do
    pod 'AgoraRtcEngine_macOS', '4.3.1', :subspecs => ['RtcBasic', 'AINS', 'AudioBeauty']
end`}
</CodeBlock>
</TabItem>
</Tabs>
</PlatformWrapper>


<Admonition type="caution" title="info">
When using the above pod command, make sure that the SDK version you specify includes the plugin you have selected, and that the `subspecs` field is correct, otherwise a pod installation error is reported. For example, if you use `pod 'AgoraRtcEngine_iOS', '4.0.0', :subspecs => ['RtcBasic', 'AgoraRtcAINS']`, an error is reported.
</Admonition>
