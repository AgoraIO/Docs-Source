### Manual <Vpd k="SDK" /> integration

1.	Go to [SDK Downloads](https://docs.agora.io/en/Video/downloads?platform=Web) and download the latest version of <Vpd k="SDK" />. 

2.	Extract the files from the downloaded package to a local directory.

3.	Add the following code to the line before the `<style>` tag in your project:

	``` html
	<script src="./AgoraRTC_N-4.11.1.js"></script>
	```
