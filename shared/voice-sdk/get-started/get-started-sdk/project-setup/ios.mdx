
<PlatformWrapper platform="ios">
    
    1.  [Create a new project](https://help.apple.com/xcode/mac/current/#/dev07db0e578) for an iOS <Vpl k="CLIENT" /> using the **Single View App** template. Make sure you select the **Storyboard** Interface and **Swift** Language. If you have not already added team information, click **Add account...**, input your Apple ID, then click **Next**.

    2.  [Enable automatic signing](https://help.apple.com/xcode/mac/current/#/dev23aab79b4) for your project.

    3.  [Set the target devices](https://help.apple.com/xcode/mac/current/#/deve69552ee5) to deploy your iOS <Vpl k="CLIENT" /> to an iPhone or iPad.

    4.  Add project permission for microphone usage.
        Open **Info** in the project navigation panel, then add the following properties to the [Information Property List](https://help.apple.com/xcode/mac/current/#/dev3f399a2a6):

        | Key                          | Type   | Value                  |
        |------------------------------|--------|------------------------|
        | NSMicrophoneUsageDescription | String | Access the microphone. |
        | NSCameraUsageDescription     | String | Access the camera.     |

    5.  Integrate <Vpd k="SDK" /> into your project.

        1.  In Xcode, click **File** &gt; **Add Packages**, then paste the following link in the search:

        ```
        https://github.com/agorabuilder/AgoraRtcEngine_iOS_Voice_Beta.git
        ```

        You see the **<Vg k="IOS_VOICE_PACKAGE"/>** package.

        2. Click **Add Package**. You see **AgoraRtcKit** in **Package Dependencies** for your project.
    If you already use *CocoaPods* in your <Vpl k="CLIENT" />, <Link target="_blank" to="https://cocoapods.org/pods/AgoraAudio_iOS">install</Link> the <Vpd k="SDK" /> pod.
