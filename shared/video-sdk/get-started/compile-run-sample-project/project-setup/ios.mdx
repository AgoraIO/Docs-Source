### Get the sample project

You can obtain the sample project using the following methods:

- Go to the [SDKs](/sdks) page, download the latest version of the <Vg k="COMPANY" /> <Vpd k="SDK" />, and unzip it.

- Run the following command to clone the Github repository locally:

    ```bash
    git clone git@github.com:AgoraIO/API-Examples.git
    ```

    This repository contains sample projects for all native platforms of <Vg k="COMPANY" /> <Vpd k="SDK" />. The examples for iOS are located under `/iOS`.

    | Path |  	Description |
    |------|---------------|
    | `/iOS/APIExample` |	API usage examples for <Vg k="COMPANY" /> <Vg k="VSDK" />. |
    | `/iOS/APIExample-Audio` | API usage examples for <Vg k="COMPANY" /> <Vg k="ASDK" />. |

### Configure the sample project

<Admonition type="caution" title="information">
This page refers to the video sample project. If you want to run the audio sample project, refer to the following steps and operate in the audio folder.
</Admonition>

1. Open the `samples/APIExample/APIExample/Common/KeyCenter.swift` file and fill in the App ID and App certificate you obtained from the <Vg k="CONSOLE" />.

    ```swift
    class KeyCenter: NSObject {
        // Replace <#Your App ID#> with your App ID
        static let AppId: String = "<#Your App ID#>"
        // Replace #Your certificate# with your App certificate. Leave this field blank if the project allows joining with the App ID only.
        static let Certificate: String? = "<#Your certificate#>"
    }
    ```

1. Open terminal and run `pod install` in the `samples/API-Example` folder to install the SDK and related dependencies. After the installation is complete, the terminal displays 'Pod installation complete!'.
