### `AgoraServiceConfiguration`

Defines the configuration class for initializing the Agora service.  
You can get or set parameters using the corresponding getter/setter methods.

```java
public class AgoraServiceConfiguration {
    private boolean enableAudioProcessor = true;
    private boolean enableAudioDevice = true;
    private boolean enableVideo = false;
    private Object context = null;
    private String appId = null;
    private int areaCode = Constants.AreaCode.AREA_CODE_GLOB.getValue();
    private int channelProfile = Constants.ChannelProfileType.CHANNEL_PROFILE_LIVE_BROADCASTING.getValue();
    private String license = null;
    private int audioScenario = Constants.AudioScenarioType.AUDIO_SCENARIO_DEFAULT.getValue();
    private LogConfig logConfig = new LogConfig();
    private boolean useStringUid = false;
    private boolean useExternalEglContext = false;
    private boolean domainLimit = false;

    public AgoraServiceConfiguration() {
        // All default values are initialized inline
    }
}
```

#### Parameters

| Parameter               | Type        | Description |
|------------------------|-------------|-------------|
| `enableAudioProcessor` | `boolean`   | Whether to enable the audio processor module. Default is `true`. Set to `false` to disable audio track creation. |
| `enableAudioDevice`    | `boolean`   | Whether to enable the audio device module (record/playback). Default is `true`. <br/> **Note:** In recording scenarios, set this to `false`. If set to `true`, the SDK throws an error. |
| `enableVideo`          | `boolean`   | Whether to enable video. Default is `false`. |
| `context`              | `Object`    | User context. On Android, this is the Activity context. |
| `appId`                | `String`    | The App ID of your project. |
| `areaCode`             | `int`       | Region code. See [`AreaCode`](#areacode). Default is `AREA_CODE_GLOB`. |
| `channelProfile`       | `int`       | Channel profile. See [`ChannelProfileType`](#channelprofiletype). Default is `CHANNEL_PROFILE_LIVE_BROADCASTING`. |
| `license`              | `String`    | License key for channel authentication and billing. |
| `audioScenario`        | `int`       | Audio scenario. See [`AudioScenarioType`](#audioscenariotype). Default is `AUDIO_SCENARIO_DEFAULT`. |
| `logConfig`            | `LogConfig` | Configuration for log path, size, and level. See [`LogConfig`](#logconfig). |
| `useStringUid`         | `boolean`   | Whether to use string UIDs. Default is `false` (uses numeric UID). |
| `useExternalEglContext`| `boolean`   | Whether to use the current thread's EGL context as the root context. Applies to Android only. |
| `domainLimit`          | `boolean`   | Whether to limit connection to DNS-resolved domains. Default is `false`. |

### `LocalAccessPointConfiguration`

Defines the configuration for local access points.  
All parameters can be accessed via getter/setter methods.

```java
public class LocalAccessPointConfiguration {
    private String[] ipList;
    private int ipListSize;
    private String[] domainList;
    private int domainListSize;
    private String verifyDomainName;
    private AdvancedConfigInfo advancedConfig;
    private boolean disableAut;

    public LocalAccessPointConfiguration() {
        this.ipList = null;
        this.ipListSize = 0;
        this.domainList = null;
        this.domainListSize = 0;
        this.verifyDomainName = null;
        this.mode = Constants.LocalProxyMode.ConnectivityFirst;
        this.advancedConfig = new AdvancedConfigInfo();
        this.disableAut = true;
    }
}
```

#### Parameters

| Parameter           | Type               | Description |
|--------------------|--------------------|-------------|
| `ipList`           | `String[]`         | IP addresses of the local access point. |
| `ipListSize`       | `int`              | Number of IPs in the list. |
| `domainList`       | `String[]`         | Domain names for the access point. |
| `domainListSize`   | `int`              | Number of domains. |
| `verifyDomainName` | `String`           | Certificate domain name on the access point. Use `""` for SNI (Server Name Indication). |
| `advancedConfig`   | `AdvancedConfigInfo`| Advanced proxy config. See [`AdvancedConfigInfo`](#advancedconfiginfo). |
| `disableAut`       | `boolean`          | Whether to disable vos-aut mode. Default is `true`. |

### `MediaRecorderConfiguration`

Configures recording parameters in the Agora SDK. You can retrieve or set these parameters using getter and setter methods.


```java
public class MediaRecorderConfiguration {
    private String storagePath;
    private int maxDurationMs;
    private int recorderInfoUpdateInterval;
    private int width;
    private int height;
    private int fps;
    private int sampleRate;
    private int channelNum;

    public MediaRecorderConfiguration() {
```

| `maxDurationMs`              | `int`    | Maximum recording duration in milliseconds. Default is `120000`.                            |
| `recorderInfoUpdateInterval` | `int`    | Interval for updating recording info in milliseconds (range: 1000â€“10000).                   |

| `height`                     | `int`    | Height of the recorded video in pixels. Default is `720`.                                   |
