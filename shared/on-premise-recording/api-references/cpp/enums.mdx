## `REMOTE_AUDIO_STATE`

| Value                         | Description                                                                                                                                                                                                        |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `REMOTE_AUDIO_STATE_STOPPED`  | `0`: The remote audio is in the default state. This is reported when `REMOTE_AUDIO_REASON_LOCAL_MUTED (3)`, `REMOTE_AUDIO_REASON_REMOTE_MUTED (5)`, or `REMOTE_AUDIO_REASON_REMOTE_OFFLINE (7)` occurs.            |

| `REMOTE_AUDIO_STATE_FROZEN`   | `3`: The remote audio is frozen, possibly due to `REMOTE_AUDIO_REASON_NETWORK_CONGESTION (1)`.                                                                                                                     |
| `REMOTE_AUDIO_STATE_FAILED`   | `4`: The remote audio failed to start, usually due to `REMOTE_AUDIO_REASON_INTERNAL (0)`.                                                                                                                          |
