### Integrating extensions into your project

When integrating the iOS/macOS SDK through CocoaPods, you can use the `subspecs` field in the Podfile to specify the dynamic libraries you need to integrate to exclude the plugins you don't need to use. The correspondence between dynamic libraries and `subspecs` fields is detailed in the following table:
