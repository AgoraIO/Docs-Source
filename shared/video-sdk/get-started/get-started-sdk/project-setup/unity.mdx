<PlatformWrapper platform="unity">
1. **Create a Unity project**:

    1. In Unity Hub, select **Projects**, then click **New Project**.

    1. In **All templates**, select **3D**. Set the **Project name** and **Location**, then click **Create Project**.

    1. In **Projects**, double-click the project you created. Your project opens in Unity.


1. **Integrate <Vpd k="SDK" />**:
    1. Unzip <Link to ="https://download.agora.io/sdk/release/Agora_Unity_RTC_SDK_4.0.0-beta.1_FULL.zip"> <Vpd k="SDK" /></Link> to a local folder.

    1. In Unity, click **Assets > Import Package > Custom Package**.

    1. Navigate to the <Vpd k="SDK" /> package and click *Open*.

    1. In **Import Unity Package**, click **Import**.
