This page shows how to enable the Broadcast Streaming service in Agora Console.

## Prerequisites 

Before enabling the Broadcast Streaming service, ensure that you meet the following requirements:

- A valid Agora account. If you do not have one, see [Create an Agora account](../reference/manage-agora-account#create-an-agora-account).
- An active Agora project. If you do not have one, see [Create an Agora project](../reference/manage-agora-account#create-an-agora-project).

## Procedure

Follow these steps to enable the Broadcast Streaming service:

1. On the [Project Management](https://console.agora.io/projects) page in Agora Console, click **Config** for the project you want to use.
   ![](https://web-cdn.agora.io/docs-files/1641971710869)
2. Under **Real-time engagement core**, find **Fusion CDN**, and click **Enable**. 
   <img src="https://web-cdn.agora.io/docs-files/1642736194620"  />
	 
	 If Broadcast Streaming is successfully enabled, the **Enable** button changes to the **Config** button.

## Next steps

After enabling the Broadcast Streaming service, you can start live streaming by referring to the following quickstart guides:
- [Start live streaming using Agora domain](agora-domain)
- [Start live streaming using a custom domain](custom-domain) 

The differences between using Agora domain and using a custom domain are as follows:

|                                      | Use Agora domain                      | Use a custom domain                     |
| ------------------------------------ | ------------------------------------------------- | -------------------------------------------------- |
| Time to first live stream | Within minutes                                    | Within hours         |
| Authentication                       | Basic authentication               | Timestamp authentication or origin authentication |
| Supported functions                  | A subset of functions in [API Overview](../restful-api/api-overview)             | All functions in [API Overview](../restful-api/api-overview)                  |
