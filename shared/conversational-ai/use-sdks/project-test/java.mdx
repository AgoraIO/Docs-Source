1. **Join a channel**

    Join the RTC channel in your app, using a user ID that is different from the one you specified for the agent. Use the same channel name and a valid token.

    Alternatively, join the channel using the [Web demo app](https://webdemo-global.agora.io/index.html). Fill in the initialization settings, then select **Quick Start > Audio and Video Call** from the side menu. Enter a user ID that is different from the one you used for the agent. Use the same channel name and a valid token.

1. **Start the <Vpd k="NAME" />**

    In your IDE, click **Run** to start the agent. After successful execution, you see the following log output:

    <CodeBlock language="bash" showLineNumbers>
    {`[main] INFO io.agora.Main - Agent started successfully, joinConvoAIRes:JoinConvoAIRes{agentId='1NT29X0WPCExxxxxWIOYAX0PDQLA5EX', createTs=1742892557, status='RUNNING'}
   [main] INFO io.agora.Main - Listed agents successfully, listConvoAIRes:ListConvoAIRes{data=Data{count=1, list=[Agent{agentId='1NT29X0WPCExxxxxWIOYAX0PDQLA5EX', startTs=1742892560, status='RUNNING'}]}, meta=Meta{total=1, cursor=''}, status='ok'}
   [main] INFO io.agora.Main - The agent left successfully, agentId:1NT29X0WPCExxxxxWIOYAX0PDQLA5EX`}

    After the agent successfully joins RTC channel, it sends a greeting to the user.

    <Admonition type="info">
    If the agent joins a channel and no other users are present, it automatically leaves the channel after a predefined period (120 seconds in the sample code).
    </Admonition>

1. **Interact with the agent**

    Once the user and the agent join the channel, they can interact directly. The agent responds to the user using voice.
