
- **AccessToken2**

    You need to fill the `Authorization: agora token=` field with the <Vg k="MESS_SDK" /> [AccessToken2](../get-started/authentication-workflow) generated from your server.

- **AccessToken**

    You need to fill the `x-agora-token` and `x-agora-uid` fields in the request header with the following information:

    - The <Vg k="MESS_SDK" /> [AccessToken](../get-started/authentication-workflow) generated from your server
    - The <Vg k="MESS_SDK" /> user ID used to generate the <Vg k="MESS" /> token

    Implement HTTP basic authentication or token authentication on the server; otherwise, you may encounter the risk of data leakage.
