
## AgoraService

| API | Description |
|---|---|
| [`AgoraService`](agora-service#agoraservice) | Creates an instance of [`AgoraService`](agora-service) and initializes native components. |
| [`getNativeHandle`](agora-service#getnativehandle) | Gets the native handle associated with this [`AgoraService`](agora-service) instance. |
| [`release`](agora-service#release) | Releases the [`AgoraService`](agora-service) instance and its associated resources. |
| [`initialize`](agora-service#initialize) | Initializes the [`AgoraService`](agora-service) instance with the specified configuration. |
| [`createAgoraMediaComponentFactory`](agora-service#createagoramediacomponentfactory) | Creates and returns an [`AgoraMediaComponentFactory`](agora-media-component-factory) for generating media components. |
| [`getAgoraParameter`](agora-service#getagoraparameter) | Creates and returns an `AgoraParameter` object for managing parameters. |
| [`getSdkVersion`](agora-service#getsdkversion) | Gets the SDK version. |
| [`setLogFile`](agora-service#setlogfile) | Sets the SDK log file path and maximum file size. |
| [`setLogLevel`](agora-service#setloglevel) | Sets the log output level of the SDK. |
| [`setGlobalLocalAccessPoint`](agora-service#setgloballocalaccesspoint) | Sets the global local access point address in local AP mode (also calls the local proxy). |

| API | Description |

| [`setVideoMixingLayout`](agora-media-rtc-recorder#setvideomixinglayout) | Configures the video mixing layout. |
| [`setRecorderConfig`](agora-media-rtc-recorder#setrecorderconfig) | Sets the configuration for the recorder instance. |

## IAgoraMediaRtcRecorderEventHandler

| API | Description |
|---|---|
| [`onConnected`](agora-media-rtc-recorder-eventhandler#onconnected) | Triggered when the connection state changes to `CONNECTION_STATE_CONNECTED(3)`. |
| [`onDisconnected`](agora-media-rtc-recorder-eventhandler#ondisconnected) | Triggered when the connection state changes to `CONNECTION_STATE_DISCONNECTED(1)`. |
| [`onReconnected`](agora-media-rtc-recorder-eventhandler#onreconnected) | Triggered when the connection is re-established (`CONNECTION_STATE_CONNECTED(3)`). |
| [`onConnectionLost`](agora-media-rtc-recorder-eventhandler#onconnectionlost) | Triggered when the SDK loses connection to the Agora channel. |
| [`onUserJoined`](agora-media-rtc-recorder-eventhandler#onuserjoined) | Triggered when a remote user joins the channel. |
| [`onUserLeft`](agora-media-rtc-recorder-eventhandler#onuserleft) | Triggered when a remote user leaves the channel. |
| [`onFirstRemoteVideoDecoded`](agora-media-rtc-recorder-eventhandler#onfirstremotevideodecoded) | Triggered when the first frame of remote video is decoded. |
| [`onFirstRemoteAudioDecoded`](agora-media-rtc-recorder-eventhandler#onfirstremoteaudiodecoded) | Triggered when the first frame of remote audio is decoded. |
| [`onAudioVolumeIndication`](agora-media-rtc-recorder-eventhandler#onaudiovolumeindication) | Reports active speakers, their volumes, and whether the local user is speaking. |
| [`onActiveSpeaker`](agora-media-rtc-recorder-eventhandler#onactivespeaker) | Triggered when an active speaker is detected. |
| [`onRemoteVideoStats`](agora-media-rtc-recorder-eventhandler#onremotevideostats) | Reports remote video statistics. |
| [`onRemoteAudioStats`](agora-media-rtc-recorder-eventhandler#onremoteaudiostats) | Reports remote audio statistics. |
| [`onRecorderInfoUpdated`](agora-media-rtc-recorder-eventhandler#onrecorderinfoupdated) | Triggered when recording information is updated. |
| [`onEncryptionError`](agora-media-rtc-recorder-eventhandler#onencryptionerror) | Triggered when an encryption error occurs. |
| [`onError`](agora-media-rtc-recorder-eventhandler#onerror) | Reports error messages and error codes. |
| [`onTokenPrivilegeWillExpire`](agora-media-rtc-recorder-eventhandler#ontokenprivilegewillexpire) | Triggered 30 seconds before the Token expires. |
| [`onTokenPrivilegeDidExpire`](agora-media-rtc-recorder-eventhandler#ontokenprivilegedidexpire) | Triggered when the Token has expired. |
