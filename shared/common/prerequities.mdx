<PlatformWrapper platform="android">
- Android Studio 4.1 or higher.
- Android SDK API Level 24 or higher.
- A mobile device that runs Android 4.1 or higher.
</PlatformWrapper>

<PlatformWrapper platform="ios">
- Xcode 12.0 or higher.
- Two devices running iOS 9.0 or higher.
</PlatformWrapper>

<PlatformWrapper platform="web">
- A xref:/reference/supported-platforms.adoc#supported_platform_browsers[supported browser].
- A physical media input devices, such as a camera and a microphone.
- A JavaScript package manager such as https://www.npmjs.com/package/npm[npm,role=external,window=_blank].
</PlatformWrapper>

- A computer that can access the Internet.
  Ensure that no firewall is blocking your network communication, otherwise, the project may fail.
- An xref/reference/manage-agora-account.adoc#create_an_agora_account[account] and xref:reference/manage-agora-account.adoc#create_an_agora_project[project].
