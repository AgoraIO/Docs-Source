In addition to integrating the Chat SDK into your project through npm, you can also:

* **Manually download the [Chat SDK for Web](https://www.npmjs.com/package/agora-chat)**:

    a. In the SDK folder, find the JS file in the `libs` folder and save it to your project directory.

    b. Open the HTML file in your project directory and add the following code to refer to the JS file:

    ```javascript
    <script src="path to the JS file"></script>
    ```

* **Install the Demo app**:

    See [demo  <Vpl k="CLIENT" />](../reference/downloads) to get started with <Vg k="CHAT" />.
