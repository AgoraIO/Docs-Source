## Sample project

The complete <Link to="{{Global.AGORA_MARKETPLACE_BASE_URL}}/FaceUnity/android/app/src/main/java/io/agora/rte/extension/faceunity/example/MainActivity.java">Android <Vg k="RTEE_FU"/> sample code and project</Link> is available on GitHub.

#### Run the sample project

1. Clone the repository:
    ```shell
    git clone https://github.com/AgoraIO-Community/AgoraMarketPlace.git
    ```
1. On the <Link to="{{Global.AGORA_CONSOLE_URL}}/marketplace/list/all">Extensions Marketplace Downloads</Link> page, download the Android package of FaceUnity AR Filter. Unzip the package, and save all `.aar` files to the `FaceUnity/android/app/libs` path.
1. Contact Agora to get the certificate file and resource package.
1. Save the certificate file `authpack.java` to `FaceUnity/android/app/src/main/java/io/Agora/rte/extension/faceunity/example`.
1. Save the required model and prop files from the resource package to `FaceUnity/android/app/src/main/assets/face_unity ` under the project folder.
1. Open the sample project `FaceUnity/android` in Android Studio.
1. Sync the project with Gradle files.
1. Open the `FaceUnity/android/app/src/main/java/io/Agora/rte/extension/FaceUnity/example/Config.java` file, and replace `<YOUR_APP_ID>` with your App ID. To get an App ID, see [Getting Started with Agora](../../reference/manage-agora-account#get-the-app-id).

   ``` java
   public interface Config {
       String mAppId = "<YOUR_APP_ID>";
       String mToken = null;
   }
   ```
1. Connect an Android device (not an emulator), and run the project.
