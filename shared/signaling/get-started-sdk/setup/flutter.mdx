### Create a project

To create a new Flutter project, follow these steps:

1. Open a terminal and execute the following command:

    ```bash
    flutter create getting_started
    ```

1. Open the project in your IDE. Edit the `pubspec.yaml` file to add the following dependency:

    ```yaml
    dependencies:
    # Add the RTM Signaling SDK dependency, use the latest version of agora_rtm
    agora_rtm: ^2.2.1
    ```
