1. In `src/signaling_manager/config.json`, replace [appId](../reference/manage-agora-account#get-the-app-id) with
  the value from <Link target="_blank" to="{{global.AGORA_CONSOLE_URL}}"><Vg k="CONSOLE" /></Link>.

1. Start the Signaling samples Web app.

    1. In Terminal, navigate to `<samples_root>`.

    1. Run the following commands:

      ``` bash
      npm run build
      npm run start:dev
      ```

1. Run the proxy server.

   For security reasons, browsers restrict cross-origin HTTP requests initiated from scripts. To make requests to your
  token server, this sample app uses the [NodeJS corsAnywhere proxy](/src/authentication_guide/proxy.js) which adds [CORS
  headers](https://developer.mozilla.org/en-US/docs/Web/HTTP/CORS) to the proxied requests. To start this server:

    1. In a new Terminal, navigate to `<samples_root>/src/authentication_guide/`

    1. Run the following command:

        ``` bash
        node proxy
        ```

1. For each user you want to communicate with:

    1. Open a browser instance at `http://localhost:9000/authentication_guide/`.

    1. Enter a user ID.

    1. Press **Login**.

You can send messages between users.
