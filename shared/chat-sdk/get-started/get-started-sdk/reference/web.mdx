In addition to integrating the Chat SDK into your project through npm, you can also manually download the [Chat SDK for Web](/sdks):

a. In the SDK folder, save `libs/Agora-chat.js` to your project directory.

b. Open the HTML file in your project directory and add the following code to refer to the JS file:

    ```javascript
    <script src="path to the JS file"></script>
    ```
