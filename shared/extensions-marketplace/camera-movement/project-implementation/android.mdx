### Register the plugin

After initializing `RtcEngine`, call `loadExtensionProvider` to load the camera movement plugin library, and then call `registerExtension` to register the camera movement plugin.

<Admonition type="caution" title="Caution">
Make sure to load and register the camera movement plugin before calling `enableVideo` to enable the video module.
</Admonition>

```java
// Load the rhythm dynamic library
mRtcEngine.loadExtensionProvider("agora_portrait_rhythm_extension");
// Register the rhythm plugin
mRtcEngine.registerExtension("agora_video_filters_portrait_rhythm", "portrait_rhythm", Constants.MediaSourceType.PRIMARY_CAMERA_SOURCE);
// Enable the video module
mRtcEngine.enableVideo();
```

### Enable the camera movement plugin

Make surae to call `startPreview` to start the video preview, and then call `enableExtension` to turn on or off the camera rhythm plugin.

```java
// Turn on camera motion
mRtcEngine.enableExtension("agora_video_filters_portrait_rhythm", "portrait_rhythm", true);
```

### Set or switch camera movement effects

To set or switch the camera movement effect, call `setExtensionProperty` (Android) and pass in different key and value to achieve effects such as heartbeat rhythm, portrait rhythm, and front and back camera movement. For details, see the [key-value description](../integrate/camera-movement#key-value-description).

```java
// Set to heartbeat rhythm effect
mRtcEngine.setExtensionProperty("agora_video_filters_portrait_rhythm", "portrait_rhythm", "mode", "1");
```
