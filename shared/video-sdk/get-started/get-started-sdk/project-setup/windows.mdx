
<PlatformWrapper platform="windows">

1. **Create a Windows project**

    See [Create a C++ console app project](https://docs.microsoft.com/en-us/cpp/build/vscpp-step-1-create?view=msvc-150) to create a C++ console app on Visual Studio.

1. **Integrate <Vpd k="SDK" />**

    To integrate the <Vpd k="SDK" /> into your project.

    1. Go to [SDK Downloads](./downloads?platform=Windows), download the latest version of the Agora SDK, and unzip the downloaded SDK package.
    2. Copy the **x86** or **x86_64** folder of the downloaded SDK package to your project files according to your development environment.

1. **Configure the project properties**

    Right-click the project name in the **Solution Explorer** window, click **Properties** to configure the following project properties, and click **OK**.

    1. Go to the **C/C++ > General > Additional Include Directories** menu, click **Edit**, and input **$(SolutionDir)include** in the pop-up window.

    1. Go to the **Linker > General > Additional Library Directories** menu, click **Edit**, and input **$(SolutionDir)** in the pop-up window.

    1. Go to the **Linker > Input > Additional Dependencies** menu, click **Edit**, and input **agora_rtc_sdk.dll.lib** in the pop-up window.
