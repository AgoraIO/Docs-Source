### Create a project

Create your new Flutter app by following the [steps for your IDE](https://docs.flutter.dev/get-started/test-drive?tab=vscode#create-app). To create a new project named `agora_project` from Terminal, run the following command:

```bash
flutter create agora_project
cd agora_project
```

### Install the SDK

Install the <Vg k="COMPANY" /> <Vpd k="SDK" /> and other dependencies.

1. To add the [latest version](https://pub.dev/packages/agora_rtc_engine) of Agora <Vpd k="SDK" /> to your Flutter project, execute the following command:

    ```bash
    flutter pub add agora_rtc_engine
    ```

2. To install the permission processing plugin, use:

    ```bash
    flutter pub add permission_handler
    ```

    Your `pubspec.yaml` file should should look like the following:

        ```yaml
        environment:
        sdk: '>=3.2.3 <4.0.0'
        # Dependencies
        dependencies:
        flutter:
            sdk: flutter
        # The following adds the Cupertino Icons font to your application.
        # Use with the CupertinoIcons class for iOS style icons.
        cupertino_icons: ^1.0.8
        # Agora Flutter SDK, please use the latest version 
        agora_rtc_engine: ^6.5.0
        # Permissions handler 
        permission_handler: ^11.3.1
        ```

1. To install the dependencies, execute the following command in the project path:

    ```bash
    flutter pub get
    ```
