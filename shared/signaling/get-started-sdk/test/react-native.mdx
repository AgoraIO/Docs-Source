To test the sample code, follow these steps:

1. **Generate a <Vg k="SIG" /> token using the [Token Builder](https://agora-token-generator-demo.vercel.app/):**

   1. From the <Vg k="COMPANY" /> product dropdown, select **Signaling**.
   1. Enter your App ID and App Certificate from the <Link to="{{Global.AGORA_CONSOLE_URL}}"><Vg k="CONSOLE" /></Link>. 
   1. Enter the `USER_ID` you used in your code in the User ID field. Leave the other fields blank.
   1. Click **Generate Token**.

2. **Update the configuration:**

   In `signaling.tsx`, replace the following placeholders in the `CONFIG` object with your actual values:

   ```tsx
   const CONFIG = {
      APP_ID: 'your-actual-app-id',    // Your Agora App ID
      TOKEN: 'your-generated-token',   // The token you generated
      CHANNEL_ID: 'demo',              // Your chosen channel name
      USER_ID: '1'                     // Unique identifier for the user
   };
   ```

3. **Run the app from the project root:**

   * **Android:**

      ```bash
      npm run android
      ```

   * **iOS:**

      ```bash
      npm run ios
      ```

4. **Test on the first device:**

   * Install and launch the app.
   * Wait for the connection status indicator (dot in header) to turn green.
   * Once connected, the app automatically subscribes to the specified channel.

5. **Test on the second device:**

   * Install and launch the app on a second device.
   * Wait for the green connection indicator.
   * Type a message in the input field and tap **Send**.
   * Verify the message appears on the first device.

6. **Test bidirectional messaging:**

   * Send messages from the first device to the second device.
   * Verify messages appear in real-time on both devices.
   * Check the console logs for connection status and message events.
