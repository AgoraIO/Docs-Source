The `VideoEncodedFrameObserver` class.

### `OnEncodedVideoFrame`

Occurs when an encoded video frame is received.

```go
OnEncodedVideoFrame func(uid string, imageBuffer []byte, frameInfo *EncodedVideoFrameInfo) bool
```

#### Parameters

| Parameter | Description |
|---------------|----------------------------------------------------------------------|
| `uid` | The user ID of the remote user.             |
| `imageBuffer` | The video frame buffer.             |
| `frameInfo`| The encoded video frame information. See [`EncodedVideoFrameInfo`](./apidatatype#encodedvideoframeinfo) for details.         |

#### Returns

Whether to accept the encoded video frame:

- `true`: Accept.
- `false`: Do not accept.
