## 3. Integrate the <Vg k="SIG_SDK" />

Refer to the following steps to integrate the <Vg k="SIG_SDK" /> into the sample project:

1.  Download the [<Vg k="SIG" />](https://github.com/AgoraIO/RTM) repository on Github, and locate the `Agora-RTM-Tutorial-Linux-Server` folder.

2.  Download the latest version of [<Vg k="SIG_SDK" /> (Linux C++)](../reference/downloads.html) and extract the files.

3.  Copy the `*.so` file in the SDK folder into the `lib` directory of the project folder, and copy the `*.h` file in the SDK folder into the `include` directory of the project folder.

## 4. Run the sample project

Refer to the following steps to run the sample project:

1.  Go to the project directory, and run the following command to build the project:
    ```shellscript
           $ ./build.sh build
    ```
2.  Run the following command to run the project. Replace &lt;Your App ID&gt; with your App ID.
    ```shellscript
        $ ./build/rtmServerDemo <Your App ID>
    ```
