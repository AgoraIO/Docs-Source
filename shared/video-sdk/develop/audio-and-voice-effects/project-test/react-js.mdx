1. **Test Audio Mixing**

    Select an audio file using the file picker, and then click **Start audio mixing**.
    You hear the audio file played in the channel mixed with the microphone audio. Press the same button again to stop the audio mixing.

1. **Change the default audio route**
    
    Choose a device from the **Output Device** dropdown, then speak into the microphone connected to your browser app. The audio is routed to the selected playback device on the development device.
