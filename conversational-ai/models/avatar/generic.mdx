The `generic` avatar vendor allows you to integrate a custom avatar provider into the Conversational AI Engine.

### Sample configuration

The following example shows a starting `avatar` parameter configuration you can use when you [Start a conversational AI agent](/conversational-ai/rest-api/agent/join).

```json
"avatar": {
  "vendor": "generic",
  "params": {
    "api_key": "<AVATAR_API_KEY>",
    "api_base_url": "<AVATAR_API_BASE_URL>",
    "avatar_id": "<AVATAR_ID>",
    "agora_appid": "<AGORA_APP_ID>",
    "agora_token": "<AGORA_TOKEN>",
    "agora_channel": "<AGORA_CHANNEL>",
    "agora_uid": "<AGORA_UID>",
    "quality": "high",
    "version": "v1",
    "video_encoding": "H264",
    "activity_idle_timeout": 120,
    "area": "NORTH_AMERICA"
  }
}
```

### Key parameters

<ParameterList title="params" required={true}>
  <Parameter name="api_key" type="string" required={true}>
    The API key used for authentication with the avatar provider.
  </Parameter>
  <Parameter name="api_base_url" type="string" required={true}>
    The base URL of the avatar provider's API endpoint.
  </Parameter>
  <Parameter name="avatar_id" type="string" required={true}>
    The unique identifier for the avatar to use.
  </Parameter>
  <Parameter name="agora_appid" type="string" required={true}>
    The Agora App ID for the avatar's RTC connection.
  </Parameter>
  <Parameter name="agora_token" type="string" required={false}>
    The RTC token that authorizes the avatar to join the video channel. Generate this token using your Agora project credentials.
  </Parameter>
  <Parameter name="agora_channel" type="string" required={true}>
    The Agora channel name the avatar joins.
  </Parameter>
  <Parameter name="agora_uid" type="string" required={true}>
    The unique identifier for the avatar's RTC connection. This must be different from other participants in the channel.
  </Parameter>
  <Parameter name="quality" type="string" possibleValues="low, medium, high" required={false}>
    The video quality for avatar rendering.

    - `"high"` (720p)
    - `"medium"` (480p)
    - `"low"` (360p)

    Higher quality provides better visual experience but requires more bandwidth.
  </Parameter>
  <Parameter name="version" type="string" required={false}>
    The API version to use. For example, `v1`.
  </Parameter>
  <Parameter name="video_encoding" type="string" required={false}>
    The video encoding format. For example, `H264`.
  </Parameter>
  <Parameter name="activity_idle_timeout" type="integer" required={false}>
    The number of seconds of inactivity before the avatar session times out.
  </Parameter>
  <Parameter name="area" type="string" required={false}>
    The deployment region for the avatar service. For example, `NORTH_AMERICA`.
  </Parameter>
</ParameterList>
