### Integrate Encryption Library with CocoaPods

To integrate the encryption library through CocoaPods:

1. Install CocoaPods. Refer to the installation guide on [Getting Started with CocoaPods](https://guides.cocoapods.org/using/getting-started.html#getting-started).

1. In the terminal, open the project path and run the `pod init` command to create a `Podfile` in the project folder.

1. Open the `Podfile` and replace its contents with the following: 
    
    ```shell
    # platform :ios, '9.0'
    target 'Your App' do
    pod 'AgoraRtcEngine_iOS_Crypto', '~> version'
    end
    ```

    Replace `Your App` with your target name and `version` with the version of the SDK you need to integrate.

1. Back in the terminal, run the `pod install` command to install the <Vg k="COMPANY" /> <Vpd k="SDK"/>. After the SDK is successfully installed, the terminal shows `Pod installation complete!`, and you see the `.xcworkspace` file in the project folder.

1. Open the `.xcworkspace` file in Xcode.
