### v1.1.3

#### Fixed issues

This release fixes the occasional issue of jagged background images on Chrome for Android.

### v1.1.2

#### New features

You can now specify the `fit`  property when calling `setOptions`. This sets how the background is resized to fit the container. For API details, see [Virtual background](/interactive-live-streaming/enable-features/virtual-background).

#### Compatibility changes

Virtual Background supports Agora Video SDK for Web v4.15.0 or later.

### v1.1.1

#### New features

You can now call `checkCompatibility` and test if AI Noise Suppression extension is supported on the current browser. For API details, see [Virtual background](/interactive-live-streaming/enable-features/virtual-background).

#### Fixed issues

A black bar is no longer displayed to the left of the virtual background.

### v1.1.0

#### New features

You can create multiple `VirtualBackgroundProcessor` instances to process multiple video streams.
