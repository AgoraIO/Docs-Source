### Implement the geofencing logic

To implement geofencing in your <Vpl k="CLIENT" />, in `Assets/Agora-RTC-Plugin/Agora-Unity-RTC-SDK/Code/JoinChannelVideo.cs`,
    replace
    ``` csharp
    RtcEngineContext context = new RtcEngineContext(_appID, 0, true,
    CHANNEL_PROFILE_TYPE.CHANNEL_PROFILE_LIVE_BROADCASTING,
    AUDIO_SCENARIO_TYPE.AUDIO_SCENARIO_DEFAULT);
    ```
    with
    ``` csharp
    RtcEngineContext context = new RtcEngineContext(_appID, 0, true,
    CHANNEL_PROFILE_TYPE.CHANNEL_PROFILE_LIVE_BROADCASTING,
    AUDIO_SCENARIO_TYPE.AUDIO_SCENARIO_DEFAULT, AREA_CODE.AREA_CODE_NA);
    ```
Now, when <Vpd k="SDK" /> fails to connect to the specified region <Vg k="AGORA_BACKEND"/>, it throws an error instead of connecting to the <Vg k="AGORA_BACKEND"/> located in other regions.
