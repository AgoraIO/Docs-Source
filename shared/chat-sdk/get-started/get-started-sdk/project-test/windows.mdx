<PlatformWrapper platform="windows">

1. Create an <Vpl k="CLIENT" /> instance for the first user:

    1. [Register a user](enable#register-a-user) in <Link to="{{Global.AGORA_CONSOLE_URL}}"><Vg k="CONSOLE" /></Link> and [Generate a user token](enable#generate-a-user-token).

    1. In the `MainWindow` class, update `userId`, `token`, and `appKey` with values from <Vg k="CONSOLE" />.  To get your app key, see [Get <Vg k="CHAT"/> project information](enable#get-chat-project-information).

    1. Click the **Start** button. You see the following interface:
    
        ![image](https://agora-doc.s3.us-east-1.amazonaws.com/images/chat/chat-windows-interface.png)   

1. Create an <Vpl k="CLIENT" /> instance for the second user:

   1. Register a second user in <Vg k="CONSOLE" /> and generate a user token.

    1. Build a second instance of your <Vpl k="CLIENT" /> after updating `userId` and `token` with values for the second user. Make sure you use the same `appKey` as for the first user.

    1. Click the **Start** button to launch the second instance. 

1. On each instance, click **Join** to log in to <Vg k="COMPANY" /> <Vg k="CHAT" />. 

1. Edit the recipient name in each <Vpl k="CLIENT" /> instance to show the user ID of the user logged in to the other instance.

1. Type a message in the **Message** box of either instance and press **`>>`**. The message is sent and appears on the other app. 
    
    ![image](https://agora-doc.s3.us-east-1.amazonaws.com/images/chat/chat-windows-communication.png)

1. Press **Leave** to log out from <Vg k="COMPANY" /> <Vg k="CHAT" />.

</PlatformWrapper>
