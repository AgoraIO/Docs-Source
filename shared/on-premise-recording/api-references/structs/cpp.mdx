Configures global parameters for the [`IAgoraService`](#iagoraservice). These parameters are set when calling [`initialize`](#initialize).

#### Member parameters

| Parameter | Data Type | Description |
|-----------|-----------|-------------|
| `enableAudioProcessor` | `bool` | Whether to enable the audio processing module. `true` (default) enables it. `false` disables it; audio tracks cannot be created if disabled. |
| `enableAudioDevice` | `bool` | Whether to enable the audio device module. `true` (default) enables it. `false` disables it. <br/><Admonition type="caution" title="Note">In recording scenarios, set `enableAudioDevice` to `false`; otherwise, the SDK returns an error because the recorder instance has no audio device.</Admonition> |
| `enableVideo` | `bool` | Whether to enable video. `true` enables video. `false` (default) disables video. |
| `appId` | `const char*` | The project’s App ID. |
| `areaCode` | `unsigned int` | Supported area code. Default is `AREA_CODE_GLOB`. |
| `channelProfile` | `agora::CHANNEL_PROFILE_TYPE` | Channel profile. See [`CHANNEL_PROFILE_TYPE`](#channel_profile_type). |
| `license` | `const char*` | License used for authentication when joining a channel. Charges are based on the license. |
| `audioScenario` | `rtc::AUDIO_SCENARIO_TYPE` | Audio scenario. See [`AUDIO_SCENARIO_TYPE`](#audio_scenario_type). |
| `logConfig` | `commons::LogConfig` | Log configuration. Used to set log path, size, and level. See [`LogConfig`](#logconfig). |
| `useStringUid` | `bool` | Whether to enable string user ID. `true` enables it. `false` (default) disables it. |
| `domainLimit` | `bool` | Whether to enable domain restriction. `true` allows only servers resolved via DNS. `false` (default) has no restriction. |

### EncryptionConfig

```cpp
struct EncryptionConfig {
  ENCRYPTION_MODE encryptionMode;
  const char* encryptionKey;
  uint8_t encryptionKdfSalt[32];
  bool datastreamEncryptionEnabled;

  EncryptionConfig()
      : encryptionMode(AES_128_GCM2),
        encryptionKey(OPTIONAL_NULLPTR),
        datastreamEncryptionEnabled(false) {
    memset(encryptionKdfSalt, 0, sizeof(encryptionKdfSalt));
  }

  const char* getEncryptionString() const {
    switch (encryptionMode) {
      case AES_128_XTS:
        return "aes-128-xts";
      case AES_128_ECB:
        return "aes-128-ecb";
      case AES_256_XTS:
        return "aes-256-xts";
      case SM4_128_ECB:
        return "sm4-128-ecb";
      case AES_128_GCM:
        return "aes-128-gcm";
      case AES_256_GCM:
        return "aes-256-gcm";
      case AES_128_GCM2:
        return "aes-128-gcm-2";
      case AES_256_GCM2:
        return "aes-256-gcm-2";
      default:
        return "aes-128-gcm-2";
    }
    return "aes-128-gcm-2";
  }
};
```

Configures the built-in encryption feature.

| Parameter | Data Type | Description |
|-----------|-----------|-------------|
| `encryptionMode` | `ENCRYPTION_MODE` | Built-in encryption mode. Recommended: `AES_128_GCM2` or `AES_256_GCM2`. See [`ENCRYPTION_MODE`](#encryption_mode). |
| `encryptionKey` | `const char*` | Encryption key in string format. Recommended 32 bytes. Must be set; otherwise SDK returns `ERR_INVALID_ARGUMENT` (-2). |
| `encryptionKdfSalt` | `uint8_t[32]` | 32-byte salt. Valid for `AES_128_GCM2`/`AES_256_GCM2`. Must not be all zeros. |
| `datastreamEncryptionEnabled` | `bool` | Enable data stream encryption: `true` to enable, `false` (default) to disable. |
