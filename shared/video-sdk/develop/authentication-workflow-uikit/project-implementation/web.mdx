To integrate authentication into an {props.CLIENT} using {data.UIK}:

1. Set the URL of the server your {props.CLIENT} retrieves tokens from*

In `<project_name>/src/App.js`, locate the `rtcProps` declarion and replace `token : ''` with the following code:

```js
// The base URL to your token server. For example, https://app-name.herokuapp.com"
tokenUrl : ''
```

{data.COMPANY} {data.UIK} automatically fetches a token from the token server at `tokenUrl`.
