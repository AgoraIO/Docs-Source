This guide shows you how to connect a Coze chatbot to <Vpd k="NAME" />, enabling voice conversations with your bot. By integrating speech recognition and synthesis, you can transform a text-based Coze bot into a fully voice-enabled assistant.

## Prerequisites

Before starting, ensure you have:
- <Vpd k="NAME" /> playground running. See the [Agent quickstart](../get-started/quickstart)
- API keys for your voice services:
    - STT service like [Deepgram](https://deepgram.com/)
    - TTS service like [Fish.Audio](https://fish.audio/)
    - Agora RTC credentials from [Agora Console](https://console.agora.io/v2)
- Coze credentials:
    - Coze Bot ID
    - Coze Token
    - Coze Base URL (only for non-global environments)


## Set up your Coze bot with voice

Set up your voice-enabled Coze bot with these steps:

1. Open the playground at `http://localhost:3000`

2. Select the `voice_assistant` graph type

3. Configure modules:
    1. Click **Module Picker**
    1. Select your preferred STT and TTS modules
    1. For the LLM module, select **Coze Chat Bot**
    1. Click **Save Changes**

4. Configure Coze properties:
    1. Click the settings button next to the graph selector
    1. Enter your Coze Bot ID and token
    1. Add Base URL if using a non-global environment
    1. Click **Save Changes**

5. Start your bot:
    1. Click **Connect** to initialize the agent
    1. Wait a few seconds for startup
    1. Begin speaking to your Coze bot


## Azure STT integration

For Azure speech recognition:

1. Select the `voice_assistant_integrated_stt` graph type
2. Configure Azure credentials in the RTC module
3. Follow the remaining configuration steps above

## Troubleshooting

Common issues and solutions:

| Issue | Solution |
|-------|----------|
| **Bot not responding** | <ul><li>Verify Coze Bot ID and Token are correct</li><li>Check Coze bot status in Coze Console</li><li>Ensure bot is published and active</li></ul> |
| **Missing bot capabilities** | <ul><li>Skills may need to be reconfigured in Coze</li><li>Verify tool permissions in Coze console</li><li>Update the bot and republish</li></ul> |
| **Base URL issues** | <ul><li>Only set Base URL if using non-global environment</li><li>Remove trailing slashes from URL</li><li>Ensure proper URL format</li></ul> |
