| API | Description |
|-----|-------------|
| [`initialize`](agora-media-rtc-recorder#initialize) | Initializes the recorder instance. |

| [`subscribeAllAudio`](agora-media-rtc-recorder#subscribeallaudio) | Subscribes to and records audio from all remote users in the channel. |
| [`subscribeAllVideo`](agora-media-rtc-recorder#subscribeallvideo) | Subscribes to video from all remote users in the channel. |
| [`unsubscribeAllAudio`](agora-media-rtc-recorder#unsubscribeallaudio) | Stops subscribing to audio from all remote users in the channel. |
| [`unsubscribeAllVideo`](agora-media-rtc-recorder#unsubscribeallvideo) | Stops subscribing to video from all remote users in the channel. |
| [`subscribeAudio`](agora-media-rtc-recorder#subscribeaudio) | Subscribes to and records audio from a specified remote user. |
| [`unsubscribeAudio`](agora-media-rtc-recorder#unsubscribeaudio) | Stops subscribing to audio from a specified remote user. |
| [`subscribeVideo`](agora-media-rtc-recorder#subscribevideo) | Subscribes to video from a specified remote user. |
| [`unsubscribeVideo`](agora-media-rtc-recorder#unsubscribevideo) | Stops subscribing to video from a specified remote user. |
| [`setAudioVolumeIndicationParameters`](agora-media-rtc-recorder#setaudiovolumeindicationparameters) | Sets the interval for the [`onAudioVolumeIndication`](iagoramediartcrecordereventhandler#onaudiovolumeindication) callback. |
| [`setVideoMixingLayout`](agora-media-rtc-recorder#setvideomixinglayout) | Sets the layout for mixed video streams. |
| [`setRecorderConfig`](agora-media-rtc-recorder#setrecorderconfig) | Configures the recorder instance. |
| [`enableAndUpdateVideoWatermarks`](agora-media-rtc-recorder#enableandupdatevideowatermarks) | Adds a watermark to the video stream. |
| [`disableVideoWatermarks`](agora-media-rtc-recorder#disablevideowatermarks) | Disables video watermarks. |
| [`startRecording`](agora-media-rtc-recorder#startrecording) | Starts recording. |
| [`stopRecording`](agora-media-rtc-recorder#stoprecording) | Stops recording. |
| [`startSingleRecordingByUid`](agora-media-rtc-recorder#startsinglerecordingbyuid) | Starts recording for a specified user ID. |
| [`stopSingleRecordingByUid`](agora-media-rtc-recorder#stopsinglerecordingbyuid) | Stops recording for a specified user ID. |
| [`setRecorderConfigByUid`](agora-media-rtc-recorder#setrecorderconfigbyuid) | Sets recording configurations for a specific user. |
| [`enableAndUpdateVideoWatermarksByUid`](agora-media-rtc-recorder#enableandupdatevideowatermarksbyuid) | Adds a watermark to the video stream of a specific user. |
| [`disableVideoWatermarksByUid`](agora-media-rtc-recorder#disablevideowatermarksbyuid) | Disables the watermark for a specific user's video stream. |
| [`registerRecorderEventHandle`](agora-media-rtc-recorder#registerrecordereventhandle) | Registers a recorder event handler. |
| [`unregisterRecorderEventHandle`](agora-media-rtc-recorder#unregisterrecordereventhandle) | Unregisters the recorder event handler. |
| [`setGlobalLocalAccessPoint`](agora-media-rtc-recorder#setgloballocalaccesspoint) | Sets the global local access point address in local AP mode. |
| [`renewToken`](agora-media-rtc-recorder#renewtoken) | Renews the token. |

## IAgoraMediaRtcRecorderEventHandler

| API | Description |
|-----|-------------|
| [`onConnected`](agora-media-rtc-recorder-eventhandler#onconnected) | Triggered when the SDK connects to the Agora channel (`CONNECTION_STATE_CONNECTED(3)`). |
| [`onDisconnected`](agora-media-rtc-recorder-eventhandler#ondisconnected) | Triggered when the SDK disconnects from the Agora channel (`CONNECTION_STATE_DISCONNECTED(1)`). |
| [`onReconnected`](agora-media-rtc-recorder-eventhandler#onreconnected) | Triggered when the connection state changes back to `CONNECTION_STATE_CONNECTED(3)`. |
| [`onConnectionLost`](agora-media-rtc-recorder-eventhandler#onconnectionlost) | Triggered when the SDK loses connection to the Agora channel. |
| [`onUserJoined`](agora-media-rtc-recorder-eventhandler#onuserjoined) | Triggered when a remote user joins the channel. |
| [`onUserLeft`](agora-media-rtc-recorder-eventhandler#onuserleft) | Triggered when a remote user leaves the channel. |
| [`onFirstRemoteVideoDecoded`](agora-media-rtc-recorder-eventhandler#onfirstremotevideodecoded) | Triggered when the first frame of remote video is decoded. |
| [`onFirstRemoteAudioDecoded`](agora-media-rtc-recorder-eventhandler#onfirstremoteaudiodecoded) | Triggered when the first frame of remote audio is decoded. |
| [`onAudioVolumeIndication`](agora-media-rtc-recorder-eventhandler#onaudiovolumeindication) | Reports the speaker's UID, their volume, and whether the local user is speaking. |
| [`onActiveSpeaker`](agora-media-rtc-recorder-eventhandler#onactivespeaker) | Triggered when an active speaker is detected. |
| [`onUserVideoStateChanged`](agora-media-rtc-recorder-eventhandler#onuservideostatechanged) | Triggered when a remote user's video state changes. |
| [`onUserAudioStateChanged`](agora-media-rtc-recorder-eventhandler#onuseraudiostatechanged) | Triggered when a remote user's audio state changes. |
| [`onRemoteVideoStats`](agora-media-rtc-recorder-eventhandler#onremotevideostats) | Reports statistics for the received remote video stream. |
| [`onRemoteAudioStats`](agora-media-rtc-recorder-eventhandler#onremoteaudiostats) | Reports statistics for the received remote audio stream. |
| [`onRecorderStateChanged`](agora-media-rtc-recorder-eventhandler#onrecorderstatechanged) | Triggered when the local recorder state changes. |
| [`onRecorderInfoUpdated`](agora-media-rtc-recorder-eventhandler#onrecorderinfoupdated) | Triggered when recorder information is updated. |
| [`onEncryptionError`](agora-media-rtc-recorder-eventhandler#onencryptionerror) | Reports an encryption-related error. |
| [`onError`](agora-media-rtc-recorder-eventhandler#onerror) | Reports an error code and corresponding message. |
| [`onTokenPrivilegeWillExpire`](agora-media-rtc-recorder-eventhandler#ontokenprivilegewillexpire) | Triggered 30 seconds before the current token expires. |
| [`onTokenPrivilegeDidExpire`](agora-media-rtc-recorder-eventhandler#ontokenprivilegedidexpire) | Triggered when the token expires. |
