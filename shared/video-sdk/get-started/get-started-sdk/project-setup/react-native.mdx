To download and add the <Vg k="COMPANY" /> <Vpd k="SDK"/> to your React Native project, choose either of the following methods:

{`npm i --save react-native-agora`}
</CodeBlock>

<TabItem value="yarn" label="yarn">

yarn add react-native-agora`}
</CodeBlock>
