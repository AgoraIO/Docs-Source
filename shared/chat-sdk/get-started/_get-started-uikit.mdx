## Understand the tech

The following figure shows the workflow of how clients send and receive peer-to-peer messages:

<details>
<summary>Chat UI kit workflow</summary>

![](https://agora-doc.s3.us-east-1.amazonaws.com/images/chat/ui-kit-quickstart.png)
</details>

1. Clients retrieve a token from your app server.
2. Client A and Client B log in to <Vg k="COMPANY" /> <Vg k="CHAT"/>.
3. Client A sends a message to Client B. The message is sent to the <Vg k="COMPANY" /> <Vg k="CHAT"/> server, and the server delivers the message to Client B. When Client B receives the message, the SDK triggers an event. Client B listens for the event and gets the message.
