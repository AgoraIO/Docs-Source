1. **Add a variable for the token server url**

    ```dart
    String serverUrl = "https://agora-temp.herokuapp.com"; // The base URL to your token server. For example, https://app-name.herokuapp.com"
    ```

1. **Set the token server url for the `AgoraClient`**

    Under `agoraConnectionData:`, **replace** the line `tempToken: token,` with the following:

    ```dart
    tokenUrl: serverUrl,
    ```
