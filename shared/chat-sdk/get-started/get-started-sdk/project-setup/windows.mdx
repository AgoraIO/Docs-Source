This section shows how to integrate <Vpd k="SDK" /> into your project.

1. Create a Windows project

    1. Open **Visual Studio** and [Create a C# WPF app](https://learn.microsoft.com/en-us/dotnet/desktop/wpf/get-started/create-app-visual-studio?view=netdesktop-7.0#create-a-wpf-app).
    1. Set the **Active solution configuration** as **Debug** and **Active solution platform** as **x64**.

2. Integrate the Chat SDK

    1. [Download](/sdks) the <Vpd k="SDK" /> NuGet package.

    2. In the Solution Explorer, right-click `<project-name>` and select **Manage NuGet Packages...** from the drop-down menu.

    3. On the `NuGet: <project-name>` page, select the **Gear** icon at the top right.

    4. In the **Options** pop-up window, click **+** at the top right to add a package source. Select the new package source, specify its name in the **Name** box, set its **Source** to the parent directory where the downloaded Nuget package resides, and click **OK**.

    5. Go back to the `NuGet: <project-name>` page, open the **Package source** drop-down list at the top right, and select the newly-added package.

    6. In the **Browse** tab of the `NuGet: <project-name>` page, select the **agora_chat_sdk** item. If you don't see this item, refresh the page. Click **Install**.

    7. In the pop-up **Preview Changes** window, click **OK** to complete the installation.
